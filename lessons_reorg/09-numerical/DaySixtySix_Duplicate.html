Section 1

Fourier Series

Any periodic function `f(x)` on `[-\pi, \pi]` can be decomposed into sines and cosines.

f(x) \sim \sum_{n=-\infty}^\infty c_n e^{inx}

Coefficients: `c_n = \frac{1}{2\pi} \int_{-\pi}^\pi f(x) e^{-inx} dx`.

Section 2

Convergence Issues

Does the series actually equal `f(x)`?

  • Absolutely? Only if `f` is smooth enough (e.g., `C^1`).
  • Pointwise? Yes, if `f` is piecewise continuous (Dirichlet condition), to `(f(x+) + f(x-))/2`.
  • L^2 Norm? Yes, for any `f \in L^2`. (Carleson's theorem says almost everywhere too!).

The Gibbs Phenomenon occurs at discontinuities: the series overshoots by ~9%.

Section 3

Fourier Transform

For non-periodic functions on `\mathbb{R}`, we use the transform.

\hat{f}(\xi) = \int_{-\infty}^\infty f(x) e^{-2\pi i x \xi} dx

Inverse:

f(x) = \int_{-\infty}^\infty \hat{f}(\xi) e^{2\pi i x \xi} d\xi

It maps time domain to frequency domain.

Section 4

Plancherel Theorem

Theorem: `\|f\|_2 = \|\hat{f}\|_2`

Total energy is conserved. The integral of the square of the signal equals the integral of the square of its spectrum.

Section 5

Convolution Theorem

The Fourier Transform turns convolution into multiplication.

\widehat{f * g} = \hat{f} \cdot \hat{g}

This is why we use it for signal processing (filtering) and solving PDEs (Turn derivatives into algebra).

Section 6

Python: FFT

Python Fast Fourier Transform 
import numpy as np
import matplotlib.pyplot as plt

# Generate a signal with two frequencies
fs = 1000 # Sampling rate
t = np.linspace(0, 1, fs)
f1, f2 = 50, 120 # Hz
signal = np.sin(2*np.pi*f1*t) + 0.5*np.sin(2*np.pi*f2*t)

# Add noise
noise = 0.5 * np.random.randn(len(t))
noisy_signal = signal + noise

# Compute FFT
fft_spectrum = np.fft.fft(noisy_signal)
freqs = np.fft.fftfreq(len(t), 1/fs)

# Plot magnitude (only positive frequencies)
mask = freqs > 0
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.plot(t[:100], noisy_signal[:100])
plt.title("Noisy Signal (Time Domain)")
plt.xlabel("Time (s)")

plt.subplot(1, 2, 2)
plt.plot(freqs[mask], np.abs(fft_spectrum[mask]))
plt.title("FFT Magnitude (Frequency Domain)")
plt.xlabel("Frequency (Hz)")
plt.xlim(0, 200) # Zoom in on interest region
plt.grid()
plt.show()

# Key result: Peaks clearly visible at 50Hz and 120Hz despite noise.
