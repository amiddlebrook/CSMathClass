Section 1

Holomorphic Functions

In Real analysis, a function can be differentiable once but not twice (e.g., `x|x|`). In Complex analysis, if `f(z)` is differentiable ONCE, it is differentiable infinitely many times.

Definition: `f: \mathbb{C} \to \mathbb{C}` is Holomorphic at `z_0` if the limit exists:
`\lim_{h \to 0} \frac{f(z_0+h) - f(z_0)}{h}`
Section 2

Cauchy-Riemann Equations

Let `f(x+iy) = u(x,y) + i v(x,y)`. For `f` to be holomorphic, `u` and `v` must satisfy:

\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}

This implies `u` and `v` are Harmonic (`\Delta u = 0`).

Geometric Interpretation: The map is conformal (preserves angles locally).

Section 3

Cauchy's Theorem

Theorem: If `f` is holomorphic on a simply connected domain `D` and `\gamma` is a closed loop in `D`:
`\oint_\gamma f(z) \, dz = 0`

Path independence!

Section 4

Cauchy Integral Formula

Values inside a loop are determined by values ON the loop.

f(a) = \frac{1}{2\pi i} \oint_\gamma \frac{f(z)}{z-a} \, dz

By differentiating under the integral sign, we get formulas for all derivatives `f^{(n)}(a)`. This proves infinite differentiability.

Section 5

Analyticity

Because `1/(z-a)` has a power series expansion (geometric series), `f(z)` has a power series expansion.

Theorem: Holomorphic `\implies` Analytic.
`f(z) = \sum_{n=0}^\infty c_n (z-a)^n`

This is unique to complex numbers. Real differentiable functions (like bump functions) are not always analytic.

Section 6

Python: Domain Coloring

Python Visualizing Complex Functions 
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.colors as mcolors

# Visualize f(z) = (z^2 - 1)(z - 2 - i)^2 / (z^2 + 2 + 2i)
# Method: Domain Coloring via HSV
# Hue = Argument (Angle), Saturation/Value = Magnitude

def func(z):
    return (z**2 - 1) * (z - 2 - 1j)**2 / (z**2 + 2 + 2j)

# Grid
x = np.linspace(-3, 3, 400)
y = np.linspace(-3, 3, 400)
X, Y = np.meshgrid(x, y)
Z = X + 1j * Y

W = func(Z)

# Map to Color
angle = np.angle(W)
mag = np.abs(W)

# Hue: Angle (-pi, pi) -> (0, 1)
hue = (angle + np.pi) / (2 * np.pi)
# Saturation: 1
sat = np.ones_like(hue)
# Value: Periodic bands based on log magnitude (to see poles/zeros)
val = (np.sin(5 * np.log(mag + 1e-6)) + 1) / 4 + 0.5 

hsv = np.dstack((hue, sat, val))
rgb = mcolors.hsv_to_rgb(hsv)

plt.figure(figsize=(8, 8))
plt.imshow(rgb, extent=[-3, 3, -3, 3], origin='lower')
plt.title("Domain Coloring: Zeros (dark centers) vs Poles (light centers)")
plt.xlabel("Re(z)")
plt.ylabel("Im(z)")
plt.show()
