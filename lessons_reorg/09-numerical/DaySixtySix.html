Section 1

What is a Field?

A Field is a commutative ring where every non-zero element has a multiplicative inverse. Basically, you can add, subtract, multiply, and divide (except by 0).

Infinite Fields: $\mathbb{Q}$ (Rationals), $\mathbb{R}$ (Reals), $\mathbb{C}$ (Complex).
Note: $\mathbb{Z}$ (Integers) is NOT a field because $1/2 \notin \mathbb{Z}$.
Finite Fields: Integers modulo a prime $p$ ($\mathbb{Z}_p$, also denoted $\mathbb{F}_p$ or $GF(p)$).
Section 2

Finite Fields (Galois Fields)

Finite fields are critical in CS. They are the only way to do arithmetic on fixed-size bits without overflow or approximation errors.

Theorem: A finite field exists if and only if its order (size) is a prime power $q = p^n$. This field is unique up to isomorphism and is denoted $GF(p^n)$.

Prime Fields ($n=1$)

Simply modulo $p$ arithmetic.

In GF(5): 2 + 4 = 1 3 × 2 = 1 (so 3⁻¹ = 2)

Extension Fields ($n > 1$)

Elements are polynomials with coefficients in $GF(p)$, modulo an irreducible polynomial of degree $n$. This is how $GF(2^8)$ (AES Byte) is constructed!

Python Prime Field Arithmetic 
# Implementing GF(7)
p = 7

def add(a, b): return (a + b) % p
def sub(a, b): return (a - b) % p
def mul(a, b): return (a * b) % p

def inv(a):
    # Fermat's Little Theorem: a^(p-2) is inverse mod p
    return pow(a, p-2, p)

print(f"3 * 4 in GF(7) = {mul(3, 4)}") # 5
print(f"Inverse of 2 in GF(7) = {inv(2)}") # 4, because 2*4=8=1
Section 3

Galois Theory Overview

The Big Idea: We can understand fields by studying the group of their symmetries (automorphisms).

Unsolvability of the Quintic

Why is there no quadratic formula for degree 5 polynomials? Galois showed that the symmetry group of a 5th degree polynomial (S5) is not "solvable" (cannot be broken down into abelian parts).

Section 4

CS Applications

Cryptography (AES)

AES treats bytes as elements of $GF(2^8)$. The "MixColumns" step is matrix multiplication over this field.

Error Correction (Reed-Solomon)

Used in QR codes and CDs. Data is treated as a polynomial over a finite field. Redundancy points are added.
