Section 1

Review: Metric Spaces

In Real Analysis, we relied on distance `d(x, y)` to define limits. Open balls `B_\epsilon(x)` were the fundamental building blocks.

But what if we don't have a distance? What if we only care about adjacency or convergence in a way that distance cannot capture (e.g., pointwise convergence)?

We abstract the "Open Ball" into just "Open Set".

Section 2

Definition of a Topology

Definition: A Topology on a set `X` is a collection `\tau` of subsets of `X` (called open sets) satisfying:
  1. `\emptyset \in \tau` and `X \in \tau`.
  2. The union of any collection of sets in `\tau` is in `\tau` (Arbitrary Unions).
  3. The intersection of any finite collection of sets in `\tau` is in `\tau` (Finite Intersections).

The pair `(X, \tau)` is a topological space.

Why finite intersections? Consider `U_n = (-1/n, 1/n)` in `\mathbb{R}`. Each is open. But `\cap_{n=1}^\infty U_n = \{0\}`, which is closed! Infinite intersections break openness.
Section 3

Open, Closed, and Clopen

  • Open: By definition, any set `U \in \tau`.
  • Closed: A set `C` is closed if its complement `X \setminus C` is open.
  • Clopen: A set that is both open and closed (e.g., `\emptyset` and `X` always).
"Closed" is NOT "Not Open". A door can be open, closed, both (revolving?), or neither (a frame with no door?). Most sets are neither.
Section 4

Basis of a Topology

Defining `\tau` by listing ALL open sets is impossible for `\mathbb{R}`. Instead, we specify a Basis `\mathcal{B}`.

A collection `\mathcal{B}` is a basis if every open set `U \in \tau` can be written as a union of elements from `\mathcal{B}`.

Example: For `\mathbb{R}`, the basis is the set of all open intervals `(a, b)`. Every open set in `\mathbb{R}` is a union of open intervals.

Section 5

Continuity Generalized

In calculus: `f` is continuous if `\lim_{x \to c} f(x) = f(c)`. In metric spaces: `\forall \epsilon, \exists \delta`...

In Topology, we have the elegant definition:

f: X \to Y \text{ is continuous if } \forall V \in \tau_Y, f^{-1}(V) \in \tau_X.

"The preimage of every open set is open."

This definition requires NO distance function.

Section 6

Compactness & Heine-Borel

Definition: A space `X` is Compact if every open cover of `X` has a finite subcover.

Open Cover: A collection `\{U_\alpha\}` of open sets such that `X \subseteq \bigcup U_\alpha`.

Finite Subcover: A finite selection `U_{\alpha_1}, \dots, U_{\alpha_n}` that still covers `X`.

Heine-Borel Theorem: $K \subseteq \mathbb{R}^n$ is compact $\iff K$ is closed and bounded.
(Warning: False in general metric spaces!)
Section 7

Connectedness

Definition: `X` is Disconnected if there exist two disjoint, non-empty open sets `U, V` such that `X = U \cup V`.

If no such separation exists, `X` is Connected.

Path Connected: You can draw a continuous line between any two points. Implies Connected.

Section 8

Applications

  • IVT: Continuous image of connected set is connected.
  • EVT: Continuous image of compact set is compact (contains max/min).
Section 9

Python: Finite Topologies

For finite sets, we can check the topology axioms explicitly.

Python Topology Validator 
from itertools import combinations

def is_topology(X, tau):
    # Convert list of lists to set of frozensets for easy set ops
    tau_set = set(frozenset(s) for s in tau)
    X_set = frozenset(X)
    
    # 1. Empty and X must be in tau
    if frozenset() not in tau_set: return False, "Missing empty set"
    if X_set not in tau_set: return False, "Missing X"
    
    # 2. Arbitrary Unions (Finite case: just all combinations)
    # 3. Finite Intersections
    # Optimization: Just check pairwise for finite tau
    for A in tau_set:
        for B in tau_set:
            if A | B not in tau_set:
                return False, f"Union of {set(A)} and {set(B)} missing"
            if A & B not in tau_set:
                return False, f"Intersection of {set(A)} and {set(B)} missing"
                
    return True, "Valid Topology"

# Example X = {1, 2, 3}
# Sierpinski Space: { {}, {1,2,3}, {1} } is NOT a topology?
# Wait: {1} U {1,2,3} = {1,2,3} (in). {1} & {1,2,3} = {1} (in).
# Standard Sierpinski on {0, 1} is {{}, {0,1}, {1}}.

X = {1, 2, 3}
# Try: {{}, {1,2,3}, {1}, {2}} -> Union {1,2} missing!
tau_bad = [[], [1, 2, 3], [1], [2]]
print(f"Bad: {is_topology(X, tau_bad)}")

# Try: {{}, {1,2,3}, {1}, {1,2}} -> Valid
tau_good = [[], [1, 2, 3], [1], [1, 2]]
print(f"Good: {is_topology(X, tau_good)}")
