Section 1

Modes of Convergence

Random Variables `X_n` are functions. Sequences of functions can converge in different ways.

  • In Probability: `\forall \epsilon, P(|X_n - X| > \epsilon) \to 0`. (Measure of bad set vanishes).
  • Almost Surely (a.s.): `P(\{\omega : \lim X_n(\omega) = X(\omega)\}) = 1`. (Pointwise convergence on set of measure 1).
  • In Distribution: CDFs converge. `F_n(x) \to F(x)`. (Weaker).
Section 2

Weak Law of Large Numbers (WLLN)

Theorem: If `X_i` are i.i.d. with finite mean `\mu`, then `\bar{X}_n \xrightarrow{p} \mu`.

Convergence in Probability. "For large `n`, the average is unlikely to be far from `\mu`."

Proof uses Chebyshev's Inequality.

Section 3

Strong Law of Large Numbers (SLLN)

Theorem: If `X_i` are i.i.d. with finite mean `\mu`, then `\bar{X}_n \xrightarrow{a.s.} \mu`.

Convergence Almost Surely. "The sample path `\bar{X}_n` will eventually stay arbitrarily close to `\mu` with probability 1."

Historically harder to prove (Kolmogorov).

Section 4

Central Limit Theorem (CLT)

The Laws of Large Numbers say `\bar{X}_n \to \mu` (spike). The CLT tells us about the shape of the fluctuations around that spike.

\sqrt{n}(\bar{X}_n - \mu) \xrightarrow{d} N(0, \sigma^2)

Convergence in Distribution. The error scales like `1/\sqrt{n}`.

Section 5

Proof Technique: Characteristic Functions

The standard proof of CLT relies on Characteristic Functions (Fourier Transforms of distributions).

`\phi_X(t) = E[e^{itX}]`.

Key Idea: If `\phi_{X_n}(t) \to \phi_X(t)`, then `X_n \to X` in distribution (Lévy's Continuity Theorem).

We show `\phi_{S_n}(t) \approx (1 - \frac{t^2}{2n})^n \to e^{-t^2/2}`, which is the char function of Normal.

Section 6

Python: Convergence Visualized

Python LLN vs CLT 
import numpy as np
import matplotlib.pyplot as plt

# Simulate N experiments, each with n samples
N = 1000
n = 1000
# Uniform(0, 1) -> Mean 0.5, Var 1/12
data = np.random.rand(N, n)

# Compute running averages for one path (SLLN check)
path = data[0]
running_avg = np.cumsum(path) / np.arange(1, n+1)

plt.figure(figsize=(12, 5))

# Plot 1: SLLN (One path settling down)
plt.subplot(1, 2, 1)
plt.plot(running_avg)
plt.axhline(0.5, color='r', linestyle='--', label='True Mean')
plt.title("Strong Law (Single Path)")
plt.xlabel("n")
plt.ylabel("Average")
plt.legend()

# Plot 2: CLT (Distribution of final averages)
# We need averages from many experiments (N)
final_avgs = np.mean(data, axis=1)
# Normalize to see standard normal
normalized = np.sqrt(n) * (final_avgs - 0.5) / np.sqrt(1/12)

plt.subplot(1, 2, 2)
plt.hist(normalized, bins=30, density=True, alpha=0.7, color='g')
# Overlay N(0,1)
xs = np.linspace(-4, 4, 100)
pdf = (1/np.sqrt(2*np.pi)) * np.exp(-xs**2/2)
plt.plot(xs, pdf, 'r-', lw=2, label='N(0,1)')
plt.title("CLT (Distribution of Averages)")
plt.legend()

plt.show()
