Section 1

Riemann vs Lebesgue

Riemann Integration (Calculus): Partitions the domain (x-axis) into vertical strips `dx`. If the function jumps around too much being erratic, these strips can't capture the area well.

Lebesgue Integration: Partitions the range (y-axis). We ask: "For which `x` is `f(x) \approx y`?" This set is measurable, and we compute `y \cdot \mu(\{x \mid f(x) \approx y\})`.

"I have to pay a certain sum... Riemann takes the coins out of his pocket in the order they appear... I empty my pockets, heap coins of the same value... and pay the heap." — Henri Lebesgue
Section 2

Simple Functions

A Simple Function `\phi` assumes only finitely many values `a_1, \dots, a_n` on sets `A_1, \dots, A_n`.

\phi(x) = \sum_{i=1}^n a_i \mathbf{1}_{A_i}(x)

The integral is defined naturally:

\int \phi \, d\mu = \sum_{i=1}^n a_i \mu(A_i)
Section 3

Defining the Integral

For a non-negative function `f \ge 0`:

\int f \, d\mu = \sup \left\{ \int \phi \, d\mu \mid 0 \le \phi \le f, \phi \text{ is simple} \right\}

It is the supremum of area under simple functions "approximating `f` from below".

For general `f`, split into positive and negative parts: `f = f^+ - f^-`. Then `\int f = \int f^+ - \int f^-`.

Section 4

Measurable Functions

For this to work, `f` must be Measurable. This means the pre-image of every open set (specifically intervals) is a measurable set.

\{x \mid f(x) > \alpha\} \in \mathcal{F} \quad \forall \alpha \in \mathbb{R}

Almost all distinct functions you've ever met (continuous, piecewise, limits of continuous sequences) are measurable.

Section 5

The Dirichlet Function

Example where Riemann fails but Lebesgue succeeds.

D(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}
  • Riemann: Fails. In any interval `[x_i, x_{i+1}]`, there is both rational and irrational. Upper sum is `1(b-a)`, lower sum is `0(b-a)`. They don't match.
  • Lebesgue: `\mathbb{Q}` is countable, so `\mu(\mathbb{Q}) = 0`. The set of irrationals has measure `1`.
    `\int D = 1 \cdot \mu(\mathbb{Q}) + 0 \cdot \mu(\mathbb{Q}^c) = 1 \cdot 0 + 0 \cdot 1 = 0`.
Section 6

Python: Riemann vs Lebesgue Logic

Python Integration Concepts 
import numpy as np
import matplotlib.pyplot as plt

# Simulate a "jagged" function
def f(x):
    return np.sin(x**2) + np.random.normal(0, 0.1, size=len(x))

x = np.linspace(0, 5, 100)
y = f(x)

# 1. Riemann Strategy (Vertical rectangles)
dx = x[1] - x[0]
riemann_sum = np.sum(y * dx)

# 2. Lebesgue Strategy (Horizontal slicing)
# We bin the Y values and see how many X fall into that bin
y_min, y_max = np.min(y), np.max(y)
dy = 0.1
y_levels = np.arange(y_min, y_max, dy)
lebesgue_sum = 0

for level in y_levels:
    # Measure of set where y is approx 'level'
    # In discrete, this is count * dx
    indices = np.where((y >= level) & (y < level + dy))[0]
    measure = len(indices) * dx
    lebesgue_sum += level * measure

print(f"Riemann Sum: {riemann_sum:.4f}")
print(f"Lebesgue Sum: {lebesgue_sum:.4f}")
print("They converge to the same value for nice functions.")

plt.plot(x, y, label='Function')
plt.hlines(y_levels[::5], 0, 5, colors='r', alpha=0.3, label='Lebesgue Slices')
plt.title("Riemann (Vertical) vs Lebesgue (Horizontal)")
plt.legend()
plt.show()
