Section 1

Your Brain is for Thinking, Not Holding

David Allen (GTD): "Your mind is for having ideas, not holding them."

You cannot remember every theorem, proof, and technique in this 120-day course. You need an external system.

Section 2

Atomic Notes (Zettelkasten)

Don't write long summaries. Write Atomic Notes. One concept per note.

Structure: 1. Concept Name 2. Definition (in your own words) 3. Example 4. Links to related concepts.

Tools: Obsidian, Roam Research, Notion.

Section 3

The Lemma Notebook

Keep a specific list of "Tricks of the Trade" or "Lemmas".

  • "When you see a sum, think Integral Test."
  • "To bound a probability, try Markov or Chebyshev."
  • "To show a matrix is PSD, check eigenvalues."

Review this list before starting any problem set.

Section 4

Spaced Repetition

For fundamental definitions that you MUST memorize, use Spaced Repetition Systems (SRS) like Anki.

Create flashcards: Front = "Definition of Convolution", Back = Formula. Review daily.

Section 5

Python: Knowledge Graph

Python Graph Visualization 
import networkx as nx
import matplotlib.pyplot as plt

# Simulate Zettelkasten Links
topics = [
    ("LinAlg", "Calculus"),
    ("Calculus", "Optimization"),
    ("LinAlg", "Optimization"),
    ("Probability", "Stats"),
    ("Stats", "Optimization"), # ML
    ("Algorithms", "Graph Theory"),
    ("Graph Theory", "LinAlg") # Spectral Graph Theory
]

G = nx.Graph()
G.add_edges_from(topics)

plt.figure(figsize=(8, 6))
pos = nx.spring_layout(G, seed=42)
nx.draw(G, pos, with_labels=True, node_color='lightblue', 
        node_size=2000, font_size=10, font_weight='bold')
plt.title("Your Knowledge Graph")
plt.show()
