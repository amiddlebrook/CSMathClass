Section 1

Hoare Triples

Formal notation for code correctness:

\{P\} C \{Q\}

Meaning: If precondition `P` is true, and command `C` executes, then postcondition `Q` will be true.

Section 2

Inference Rules

  • Assignment: `\{Q[x/E]\} x := E \{Q\}`. (Backwards substitution).
  • Composition: If `\{P\} S1 \{R\}` and `\{R\} S2 \{Q\}`, then `\{P\} S1; S2 \{Q\}`.
  • Conditional: If `\{P \land B\} S1 \{Q\}` and `\{P \land \neg B\} S2 \{Q\}`, then `\{P\} \text{if } B \text{ then } S1 \text{ else } S2 \{Q\}`.
Section 3

Loop Invariants

For loops `while B do S`, we need an Invariant `I` such that:

  • `\{I \land B\} S \{I\}` (Maintained by body)
  • `P \implies I` (True at start)
  • `(I \land \neg B) \implies Q` (Implies goal at end)

Finding loop invariants is the hardest part of verification (and is undecidable in general).

Section 4

Total Correctness

Partial Correctness: If it terminates, the answer is right.

Total Correctness: Partial Correctness + Termination.

To prove termination, we use a Variant Function (or ranking function) `v` that maps state to a well-ordered set (like natural numbers) and decreases with every loop iteration.

Section 5

Model Checking

Instead of manual proofs, we can check a finite state machine against a property (in Temporal Logic like LTL).

Tools like TLA+ or Spin explore all reachable states (up to a bound) to find bugs like deadlocks or race conditions.

Section 6

Python: Assertions

Python Runtime Verification 
def gcd(a, b):
    # Precondition: Inputs are positive logic
    assert a > 0 and b > 0
    
    # Save purely for verification logic
    a_orig, b_orig = a, b
    
    while b != 0:
        # Invariant: gcd(a, b) == gcd(a_orig, b_orig)
        # Variant: b decreases (or a+b decreases)
        (a, b) = (b, a % b)
        
    # Postcondition
    result = a
    assert result > 0
    assert a_orig % result == 0
    assert b_orig % result == 0
    return result

# In formal verification, we wouldn't just assert (runtime),
# we would use a solver (like Z3) to PROVE the asserts never fail.

print(gcd(48, 18))
