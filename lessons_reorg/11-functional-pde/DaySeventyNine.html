Section 1

Symmetric Key

Alice and Bob share a secret key `K`. `E(m, K) = c`, `D(c, K) = m`.

Example: One-Time Pad (XOR with random key). Proven perfectly secure if key is truly random and never reused.

Issue: Key Exchange. How to share `K` securely?

Section 2

Public Key Cryptography

Uses two keys: Public `PK` (Encryption) and Private `SK` (Decryption).

Anyone can lock the box (Pub), only you can open it (Priv).

Relies on One-Way Functions: Easy to compute `f(x)`, hard to find `x` given `y=f(x)`.

Section 3

RSA

Relies on the hardness of Factoring.

  1. Choose primes `p, q`. Let `n = p q`.
  2. Compute `\phi(n) = (p-1)(q-1)`.
  3. Choose `e` coprime to `\phi(n)`. Find `d` such that `d e \equiv 1 \pmod{\phi(n)}`.
  4. Public: `(e, n)`. Private: `d`.
  5. Encrypt: `c = m^e \pmod n`.
  6. Decrypt: `m = c^d \pmod n`.
Section 4

Diffie-Hellman Key Exchange

Allows two parties to agree on a secret key over an insecure channel.

Relies on Discrete Logarithm Problem: Given `g^a \pmod p`, find `a`.

  • Alice sends `g^a`. Bob sends `g^b`.
  • Alice computes `(g^b)^a`. Bob computes `(g^a)^b`.
  • They share `g^{ab}`. Secure!
Section 5

Zero-Knowledge Proofs

Proving you know a secret without revealing the secret.

Example: Finding Waldo. I can prove I found Waldo by covering the rest of the image, showing you only Waldo through a hole. You know I know where he is, but you don't know where (relative to the full page).

Section 6

Python: RSA Text

Python Toy RSA 
def modinv(a, m):
    # Extended Euclidean Algorithm
    m0, x0, x1 = m, 0, 1
    if m == 1: return 0
    while a > 1:
        q = a // m
        m, a = a % m, m
        x0, x1 = x1 - q * x0, x0
    if x1 < 0: x1 += m0
    return x1

# 1. Setup
p, q = 61, 53
n = p * q
phi = (p-1) * (q-1)
e = 17 # Public Exponent
d = modinv(e, phi) # Private Exponent

print(f"Public Key (e,n): ({e}, {n})")
print(f"Private Key d: {d}")

# 2. Encrypt Message (must be < n)
m = 123
c = pow(m, e, n)
print(f"Encrypted: {c}")

# 3. Decrypt
m_dec = pow(c, d, n)
print(f"Decrypted: {m_dec}")

assert m == m_dec
