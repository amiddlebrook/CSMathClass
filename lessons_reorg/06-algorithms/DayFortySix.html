Section 1

Riemann Sums

We approximate the area under $f(x)$ on $[a, b]$ by partitioning the interval into $n$ sub-intervals using rectangles.

$S = \sum_{i=1}^n f(x_i^*) \Delta x_i$

As the width of the largest sub-interval approaches 0 (limit as $n \to \infty$), if the sum converges to a unique value $I$, then $I = \int_a^b f(x) dx$.

Section 2

Integrability Criteria

A bounded function on $[a,b]$ is Riemann integrable if and only if it is continuous almost everywhere (Lebesgue's Criterion).

Continuous functions are always integrable. Monotonic functions are always integrable. Dirichlet function (1 on rationals, 0 on irrationals) is NOT Riemann integrable.

Section 3

Fundamental Theorem of Calculus

Connects differentiation and integration as inverse processes.

Part 1: If $F(x) = \int_a^x f(t) dt$, then $F'(x) = f(x)$.
Part 2: $\int_a^b f(x) dx = F(b) - F(a)$, where $F' = f$.
Section 4

Python Integration

Python Numerical Integration (Trapezoidal) 
import numpy as np

# Function to integrate: f(x) = x^2
f = lambda x: x**2
a, b = 0, 1
n = 1000 # Number of trapezoids

# Create x values
x = np.linspace(a, b, n+1)
y = f(x)

# Trapezoidal Rule: np.trapz(y, x)
area = np.trapz(y, x)
exact = (1**3)/3 - (0**3)/3 # 1/3

print(f"Approx Area: {area:.6f}")
print(f"Exact Area: {exact:.6f}")

# Using SciPy for adaptive quadrature (quad)
from scipy.integration import quad
res, err = quad(f, a, b)
print(f"SciPy Quad: {res:.10f}")
