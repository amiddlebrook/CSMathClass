Section 1

Joint Probability

Real-world problems involve multiple interacting variables. The Joint Distribution describes random variables $X$ and $Y$ occurring together.

Joint PMF/PDF

Discrete: p_{XY}(x,y) = P(X=x, Y=y) Continuous: f_{XY}(x,y)

Sum/Integral over all $x,y$ must equal 1.

Marginal Distribution

Recovering the distribution of just $X$ by summing out $Y$.

p_X(x) = \sum_y p_{XY}(x,y) f_X(x) = \int f_{XY}(x,y) dy

Independence

$X$ and $Y$ are independent iff: $$P(X=x, Y=y) = P(X=x)P(Y=y)$$ For all $x, y$.
Section 2

Covariance & Correlation

How do we measure if $X$ and $Y$ "move together"?

Covariance

Cov(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]

If $X, Y$ independent $\implies Cov(X,Y) = 0$.
WARNING: The converse is FALSE. Zero covariance does not imply independence (only Linear independence).

Correlation ($\rho$)

Covariance scaled between -1 and 1.

\rho_{XY} = \frac{Cov(X,Y)}{\sigma_X \sigma_Y}
  • $\rho = 1$: Perfect positive linear relationship.
  • $\rho = -1$: Perfect negative linear relationship.
  • $\rho = 0$: Uncorrelated.

Variance of Sums

Var(X + Y) = Var(X) + Var(Y) + 2Cov(X, Y)

This explains why diversification reduces risk in finance ($Cov < 0$).

Section 3

Markov's Inequality

The First Tail Bound: For non-negative X and $a > 0$: $$P(X \geq a) \leq \frac{E[X]}{a}$$

Requires almost no assumptions (only non-negativity). Consequently, it is loose.
Example: If average lifespan is 75, at most 50% of people can live to 150.

Section 4

Chebyshev's Inequality

Using Variance: For ANY random variable with finite variance: $$P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}$$

This bound holds for any distribution. It guarantees that extreme outliers are rare.

Section 5

Chernoff Bounds

When $X$ is the sum of many independent random variables, the concentration is much stronger than Chebyshev suggests—it is exponential.

Hoeffding's Inequality (Additive Form)
P(|X - \mu| \geq t) \leq 2 \exp\left(-\frac{2t^2}{n}\right)
Why is this huge?
With Chebyshev, to get $1\%$ error, standard deviation decreases by $\sqrt{n}$. To get $10^{-6}$ probability, you need massive $k$.
With Chernoff, the probability drops exponentially with $n$. This is the mathematical foundation for why Machine Learning works (PAC Learning).
Real-world Example: Polling How many people do you need to survey to be 99% confident your result is within 3% of the truth? Using Hoeffding: n >= ln(2/0.01) / (2 * 0.03^2) ≈ 2944 people. Ideally independent of population size!

Python Simulations

Python Correlation & Bounds 
import numpy as np

# 1. Correlation Experiment
n = 1000
x = np.random.randn(n)
# y is dependent on x plus noise
y = 0.8 * x + 0.6 * np.random.randn(n) 

corr = np.corrcoef(x, y)[0, 1]
print(f"Sample Correlation: {corr:.4f}") # Should be close to 0.8

# 2. Chernoff vs Chebyshev
t = 100 # deviation
n_coins = 10000
sigma_coins = np.sqrt(n_coins * 0.25) # 50

# Chebyshev bound for P(|X - mu| >= 100) -> k=2 (since sigma=50)
cheb = 1 / (2**2) # 0.25

# Hoeffding bound
hoeff = 2 * np.exp(-2 * t**2 / n_coins) # 2 * exp(-2 * 10000 / 10000) = 2 * e^-2
# 2 * 0.135 = 0.27 (Wait, Hoeffding is better for larger deviations)

print(f"Chebyshev Bound: {cheb:.4f}")
print(f"Hoeffding Bound: {hoeff:.4f}")
