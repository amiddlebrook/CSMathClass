Section 1

The Definition

We say $\lim_{x \to c} f(x) = L$ if for every $\epsilon > 0$, there exists a $\delta > 0$ such that:

$0 < |x - c| < \delta \implies |f(x) - L| < \epsilon$

Intuitively: "No matter how close ($\epsilon$) you want the output to be to $L$, I can find a proximity ($\delta$) for the input to guarantee it."

This was formalized by Weierstrass in the 19th century to put calculus on solid ground.
Section 2

Limit Laws

If limits exist, they behave nicely:

  • $\lim (f + g) = \lim f + \lim g$
  • $\lim (fg) = (\lim f)(\lim g)$
  • $\lim (f/g) = (\lim f)/(\lim g)$ (if $\lim g \neq 0$)
Section 3

Squeeze Theorem

If $g(x) \le f(x) \le h(x)$ near $c$, and $\lim_{x \to c} g(x) = \lim_{x \to c} h(x) = L$, then: $$\lim_{x \to c} f(x) = L$$

Useful for oscillating functions like $x^2 \sin(1/x)$.

Section 4

Continuity

A function $f$ is continuous at $c$ if:

1. f(c) is defined 2. \lim_{x \to c} f(x) exists 3. \lim_{x \to c} f(x) = f(c)

In $\epsilon-\delta$ terms: $\forall \epsilon > 0, \exists \delta > 0$ such that $|x - c| < \delta \implies |f(x) - f(c)| < \epsilon$.

Section 5

Topology of R

Real Analysis relies on the structure of the Real line.

Open Set: A set $U$ is open if for every $x \in U$, there is a neighborhood of $x$ completely inside $U$.
Example: $(0, 1)$ is open. $[0, 1]$ is not.
Closed Set: A set $F$ is closed if its complement is open. Alternatively, it contains all its limit points.
Example: $[0, 1]$ is closed.
Theorem: $f$ is continuous $\iff$ the inverse image of every open set is open.
Section 6

Intermediate Value Theorem

If $f$ is continuous on $[a, b]$ and $f(a) < y < f(b)$, then there exists some $c \in (a, b)$ such that $f(c)=y$.

"If you go from -1 to 1 without jumping, you MUST cross 0."
This is useful for finding roots.

Section 7

SymPy Limits

SymPy can compute symbolic limits, handling $0/0$ and $\infty$ cases.

Python Computing Limits 
from sympy import symbols, limit, sin, oo

x = symbols('x')

# 1. Standard approach
expr1 = (x**2 - 1) / (x - 1)
lim1 = limit(expr1, x, 1)
print(f"Limit of (x^2-1)/(x-1) as x->1: {lim1}") # 2

# 2. Famous sin(x)/x
expr2 = sin(x)/x
lim2 = limit(expr2, x, 0)
print(f"Limit of sin(x)/x as x->0: {lim2}") # 1

# 3. Infinity
expr3 = (1 + 1/x)**x
lim3 = limit(expr3, x, oo)
print(f"Limit of (1+1/x)^x as x->oo: {lim3}") # E (Euler's number)
Section 8

Python Root Finding

We can use the Bisection Method (based on IVT) to find roots.

Python Bisection Method 
def solve_ivt(f, a, b, tol=1e-5):
    """Find root of f in [a, b] using Bisection."""
    if f(a) * f(b) > 0:
        return None # IVT doesn't apply
    
    mid = (a + b) / 2
    while abs(f(mid)) > tol:
        if f(a) * f(mid) < 0:
            b = mid
        else:
            a = mid
        mid = (a + b) / 2
    return mid

# Function: x^2 - 2 (Root is sqrt(2))
f = lambda x: x**2 - 2

root = solve_ivt(f, 0, 2)
print(f"Approx root of x^2 - 2: {root}")
print(f"Precision: {root**2}")
