Section 1

Partial Derivatives

For $f(x, y)$, the derivative with respect to $x$ (holding $y$ constant) is $\frac{\partial f}{\partial x}$.

$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}$
Section 2

The Gradient

The vector of all partial derivatives. $$ \nabla f = \begin{bmatrix} \frac{\partial f}{\partial x_1} \\ \vdots \\ \frac{\partial f}{\partial x_n} \end{bmatrix} $$

The gradient points in the direction of steepest ascent.
$-\nabla f$ is the direction of steepest descent (used in Machine Learning).
Section 3

Hessian & Jacobian

Hessian ($H$): Matrix of second order partials.
If $H$ is Positive Definite at a critical point, it's a local minimum.
Jacobian ($J$): Matrix of first order partials for vector-valued functions.
Used in change of variables (integration) and robotics.
Section 4

Gradient Descent

Visualizing optimization on a 2D surface.

Python Gradient Descent 
import numpy as np

# Function: f(x, y) = x^2 + 2y^2
# Gradient: [2x, 4y]
def grad_f(v):
    x, y = v
    return np.array([2*x, 4*y])

# Initial guess
start = np.array([3.0, 2.0])
lr = 0.1 # Learning Rate

current = start
print("Step \t Location \t Gradient")
print("-"*40)

for i in range(5):
    grad = grad_f(current)
    print(f"{i} \t {np.round(current, 2)} \t {np.round(grad, 2)}")
    current = current - lr * grad

# See how it moves towards [0, 0]
