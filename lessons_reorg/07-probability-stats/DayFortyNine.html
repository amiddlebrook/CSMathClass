Section 1

Metric Space Definition

A pair $(M, d)$ is a metric space if $d: M \times M \to \mathbb{R}$ satisfies:

1. d(x, y) \ge 0 (Non-negativity) 2. d(x, y) = 0 \iff x = y (Identity) 3. d(x, y) = d(y, x) (Symmetry) 4. d(x, z) \le d(x, y) + d(y, z) (Triangle Inequality)
Section 2

Common Metrics

Euclidean ($L_2$): $\sqrt{\sum (x_i - y_i)^2}$
Manhattan ($L_1$): $\sum |x_i - y_i|$ (Taxicab definition)
Chebyshev ($L_\infty$): $\max |x_i - y_i|$ (Chess king moves)
Discrete Metric: $d(x, y) = 1$ if $x \neq y$, else 0.
Section 3

Open Balls & Topology

We define an open ball $B_r(x)$ as $\{y \in M \mid d(x, y) < r\}$.
This allows us to do calculus (limits, continuity, etc.) on abstract sets like strings, graphs, or functions!

Section 4

Completeness

A metric space is complete if every Cauchy sequence converges to a point within the space.
$\mathbb{Q}$ (Rationals) is NOT complete (since $\sqrt{2}$ is missing). $\mathbb{R}$ is complete.

Banach Fixed Point Theorem (Contraction Mapping): A contraction $f: M \to M$ on a complete metric space has a unique fixed point. This proves solutions to Differential Equations exist!
Section 5

Computing Distances

Using SciPy for various metrics.

Python Distance Metrics 
import numpy as np
from scipy.spatial import distance

p1 = np.array([1, 2, 3])
p2 = np.array([4, 6, 8])

# 1. Euclidean
d_euc = distance.euclidean(p1, p2)
print(f"Euclidean: {d_euc:.2f}")

# 2. Manhattan
d_man = distance.cityblock(p1, p2)
print(f"Manhattan: {d_man}")

# 3. Chebyshev
d_cheb = distance.chebyshev(p1, p2)
print(f"Chebyshev: {d_cheb}")

# 4. Hamming (for boolean/categorical vectors)
v1 = [1, 0, 1, 1]
v2 = [1, 1, 0, 1]
d_ham = distance.hamming(v1, v2)
print(f"Hamming (% diff): {d_ham}")
