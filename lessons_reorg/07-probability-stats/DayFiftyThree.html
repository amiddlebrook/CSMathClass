Section 1

Fields Defined

Definition

Field (F, +, ×): A Commutative Ring with Unity where every non-zero element has a multiplicative inverse.
Essentially: You can add, subtract, multiply, and divide (except by 0). Examples: • ℚ (Rationals), ℝ (Reals), ℂ (Complex) • ℤ_p (Integers mod prime p) • Gal(pⁿ) (Finite fields of prime power order) Non-example: ℤ (no division, e.g., 1/2 ∉ ℤ).
Section 2

Field Extensions

Extending a Field

If K is a field containing F, we say K/F is an extension. K becomes a Vector Space over F! Degree [K:F]: The dimension of K as an F-vector space. EXAMPLE: ℂ = ℝ(i) = {a + bi | a,b ∈ ℝ} Basis: {1, i} Degree [ℂ:ℝ] = 2.

Finite Fields (Galois Fields)

GF(pⁿ): A field with pⁿ elements exists for any prime p and n ≥ 1. Crucial for Cryptography (AES uses GF(2⁸)). Implementation: Polynomials mod an irreducible polynomial.
Section 3

Galois Theory Basics

Galois Group Gal(K/F): Automorphisms of K that fix every element of F. σ ∈ Gal(K/F) ⟹ σ(x) = x for all x ∈ F. Fundamental Theorem: There is a 1-to-1 correspondence between: Subgroups of Gal(K/F) ⟷ Intermediate fields between F and K.

Python: Finite Field Arithmetic

Simulating GF(2^8) (AES field) logic.

Python GF(256) Addition 
# In GF(2^n), Addition is XOR!
# Multiplication requires modulo an irreducible polynomial.

def add_gf256(a, b):
    return a ^ b  # XOR

a = 0x53
b = 0xCA
result = add_gf256(a, b)

print(f"{a:02x} + {b:02x} = {result:02x}")
