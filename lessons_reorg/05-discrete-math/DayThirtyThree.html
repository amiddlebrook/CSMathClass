Section 1

Tree Basics

Tree: Connected acyclic graph with n nodes and n-1 edges.
root / \ child child / \ leaf leaf Terminology: • Root: top node (no parent) • Leaf: node with no children • Height: longest path from root to leaf • Depth: path length from root to node
Section 2

Binary Search Trees

BST Property: Left subtree < Node < Right subtree
BST Operations (balanced): Search: O(log n) Insert: O(log n) Delete: O(log n) Unbalanced BST degrades to O(n)!
Python BST search 
def search(root, target):
    if not root:
        return None
    if target == root.val:
        return root
    elif target < root.val:
        return search(root.left, target)
    else:
        return search(root.right, target)
Self-balancing trees: AVL, Red-Black guarantee O(log n) height.
Section 3

Heaps

Min-Heap

Parent ≤ Children Root = minimum peek: O(1) push: O(log n) pop: O(log n)

Max-Heap

Parent ≥ Children Root = maximum Used in heapsort
Python Using heapq (min-heap) 
import heapq

h = []
heapq.heappush(h, 5)
heapq.heappush(h, 2)
heapq.heappush(h, 8)

print(h[0])         # 2 (min)
print(heapq.heappop(h))  # 2
Section 4

Tries (Prefix Trees)

Trie: Tree for storing strings where each node is a character.
Insert "cat", "car", "card": root | c | a / \ t r | d Search/Insert: O(length of string)

Use cases: autocomplete, spell check, IP routing tables.

Section 5

Graph Basics

Graph G = (V, E) where V = vertices, E = edges
Types: • Directed vs Undirected • Weighted vs Unweighted • Cyclic vs Acyclic (DAG) • Connected vs Disconnected |V| = n (vertices), |E| = m (edges) Sparse: m ≪ n² Dense: m ≈ n²
Section 6

Adjacency List

Adjacency List: For each vertex, store list of neighbors.
Space: O(V + E) 0: [1, 2] 1: [0, 2, 3] 2: [0, 1] 3: [1]
Python Adjacency list with dict 
# Using dict of lists
graph = {
    0: [1, 2],
    1: [0, 2, 3],
    2: [0, 1],
    3: [1]
}

# Get neighbors of node 1
neighbors = graph[1]  # [0, 2, 3]
Section 7

Adjacency Matrix

Adjacency Matrix: 2D array where M[i][j] = 1 if edge exists.
Space: O(V²) 0 1 2 3 0 [ 0 1 1 0 ] 1 [ 1 0 1 1 ] 2 [ 1 1 0 0 ] 3 [ 0 1 0 0 ]
Python Adjacency matrix 
matrix = [
    [0, 1, 1, 0],
    [1, 0, 1, 1],
    [1, 1, 0, 0],
    [0, 1, 0, 0]
]

# Check edge between 1 and 3
has_edge = matrix[1][3] == 1  # True
Section 8

Comparison

OPERATION | ADJ LIST | ADJ MATRIX -------------------|----------|------------ Space | O(V + E) | O(V²) Add edge | O(1) | O(1) Remove edge | O(E) | O(1) Check edge exists | O(degree)| O(1) Get all neighbors | O(degree)| O(V) Iterate all edges | O(V + E) | O(V²)
Use adjacency list for sparse graphs (most real-world graphs).
Use adjacency matrix for dense graphs or when you need fast edge lookup.
