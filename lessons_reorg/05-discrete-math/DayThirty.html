Section 1

Connectivity

Path: A sequence of non-repeating vertices connected by edges.
Cycle: A path that starts and ends at the same vertex.
Example: Triangle (A-B-C-A).
Eulerian Path: Visits every EDGE exactly once. Exists iff graph is connected and has 0 or 2 odd degree vertices.
Hamiltonian Path: Visits every VERTEX exactly once. (Hard problem!)
Section 2

Trees

A Tree is a connected graph with no cycles.

Properties of a tree with $n$ vertices:

  • Has exactly $n-1$ edges.
  • There is a unique path between any two vertices.
  • Removing any edge disconnects the graph.
  • Adding any edge creates a cycle.
Section 3

Planar Graphs

A graph is planar if it can be drawn without edges crossing.

Euler's Formula
$v - e + f = 2$

Where $f$ is the number of faces (regions bounded by edges, including the outer infinite region).

Kuratowski's Theorem: A graph is non-planar iff it contains a subgraph homeomorphic to $K_5$ or $K_{3,3}$.
Section 4

Graph Coloring

Assigning colors to vertices such that no two adjacent vertices share the same color. The Chromatic Number $\chi(G)$ is the minimum colors needed.

Four Color Theorem: Every planar graph can be colored with at most 4 colors.

Applications: Register allocation in compilers, scheduling.

Section 5

Greedy Coloring

Finding the exact chromatic number is NP-Hard. We use heuristics like greedy coloring.

Python Greedy Coloring Implementaton 
def greedy_color(graph):
    # graph is dict: vertex -> list of neighbors
    colors = {}
    
    # Order vertices (e.g., by degree descending)
    nodes = sorted(graph.keys(), key=lambda x: len(graph[x]), reverse=True)
    
    for node in nodes:
        neighbor_colors = {colors.get(neigh) for neigh in graph[node]}
        
        # Find first available color (0, 1, 2...)
        color = 0
        while color in neighbor_colors:
            color += 1
            
        colors[node] = color
        
    return colors

G = {
    'A': ['B', 'C', 'D'],
    'B': ['A', 'C'],
    'C': ['A', 'B', 'D'],
    'D': ['A', 'C']
}

coloring = greedy_color(G)
print(coloring)
