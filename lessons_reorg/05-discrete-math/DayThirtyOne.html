Section 1

Divisibility

We say $a$ divides $b$ ($a \mid b$) if there exists an integer $k$ such that $b = ak$.

Properties:

  • If $a \mid b$ and $b \mid c$, then $a \mid c$.
  • If $a \mid b$ and $a \mid c$, then $a \mid (mb + nc)$ for any integers $m, n$.
The Division Algorithm: For any integer a and b > 0, there exist unique q and r such that: a = bq + r, where 0 ≤ r < b
Section 2

Modular Arithmetic

Clock arithmetic. We say $a \equiv b \pmod m$ if $m \mid (a-b)$. Alternatively, $a$ and $b$ have the same remainder when divided by $m$.

Addition: $(a + b) \pmod m = ((a \pmod m) + (b \pmod m)) \pmod m$.
Multiplication: $(a \cdot b) \pmod m = ((a \pmod m) \cdot (b \pmod m)) \pmod m$.
Exponentiation: Fast Modular Exponentiation allows computing large powers efficiently.
Section 3

Prime Numbers

A prime $p > 1$ has only divisors 1 and $p$.
Fundamental Theorem of Arithmetic: Every integer > 1 is a unique product of primes.

There are infinitely many primes (Euclid's proof).
The Prime Number Theorem says $\pi(x) \approx x / \ln x$, where $\pi(x)$ is the count of primes $\le x$.

Section 4

Euclidean Algorithm

To find $GCD(a, b)$:
If $b=0$, return $a$.
Else, return $GCD(b, a \pmod b)$.

Bézout's Identity: There exist integers $x, y$ such that $ax + by = GCD(a, b)$. We find them using the Extended Euclidean Algorithm.
Section 5

Modular Inverse

We want to solve $ax \equiv 1 \pmod m$.
The solution exists if and only if $GCD(a, m) = 1$.
The inverse $x$ is the coefficient found by Extended Euclid.

Example: Find inverse of 3 mod 7. 3 * 5 = 15 = 2 * 7 + 1. So 3 * 5 ≡ 1 mod 7. Inverse is 5.
Section 6

RSA Encryption

Key Generation

  1. Pick two large primes $p, q$.
  2. Compute $n = pq$ and $\phi(n) = (p-1)(q-1)$.
  3. Choose public exponent $e$ such that $GCD(e, \phi(n)) = 1$ (usually 65537).
  4. Compute private key $d$ such that $ed \equiv 1 \pmod{\phi(n)}$.

Encryption / Decryption

  • Encrypt: $C = M^e \pmod n$
  • Decrypt: $M = C^d \pmod n$

Security relies on the difficulty of factoring $n$ back into $p$ and $q$.

Section 7

Python Number Ops

Python handles arbitrarily large integers, making it great for number theory.

Python Primes & Modulo 
import math

# Check prime (naive)
def is_prime(n):
    if n < 2: return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

print(f"Is 2017 prime? {is_prime(2017)}")

# Modular Exponentiation: (base^exp) % mod
# pow(base, exp, mod) is efficient O(log exp)
val = pow(5, 100000, 2017)
print(f"5^100000 mod 2017 = {val}")

# GCD
print(f"GCD(48, 18) = {math.gcd(48, 18)}")
