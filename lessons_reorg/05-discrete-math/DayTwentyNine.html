Section 1

What is a Graph?

A graph $G = (V, E)$ consists of a set of vertices $V$ and edges $E$.

V = {A, B, C, D} E = {(A,B), (B,C), (C,A)} (Triangle)
Section 2

Types of Graphs

Simple Graph: Undirected, no loops, no multiple edges between same vertices.
Directed Graph (Digraph): Edges have direction (A -> B is different from B -> A).
Complete Graph ($K_n$): Every vertex is connected to every other vertex. Edges = $n(n-1)/2$.
Bipartite Graph: Vertices can be split into two sets $V_1, V_2$ where edges only go between $V_1$ and $V_2$ (no internal connections).
Section 3

The Handshaking Lemma

In any undirected graph, the sum of degrees is equal to twice the number of edges.
$\sum_{v \in V} \deg(v) = 2|E|$

Corollary: An undirected graph has an even number of vertices of odd degree.
Try shaking hands at a party; the number of people who shook an odd number of hands is even.

Section 4

Graph Isomorphism

Two graphs $G_1$ and $G_2$ are isomorphic if there is a bijection between their vertices that preserves adjacency.

Hard Problem: Determining if two arbitrary graphs are isomorphic is computationally difficult (not known to be P or NP-complete).

Invariants (Quick Checks)

  • Same number of vertices?
  • Same number of edges?
  • Same degree sequence?

If any of these fail, they are definitely NOT isomorphic.

Section 5

Graphing in Python

We use the networkx library for graph theory.

Python Basic Graph Ops (Pseudocode/Concepts) 
import networkx as nx
import matplotlib.pyplot as plt

# Create a graph
G = nx.Graph()

# Add nodes
G.add_nodes_from([1, 2, 3, 4])

# Add edges
G.add_edge(1, 2)
G.add_edge(2, 3)
G.add_edge(3, 1)
G.add_edge(3, 4)

print(f"Nodes: {G.number_of_nodes()}")
print(f"Edges: {G.number_of_edges()}")
print(f"Degrees: {G.degree()}")

# Check Isomorphism against another graph
H = nx.Graph()
H.add_edges_from([('A','B'), ('B','C'), ('C','A'), ('C','D')])

is_iso = nx.is_isomorphic(G, H)
print(f"Isomorphic? {is_iso}") # True
