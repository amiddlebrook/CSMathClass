Section 1

The Decomposition

Every matrix $A$ (even rectangular ones) can be factored into:

SVD
$A = U \Sigma V^T$
  • $U$: Orthogonal matrix ($AA^T$ eigenvectors). Columns are "Left Singular Vectors",
  • $\Sigma$: Diagonal matrix of "Singular Values" ($\sigma_i \ge 0$). Sq roots of $A^T A$ eigenvalues.
  • $V^T$: Orthogonal matrix ($A^T A$ eigenvectors). Rows are "Right Singular Vectors".
Section 2

Geometric Interpretation

1. $V^T$ rotates space to align basis with singular vectors.
2. $\Sigma$ stretches space along axes by factors $\sigma_i$.
3. $U$ rotates the result back to the standard orientation.

Any linear transformation sends the unit sphere to a hyper-ellipse. The lengths of the ellipse axes are the singular values.
Section 3

Low Rank Approximation

Theorem (Eckart-Young): The best rank-$k$ approximation of $A$ (in Frobenius norm) is obtained by keeping only the largest $k$ singular values.

$$ A_k = \sum_{i=1}^k \sigma_i u_i v_i^T $$

This is used for image compression, data denoisng, and Latent Semantic Indexing.

Section 4

Image Compression

Images are matrices. We can compress them using SVD.

Python SVD Compression 
import numpy as np

# Create a dummy 5x5 image (gradient)
Image = np.array([
    [10, 10, 10, 0, 0],
    [10, 10, 10, 0, 0],
    [10, 10, 10, 0, 0],
    [2,  2,  2, 5, 5],
    [2,  2,  2, 5, 5]
])

# Compute SVD
U, S, Vt = np.linalg.svd(Image)
print("Singular values:", S) 
# Prob has 2 non-zero singular values because Rank is 2

# Reconstruct with Rank 1
# S is a vector, need diagonal matrix
k = 1
Recon = np.matrix(U[:, :k]) * np.diag(S[:k]) * np.matrix(Vt[:k, :])

print("\nRank 1 Approximation:")
print(np.around(Recon, 1))
