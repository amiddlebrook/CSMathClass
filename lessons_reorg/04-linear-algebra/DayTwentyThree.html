Section 1

A Functional View

A function $T: V \to W$ is a Linear Transformation if it preserves vector addition and scalar multiplication:

1. T(u + v) = T(u) + T(v) 2. T(cu) = cT(u)

Basically: "Grid lines remain parallel and evenly spaced, and the origin stays fixed."

Section 2

Matrix Representation

Every linear transformation $T$ (on finite-dimensional spaces) can be represented by a matrix $A$.

The columns of $A$ are simply the images of the basis vectors.
$A = [T(e_1) \ T(e_2) \ \dots \ T(e_n)]$

Section 3

2D Examples

Scaling: $[ k \ 0 ; 0 \ k ]$
Rotation (90 deg): $[ 0 \ -1 ; 1 \ 0 ]$
Shear: $[ 1 \ k ; 0 \ 1 ]$
Projection (x-axis): $[ 1 \ 0 ; 0 \ 0 ]$
Section 4

Change of Basis

If we look at the same transformation in a different basis $B$, the matrix changes to: $$ M = P^{-1} A P $$ Where $P$ is the change-of-basis matrix (columns are new basis vectors). Matrices $M$ and $A$ are Similar.

Section 5

Visualizing Transforms

We can apply a matrix to a set of points to see the transformation.

Python Applying Transformation 
import numpy as np

# Unit Square corners
points = np.array([
    [0, 0],
    [1, 0],
    [1, 1],
    [0, 1]
]).T # 2x4 matrix

# Shear Matrix
k = 1.5
Shear = np.array([
    [1, k],
    [0, 1]
])

# Transform
transformed_points = Shear @ points

print("Original (columns):\n", points)
print("Transformed:\n", transformed_points)
