Section 1

The Determinant

The Determinant ($\det(A)$ or $|A|$) is a scalar that describes how a linear transformation changes volume.

If $|A| = 0$, the matrix squashes space into a lower dimension (Singular). Volume becomes 0.
If $|A| < 0$, the matrix flips orientation (like a mirror).
$|AB| = |A||B|$ and $|A^{-1}| = 1/|A|$.
Section 2

Eigenvalues & Eigenvectors

An eigenvector $v$ is a non-zero vector that, when transformed by $A$, only gets scaled by a factor $\lambda$ (the eigenvalue). $$ A v = \lambda v $$

To find them:
1. Solve $\det(A - \lambda I) = 0$ for $\lambda$ (characteristic equation).
2. For each $\lambda$, solve $(A - \lambda I)v = 0$ for $v$.

Section 3

Diagonalization

If a square matrix $A$ has $n$ linearly independent eigenvectors, we can decompose it: $$ A = S \Lambda S^{-1} $$ Where $S$ has eigenvectors as columns, and $\Lambda$ is a diagonal matrix of eigenvalues.

This makes computing powers easy: $A^k = S \Lambda^k S^{-1}$.

Section 4

Applications

  • Google PageRank: The steady state of a Markov Chain is an eigenvector with eigenvalue 1.
  • PCA (Principal Component Analysis): Used in ML for dimensionality reduction. Uses eigenvectors of the covariance matrix.
  • Vibrations/Physics: Eigenvalues represent resonant frequencies.
Section 5

Python Eigendecomposition

Python Finding Eigenvalues 
import numpy as np

# Matrix A
A = np.array([
    [4, -2],
    [1, 1]
])

# Calculate eigenvalues and eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(A)

print("Eigenvalues:", eigenvalues)
print("Eigenvectors (columns):\n", eigenvectors)

# Verify: Av = λv for first eigenpair
lambda_1 = eigenvalues[0]
v_1 = eigenvectors[:, 0]

Av = A @ v_1
lambda_v = lambda_1 * v_1

print(f"\nA @ v: {Av}")
print(f"λ * v: {lambda_v}")
# They should match!
