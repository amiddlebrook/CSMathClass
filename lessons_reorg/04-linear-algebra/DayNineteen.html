Section 1

Matrix Multiplication

For matrices $A$ (m×n) and $B$ (n×p), the product $C = AB$ is m×p:

$$ C_{ij} = \sum_{k=1}^{n} A_{ik} B_{kj} $$ (Row $i$ of A) · (Column $j$ of B)
Not commutative! $AB \neq BA$ in general.
[1 2] × [5 6] = [1×5+2×7 1×6+2×8] = [19 22] [3 4] [7 8] [3×5+4×7 3×6+4×8] [43 50]
Section 2

Systems of Linear Equations

A system $A\mathbf{x} = \mathbf{b}$ represents multiple equations:

2x + 3y = 8 [2 3] [x] [8] x - y = 1 → [1 -1][y] = [1]
Interpretation: Find the vector $\mathbf{x}$ that when multiplied by $A$ gives $\mathbf{b}$.
Section 3

Gaussian Elimination

Transform the augmented matrix to row echelon form using:

  1. Swap rows
  2. Multiply row by non-zero scalar
  3. Add multiple of one row to another
Goal: Upper triangular form, then back-substitute.
[2 3 | 8] → [2 3 | 8] → [2 3 | 8 ] [1 -1 | 1] [0 -5 |-6] [0 -5 |-6 ] Back-sub: -5y = -6 → y = 1.2 → 2x + 3(1.2) = 8 → x = 2.2
Section 4

Solving with NumPy

NumPy's linalg.solve handles this efficiently:

PythonSolving Ax = b
import numpy as np

# System: 2x + 3y = 8, x - y = 1
A = np.array([[2, 3], [1, -1]])
b = np.array([8, 1])

x = np.linalg.solve(A, b)
print(f"Solution: x = {x[0]:.2f}, y = {x[1]:.2f}")

# Verify
print("Check:", A @ x)  # Should be [8, 1]

# Matrix multiplication
C = np.array([[1, 2], [3, 4]])
D = np.array([[5, 6], [7, 8]])
print("C @ D =", C @ D)  # [[19 22], [43 50]]
Section 5

Determinants

The determinant is a scalar that encodes whether a matrix is invertible.

2×2 Determinant: $$ \det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc $$
det([3 1]) = 3×4 - 1×2 = 12 - 2 = 10 ([2 4])
Geometric meaning: |det(A)| = area of parallelogram formed by column vectors.
Section 6

Determinant Properties

• det(AB) = det(A) × det(B) • det(A^T) = det(A) • det(A⁻¹) = 1/det(A) • det(cA) = c^n × det(A) for n×n matrix • det(I) = 1
Key test: A is invertible ⟺ det(A) ≠ 0
Section 7

Matrix Inverse

If det(A) ≠ 0, then $A^{-1}$ exists such that $AA^{-1} = A^{-1}A = I$.

2×2 Inverse: $$ A^{-1} = \frac{1}{ad-bc}\begin{pmatrix} d & -b \\ -c & a \end{pmatrix} $$

For larger matrices, use Gauss-Jordan elimination or NumPy.

Section 8

NumPy det & inv

PythonDeterminants & Inverses
import numpy as np

A = np.array([[3, 1], [2, 4]])

# Determinant
det_A = np.linalg.det(A)
print(f"det(A) = {det_A:.0f}")  # 10

# Inverse
A_inv = np.linalg.inv(A)
print("A⁻¹ =", A_inv)

# Verify: A @ A⁻¹ = I
print("A @ A⁻¹ =", A @ A_inv)
