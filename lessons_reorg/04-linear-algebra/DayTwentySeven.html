Section 1

The Basics

Product Rule (AND)

If task A has $m$ ways and task B has $n$ ways, doing A AND B has $m \times n$ ways.

Sum Rule (OR)

If task A has $m$ ways and task B has $n$ ways (and they are disjoint), doing A OR B has $m + n$ ways.

Section 2

Permutations (Order Matters)

An arrangement of $n$ distinct items.

P(n, k) = \frac{n!}{(n-k)!}

Example: Ordering 3 people (A, B, C) in a line.
$3! = 3 \times 2 \times 1 = 6$ ways (ABC, ACB, BAC, BCA, CAB, CBA).

Section 3

Combinations (Order Doesn't Matter)

Choosing $k$ items from $n$, where order is irrelevant.

Binomial Coefficient
$C(n, k) = \binom{n}{k} = \frac{n!}{k!(n-k)!}$

Example: Picking a committee of 2 from 3 people (A, B, C).
{A,B}, {A,C}, {B,C} $\implies 3$ ways.
$\binom{3}{2} = \frac{3!}{2!1!} = \frac{6}{2} = 3$.

Section 4

Stars and Bars

The number of ways to distribute $k$ identical items into $n$ distinct bins is:
$\binom{n+k-1}{k}$

Intuition: We arrange $k$ stars (items) and $n-1$ bars (separators) in a line.

Section 5

Python Itertools

Python's standard library `itertools` does exactly this.

Python Permutations & Combinations 
import itertools

items = ['A', 'B', 'C']

print("Permutations of 2 items (Order matters):")
perms = list(itertools.permutations(items, 2))
for p in perms:
    print(p)
# ('A', 'B'), ('A', 'C'), ('B', 'A'), ... (6 total)

print("\nCombinations of 2 items (Order irrelevant):")
combs = list(itertools.combinations(items, 2))
for c in combs:
    print(c)
# ('A', 'B'), ('A', 'C'), ('B', 'C') (3 total)

# Calculate without listing
import math
print(f"\ncalculated 10C3: {math.comb(10, 3)}")
