Section 1

The Dot Product

The dot product (or inner product) is defined as: $$ \vec{u} \cdot \vec{v} = u^T v = \sum u_i v_i = ||u|| ||v|| \cos \theta $$

Length (Norm): $||v|| = \sqrt{v \cdot v}$
Angle: $\cos \theta = \frac{u \cdot v}{||u|| ||v||}$
Section 2

Orthogonality

Two vectors are Orthogonal if their dot product is 0.
$\vec{u} \perp \vec{v} \iff \vec{u} \cdot \vec{v} = 0$.

Orthonormal Basis: A basis where all vectors are unit length and mutually orthogonal. This is the "nicest" kind of basis (like the standard $i, j, k$).

Gram-Schmidt Process: An algorithm to convert any basis into an orthonormal basis.
Section 3

Projections

Projecting a vector $b$ on to a line $a$: $$ p = \text{proj}_a b = \frac{a \cdot b}{a \cdot a} a $$

More generally, projecting onto a subspace (like a plane) finds the point in the subspace closest to $b$. The error vector $e = b - p$ is orthogonal to the subspace.

Section 4

Least Squares

When $Ax=b$ has no solution (often because there are more equations than variables), we solve for the "best" approximate solution $\hat{x}$.

Normal Equation
$A^T A \hat{x} = A^T b$

This minimizes the squared error $||Ax - b||^2$. This is the basis of Linear Regression.

Section 5

Python Regression

Using Least Squares to fit a line to data points.

Python Linear Regression 
import numpy as np

# Data points (x, y)
X = np.array([1, 2, 3, 4])
Y = np.array([6, 5, 7, 10])

# We want y = mx + c
# System:
# 1*m + 1*c = 6
# 2*m + 1*c = 5 ...

# Construct Matrix A (columns: x values, then 1s)
A = np.vstack([X, np.ones(len(X))]).T
print("Matrix A:\n", A)

# Solve Normal Equation directly: A.T @ A @ x = A.T @ Y
# Or use numpy's least squares
m, c = np.linalg.lstsq(A, Y, rcond=None)[0]

print(f"Best fit line: y = {m:.2f}x + {c:.2f}")
# Checks out?
# x=1 -> y ~ 6
# x=4 -> y ~ 10
