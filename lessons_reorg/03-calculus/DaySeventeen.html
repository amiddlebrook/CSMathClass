Why Multivariable Calculus Matters

ML runs on gradients. Every neural network training loop computes gradients to update weights. Understanding them deeply is essential.

Where you'll use this

  • Machine Learning: Gradient descent, backpropagation
  • Computer Graphics: Surface normals, texture mapping
  • Physics Simulation: Force fields, fluid dynamics
  • Optimization: Finding minima/maxima of cost functions
  • Robotics: Jacobians for motion planning
Section 1

Partial Derivatives

Definition

Partial Derivative
∂f/∂x = lim[h→0] (f(x+h, y) - f(x, y)) / h

Differentiate with respect to one variable, treating others as constants.

Notation

∂f/∂x or fₓ or ∂ₓf or f_x For f(x, y) = x²y + 3xy²: ∂f/∂x = 2xy + 3y² (treat y as constant) ∂f/∂y = x² + 6xy (treat x as constant)

Geometric interpretation

For a surface z = f(x, y): ∂f/∂x = slope in the x-direction (y held constant) ∂f/∂y = slope in the y-direction (x held constant) Like slicing the surface with planes parallel to axes.

In code

PythonNumerical partial derivative
import numpy as np

def partial_x(f, x, y, h=1e-5):
    return (f(x + h, y) - f(x - h, y)) / (2 * h)

def partial_y(f, x, y, h=1e-5):
    return (f(x, y + h) - f(x, y - h)) / (2 * h)

# Example: f(x,y) = x²y
f = lambda x, y: x**2 * y
partial_x(f, 2, 3)  # ≈ 12 (exact: 2xy = 2*2*3 = 12)
Section 2

The Gradient ∇f

Definition

Gradient
∇f = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)

The gradient is a vector of all partial derivatives. It points in the direction of steepest increase.

Key properties

1. ∇f points toward steepest ascent 2. -∇f points toward steepest descent 3. |∇f| = rate of maximum change 4. ∇f ⊥ level curves (contour lines)

Gradient descent

PythonBasic gradient descent
def gradient_descent(f, grad_f, x0, lr=0.01, steps=100):
    """Minimize f starting from x0"""
    x = x0
    for _ in range(steps):
        x = x - lr * grad_f(x)  # move opposite to gradient
    return x

# Example: minimize f(x,y) = x² + y²
grad_f = lambda p: np.array([2*p[0], 2*p[1]])
gradient_descent(f, grad_f, np.array([5.0, 3.0]))
# → approaches [0, 0] (the minimum)

Directional derivative

Rate of change in direction u (unit vector): D_u f = ∇f · u = |∇f| cos(θ) Maximum when u parallel to ∇f (θ = 0) Zero when u ⊥ ∇f (along level curve)
Section 3

Higher-Order Derivatives & Hessian

Second-order partials

f_xx = ∂²f/∂x² = ∂/∂x(∂f/∂x) f_yy = ∂²f/∂y² f_xy = ∂²f/∂x∂y = ∂/∂x(∂f/∂y) f_yx = ∂²f/∂y∂x = ∂/∂y(∂f/∂x) Clairaut's Theorem: If mixed partials are continuous, f_xy = f_yx (order doesn't matter)

The Hessian matrix

Hessian
H = [f_xx f_xy] [f_yx f_yy]
The Hessian describes curvature: • H positive definite → local minimum • H negative definite → local maximum • H indefinite → saddle point Used in Newton's method: x ← x - H⁻¹∇f
Section 4

The Jacobian Matrix

Definition

For a function F: ℝⁿ → ℝᵐ with components (f₁, f₂, ..., fₘ):

Jacobian Matrix
J = [∂f₁/∂x₁ ∂f₁/∂x₂ ... ∂f₁/∂xₙ] [∂f₂/∂x₁ ∂f₂/∂x₂ ... ∂f₂/∂xₙ] [ ⋮ ⋮ ⋱ ⋮ ] [∂fₘ/∂x₁ ∂fₘ/∂x₂ ... ∂fₘ/∂xₙ]

Example: Polar coordinates

x = r cos(θ), y = r sin(θ) J = [∂x/∂r ∂x/∂θ] = [cos(θ) -r sin(θ)] [∂y/∂r ∂y/∂θ] [sin(θ) r cos(θ)] det(J) = r cos²(θ) + r sin²(θ) = r This is why ∫∫ dxdy = ∫∫ r dr dθ

Jacobian determinant

Key insight: |det(J)| measures how the transformation scales area/volume. Used in change of variables for integration.
Section 5

Multivariable Chain Rule

The formula

If z = f(x, y) where x = g(t), y = h(t): dz/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt) Matrix form: dz/dt = [∂f/∂x ∂f/∂y] · [dx/dt] [dy/dt] = ∇f · v where v = (dx/dt, dy/dt)

Backpropagation connection

PythonChain rule in neural networks
# Forward: z = f(g(x))
# Backward: ∂L/∂x = (∂L/∂z)(∂z/∂g)(∂g/∂x)

# Each layer passes gradient back via chain rule
def backward(dL_dz, local_grad):
    return dL_dz * local_grad  # chain rule!

# For many inputs: use Jacobian
# dL/dx = Jᵀ · dL/dz
