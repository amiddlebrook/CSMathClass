Section 1

The Heat Equation

The prototype parabolic PDE:

\frac{\partial u}{\partial t} = \Delta u

It describes diffusion. In 1D: `u_t = u_{xx}`.

Section 2

Smoothing

The heat equation instantly smooths rough initial data. If `u(x, 0)` is merely `L^2`, then `u(x, t)` is `C^\infty` for all `t > 0`.

Geometric Insight: Heat flow tries to minimize "energy" (gradient) by flattening the curve.
Section 3

Ricci Flow

Heat equation for the metric tensor `g_{ij}` of a manifold.

\frac{\partial g_{ij}}{\partial t} = -2 R_{ij}

Here `R_{ij}` is the Ricci curvature. Positive curvature areas shrink; negative areas expand. It tries to make the curvature constant.

Section 4

Poincaré Conjecture

"Every simply connected, closed 3-manifold is homeomorphic to the 3-sphere."

Perelman proved this by running Ricci Flow. He showed that any 3-manifold evolves into a set of recognizable pieces (Geometrization). He famously dealt with singularities (pinched necks) by surgery.

Section 5

Python: Heat Equation Clean-Up

Simulating diffusion on a noisy signal.

Python 1D Heat Equation 
import numpy as np
import matplotlib.pyplot as plt

# 1. Setup Grid
N = 100
dx = 1.0 / N
dt = 0.0001 # Stability condition: dt/dx^2 < 0.5
steps = 500

x = np.linspace(0, 1, N)
u = np.sin(2 * np.pi * x) + 0.5 * np.random.rand(N) # Noisy sine
u_start = u.copy()

# 2. Iterate (Finite Difference)
for k in range(steps):
    # u_xx approx (u[i+1] - 2u[i] + u[i-1]) / dx^2
    u_xx = (np.roll(u, -1) - 2*u + np.roll(u, 1)) / dx**2
    u = u + dt * u_xx

# Plot
plt.figure(figsize=(10, 5))
plt.plot(x, u_start, color='gray', alpha=0.5, label='Initial (Noisy)')
plt.plot(x, u, color='red', lw=2, label=f'Solved (t={steps*dt})')
plt.title("Heat Equation: Smoothing Data")
plt.legend()
plt.show()
