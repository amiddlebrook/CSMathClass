Section 1

The Group

A Group is the most fundamental algebraic structure. It captures the notion of "symmetry" and "reversible actions."

Definition: A group is a set $G$ combined with an operation $\cdot$ satisfying:
  1. Closure: For all $a,b \in G, a \cdot b \in G$.
  2. Associativity: $(a \cdot b) \cdot c = a \cdot (b \cdot c)$.
  3. Identity: There exists $e \in G$ such that $a \cdot e = a$.
  4. Inverses: For every $a$, there exists $a^{-1}$ such that $a \cdot a^{-1} = e$.

Examples

Integers under Addition ($\mathbb{Z}, +$):
Identity is 0. Inverse of $x$ is $-x$. This is an infinite group.
Non-zero Reals under Multiplication ($\mathbb{R}^*, \times$):
Identity is 1. Inverse of $x$ is $1/x$.
General Linear Group ($GL_n(\mathbb{R})$):
Invertible $n \times n$ matrices. Operation is matrix multiplication. (Non-abelian: $AB \neq BA$).
Modular Arithmetic ($\mathbb{Z}_n$):
Integers modulo $n$. $\mathbb{Z}_5 = \{0, 1, 2, 3, 4\}$.
Section 2

Subgroups & Lagrange

Structures within Structures

A subset $H \subseteq G$ is a subgroup if it is a group in its own right under the same operation.

Example: Even integers (2$\mathbb{Z}$) are a subgroup of integers ($\mathbb{Z}$). Closed under addition? Yes (Even + Even = Even). Has identity? Yes (0). Has inverses? Yes (-Even = Even).
Lagrange's Theorem: If $G$ is a finite group and $H$ is a subgroup, then the size of $H$ divides the size of $G$. $$|G| = k \cdot |H|$$
Section 3

Rings

A Ring has two operations, usually called addition ($+$) and multiplication ($\cdot$).

Properties

  • Under ($+$), it is an Abelian (commutative) Group. Identity is $0$.
  • Under ($\cdot$), it is associative and has an identity $1$.
  • Distributive Laws hold: $a(b+c) = ab + ac$.
Examples: 1. Integers ($\mathbb{Z}$): + and * work as expected. 2. Polynomials ($R[x]$): Polynomials with coefficients in a ring. 3. Matrices ($M_n(\mathbb{R})$): Matrix addition and multiplication.
Section 4

Symbolic Algebra with Python

Python Exploring Modular Groups 
import math

class ZGroup:
    """Additive Group of Integers Modulo n"""
    def __init__(self, n):
        self.n = n
        self.elements = list(range(n))

    def add(self, a, b):
        return (a + b) % self.n

    def inverse(self, a):
        return (self.n - a) % self.n

    def is_subgroup(self, subset):
        if 0 not in subset: return False
        for x in subset:
            if self.inverse(x) not in subset: return False
            for y in subset:
                if self.add(x, y) not in subset: return False
        return True

# Example: Z_6
g = ZGroup(6)
print(f"Elements: {g.elements}")
print(f"2 + 5 = {g.add(2, 5)}") # 1

# Testing subgroups
h1 = {0, 2, 4}
h2 = {0, 3}
h3 = {0, 1, 2}

print(f"Is {h1} a subgroup? {g.is_subgroup(h1)}") # True
print(f"Is {h3} a subgroup? {g.is_subgroup(h3)}") # False (1+2=3, not in set)
