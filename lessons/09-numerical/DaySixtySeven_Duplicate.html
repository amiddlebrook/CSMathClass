Section 1

Test Functions

Standard functions map numbers to numbers. Distributions map functions to numbers.

We start with the space of Test Functions `\mathcal{D}(\Omega) = C_c^\infty(\Omega)`.

These are infinitely smooth functions with compact support (bumps). They are very "nice" and "safe".

Section 2

Definition of Distribution

Definition: A Distribution `T` is a continuous linear functional on `\mathcal{D}`.
`T: C_c^\infty \to \mathbb{R}`

Notation: We often write `\langle T, \phi \rangle` for `T(\phi)`.

Every locally integrable function `f` is a distribution via `\langle f, \phi \rangle = \int f(x)\phi(x) dx`.

Section 3

Dirac Delta

The delta "function" `\delta(x)` is 0 everywhere except `x=0`, and integral 1. This makes no sense as a function.

As a distribution, it is perfectly rigorous:

\langle \delta, \phi \rangle = \phi(0)

It simply evaluates the test function at the origin. It is a "sampling" distribution.

Section 4

Distributional Derivative

In Calculus, `|x|` is not differentiable at 0. In Distributions, everything is differentiable infinitely many times.

We define `T'` by Integration by Parts:

\langle T', \phi \rangle = - \langle T, \phi' \rangle

Since `\phi` is smooth, `\phi'` always exists. So `T'` always exists.

Example: Let `H(x)` be the Heaviside step function. `\langle H', \phi \rangle = - \int H \phi' = - \int_0^\infty \phi' = -(\phi(\infty) - \phi(0)) = \phi(0)`. So `H' = \delta`.

Section 5

Sobolev Spaces

PDEs are solved in Sobolev Spaces `W^{k,p}`.

`W^{1,2}` (or `H^1`) consists of functions in `L^2` whose distributional derivatives are also in `L^2`.

This allows "kinks" (like triangle waves) but rules out jumps (like step functions, whose derivative is a delta, not in `L^2`).

Section 6

Python: Approximating Delta

Python Diff of Sigmoid 
import numpy as np
import matplotlib.pyplot as plt

# The Heaviside step function H(x) is the limit of Sigmoid functions.
# The Delta function d(x) is the limit of the derivatives of Sigmoids.

x = np.linspace(-3, 3, 500)

def sigmoid(x, k):
    return 1 / (1 + np.exp(-k * x))

def sigmoid_prime(x, k):
    s = sigmoid(x, k)
    return k * s * (1 - s)

ks = [1, 5, 20, 100]

plt.figure(figsize=(10, 8))

# Plot Converging Steps
plt.subplot(2, 1, 1)
for k in ks:
    plt.plot(x, sigmoid(x, k), label=f'k={k}')
plt.title("Approximation of Heaviside H(x)")
plt.legend()
plt.grid(True)

# Plot Converging Deltas
plt.subplot(2, 1, 2)
for k in ks:
    plt.plot(x, sigmoid_prime(x, k), label=f'k={k}')
plt.title("Approximation of Delta d(x) = H'(x)")
plt.legend()
plt.grid(True)

plt.show()

# Observe: Area under all bell curves is exactly 1.
# As k -> infinity, width -> 0, height -> infinity.
