Section 1

Measuring Information

Information is resolution of uncertainty. A coin flip gives 1 bit. A die roll gives `log₂ 6 ≈ 2.58` bits.

I(p) = -log₂(p)
Section 2

Shannon Entropy

Entropy H(X) is the average information content.

H(X) = -∑ p(x) log₂ p(x)

It represents the fundamental limit of data compression (lossless).

Section 3

Source Coding Theorem

Theorem: N i.i.d. random variables each with entropy H(X) can be compressed into slightly more than N·H(X) bits with negligible risk of information loss.

Huffman coding achieves this optimum.

Section 4

KL Divergence

How do we measure the "distance" between two distributions P and Q?

D_KL(P || Q) = ∑ P(x) log (P(x) / Q(x))

It's essentially "how many EXTRA bits do I need to code P if I use a code optimized for Q?".

Section 5

Python: Calculating Entropy

Python Entropy Calc 
import numpy as np

def entropy(probs):
    probs = np.array(probs)
    # Avoid log(0)
    probs = probs[probs > 0]
    return -np.sum(probs * np.log2(probs))

print(entropy([0.5, 0.5]))      # 1.0 bit
print(entropy([0.9, 0.1]))      # 0.47 bits (less surprise)
print(entropy([0.25]*4))        # 2.0 bits
