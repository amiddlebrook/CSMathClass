Section 1

The Quantum Threat

Shor's Algorithm can factor large numbers and compute discrete logs in Polynomial Time.

If a large quantum computer is built, RSA and ECC are dead. AES is mostly fine (just need double key size).
Section 2

Lattice Cryptography

Lattices are grids of points in high-dimensional space. The Shortest Vector Problem (SVP) is to find the closest grid point to the origin.

It turns out SVP is hard even for quantum computers.

Section 3

Learning With Errors (LWE)

Solving a system of linear equations is easy (Gaussian Elimination). But if we add a tiny bit of "noise" or error to each equation, it becomes incredibly hard.

A*s + e ≈ b Find s.
Section 4

Homomorphic Encryption

The Holy Grail of crypto: Processing data while it is still encrypted.

Enc(x) + Enc(y) = Enc(x+y) Enc(x) * Enc(y) = Enc(x*y)

This allows cloud computing on private data without the cloud provider seeing the data. Lattices make this possible.
