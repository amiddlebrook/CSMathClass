Section 1

Symmetric vs Public Key

Symmetric: Alice and Bob share a secret key. (AES, One-Time Pad). Fast.

Public Key (Asymmetric): Alice has a Public Key (Lock) and Private Key (Key). Anyone can lock a message, only Alice can open it.

Section 2

RSA

Security relies on the difficulty of factoring large numbers.

1. Pick primes p, q. N = p*q. 2. Compute φ(N) = (p-1)(q-1). 3. Pick e (public exponent). 4. Compute d (private key) s.t. d*e ≡ 1 mod φ(N).

Encrypt: `c = m^e mod N`

Decrypt: `m = c^d mod N`

Section 3

Why it works

Euler's Theorem: `m^{φ(N)} ≡ 1 mod N`.

So `c^d = (m^e)^d = m^{ed} = m^{kφ(N) + 1} = (m^{φ(N)})^k * m = 1^k * m = m`. Magic!

Section 5

Python: Modular Exponentiation

Python Square and Multiply 
def mod_exp(base, exp, mod):
    res = 1
    base %= mod
    while exp > 0:
        if exp % 2 == 1:
            res = (res * base) % mod
        base = (base * base) % mod
        exp //= 2
    return res

# RSA Example
p, q = 61, 53
n = p*q
phi = (p-1)*(q-1)
e = 17
d = pow(e, -1, phi) # Modular inverse

msg = 65
enc = mod_exp(msg, e, n)
dec = mod_exp(enc, d, n)

print(f"Original: {msg}, Decrypted: {dec}")
