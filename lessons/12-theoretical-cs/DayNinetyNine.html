Section 1

The Qubit

A classical bit is 0 or 1. A Qubit is a unit vector in a 2D complex vector space.

|\psi\rangle = \alpha |0\rangle + \beta |1\rangle

Where `\alpha, \beta \in \mathbb{C}` and `|\alpha|^2 + |\beta|^2 = 1`.

Measurement forces the qubit to collapse: to `0` with prob `|\alpha|^2` and `1` with prob `|\beta|^2`.

Section 2

Superposition

While the qubit is unmeasured, it holds both states simultaneously with complex amplitudes.

This interference allows negative probabilities (amplitudes) to cancel out wrong answers during computation.

Section 3

Quantum Gates

Gates are Unitary Matrices (preserve length/probability).

  • X Gate (NOT): Flips 0 and 1.
  • H Gate (Hadamard): Puts a definite state into a 50/50 superposition. `H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)`.
Section 4

Multiple Qubits

Two qubits exist in a 4D space spanned by `|00\rangle, |01\rangle, |10\rangle, |11\rangle`.

`n` qubits exist in a `2^n` dimensional space. Nature does exponential work for us.

The state is the Tensor Product: `|\psi\rangle \otimes |\phi\rangle`.

Section 5

Entanglement

A Bell State: `\frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)`.

You cannot write this as `(\alpha|0\rangle + \beta|1\rangle) \otimes (\gamma|0\rangle + \delta|1\rangle)`.

Measuring the first qubit immediately determines the second, regardless of distance ("Spooky action at a distance").

Section 6

Python: Qubit Simulation

Python Simulating H|0> 
import numpy as np

# Basis States
zero = np.array([1, 0])
one  = np.array([0, 1])

# Hadamard Gate
H = (1/np.sqrt(2)) * np.array([
    [1,  1],
    [1, -1]
])

# Pauli X Gate
X = np.array([
    [0, 1],
    [1, 0]
])

# Apply H to |0>
psi = np.dot(H, zero)
print(f"State H|0>: {psi}") # [0.707, 0.707]

# Probabilities
prob_0 = abs(psi[0])**2
prob_1 = abs(psi[1])**2
print(f"Prob(0): {prob_0:.2f}, Prob(1): {prob_1:.2f}")

# Measurement Simulation
measured = np.random.choice([0, 1], p=[prob_0, prob_1])
print(f"Measured: {measured}")
