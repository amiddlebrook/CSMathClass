Section 1

Turing Machines

A Turing Machine (TM) is an infinite tape with a read/write head and a finite controller.

• Can move Left or Right. • Can read and write symbols. • Has infinite memory.

Despite its simplicity, it can compute anything recognizable by any algorithm.

Section 2

Church-Turing Thesis

The Thesis: "Anything obtainable by a 'mechanical' method of calculation is computable by a Turing Machine."

This means your Python code, C++, and supercomputers are all just shiny Turing Machines.

Section 3

The Halting Problem

Is there a program `H` that takes code `P` and input `x`, and tells you if `P(x)` halts?

Assume H exists. Construct D(P): if H(P, P) says "halts", then D loops forever. if H(P, P) says "loops", then D halts. Run D(D)... Contradiction!
Conclusion: The Halting Problem is Undecidable. There are things computers CANNOT do.
Section 4

Rice's Theorem

It gets worse.

Rice's Theorem: Any non-trivial semantic property of a Turing Machine's language is undecidable.

You cannot write a static analyzer that perfectly checks "Does this code ever output a 5?" or "is this code bug free?".
