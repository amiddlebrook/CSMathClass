Section 1

Context-Free Grammars (CFG)

Regular languages cannot handle nested structures like `((()))`. CFGs can.

A CFG consists of variables, terminals, and rules like `S \to A | B`.

Example: `S \to 0S1 | \epsilon` generates `\{0^n 1^n\}`.

"Context-Free" means we can replace a variable `A` regardless of what surrounds it.

Section 2

Pushdown Automata (PDA)

Theorem: A language is Context-Free `\iff` it is recognized by a Non-deterministic PDA.

A PDA is an NFA with a Stack (LIFO memory).

Important: Deterministic PDAs (DPDA) are strictly weaker than Non-deterministic PDAs. This is unlike Finite Automata where DFA=NFA. Programming languages usually target DPDAs for efficient parsing.

Section 3

Ambiguity

A grammar is ambiguous if a string has multiple distinct parse trees.

Example: `E \to E+E | E*E | a`. The string `a+a*a` can be parsed as `(a+a)*a` or `a+(a*a)`.

We fix this in compilers using operator precedence. Some languages are inherently ambiguous.

Section 4

Chomsky Normal Form

Any CFG can be simplified to rules of the form:

  • `A \to BC` (Two variables)
  • `A \to a` (One terminal)

This canonical form enables the CYK Algorithm, which checks if a string is in the grammar in `O(n^3)` time.

Section 5

Pumping Lemma for CFLs

If a parse tree is deep enough, a path must repeat a variable `R`. We can "pump" the subtree under `R`.

w = uvxyz, \quad |vxy| \le p, \quad |vy| > 0, \quad uv^i xy^i z \in L

Used to prove languages like `\{0^n 1^n 2^n\}` are NOT Context-Free.

Section 6

Python: Recursive Descent (Top-Down Parsing)

Python Simple Parser 
class SimpleParser:
    """
    Parses S -> 'a' S 'b' | epsilon
    Language: a^n b^n
    """
    def __init__(self, token_stream):
        self.tokens = list(token_stream)
        self.pos = 0

    def peek(self):
        return self.tokens[self.pos] if self.pos < len(self.tokens) else None

    def consume(self, expected):
        if self.peek() == expected:
            self.pos += 1
            return True
        return False

    def parse_S(self):
        # S -> 'a' S 'b'
        if self.consume('a'):
            if self.parse_S():
                if self.consume('b'):
                    return True
                else:
                    return False # Expected 'b'
            return False
        # S -> epsilon (valid if we don't see 'a')
        return True

    def parse(self):
        # Must match S and reach End of Stream
        valid = self.parse_S()
        return valid and self.pos == len(self.tokens)

# Test
print(f"aabb: {SimpleParser('aabb').parse()}") # True
print(f"aabbb: {SimpleParser('aabbb').parse()}") # False (leftover b)
print(f"ba: {SimpleParser('ba').parse()}") # False (starts with b)
