Section 1

Quantum Parallelism

If we apply a function `f` to a superposition of all inputs `\sum |x\rangle`, we get a superposition of all outputs `\sum |x, f(x)\rangle`.

However, we can only measure once. The art of quantum algorithms is to make the wrong answers cancel out using Interference.

Section 2

Deutsch-Jozsa Algorithm

Problem: Is `f(x)` constant (all 0 or all 1) or balanced (half 0, half 1)?

Classical: Requires looking at half the inputs. `O(2^{n-1})`.

Quantum: Solves it in 1 query. It was the first proof of quantum speedup.

Section 3

Grover's Algorithm

Problem: Search an unsorted database of `N` items.

Classical: `O(N)`.

Quantum: `O(\sqrt{N})`. This is a quadratic speedup. It implies brute-forcing AES-256 effectively becomes AES-128.

Section 4

Shor's Algorithm

Problem: Integer Factorization (breaking RSA).

Classical: Exponential time (General Number Field Sieve).

Quantum: Polynomial time! It uses the Quantum Fourier Transform (QFT) to find the period of a function `f(x) = a^x \pmod N`.

Section 5

Complexity Class BQP

BQP (Bounded-Error Quantum Polynomial Time) is the class of problems quantum computers can solve.

`P \subseteq BQP \subseteq PSPACE`.

We suspect `NP \not\subseteq BQP`. So Quantum Computers probably can't solve NP-Complete problems efficiently (like TSP), but they can factor integers (which is likely in `NP` but not `NP-complete`).

Section 6

Python: Period Finding

Python Shor's Insight (Classical) 
import math

# Shor's algorithm depends on finding the period 'r'
# of f(x) = a^x mod N.
# If we find r, we can likely factor N.

def find_period_classical(a, N):
    # This part is slow classically!
    # Quantum Computers do this in O(poly) using QFT.
    r = 1
    while pow(a, r, N) != 1:
        r += 1
    return r

def shors_factorization_toy(N):
    # 1. Pick random a < N
    a = 2 # Simplified for example
    
    # Check if we got lucky
    k = math.gcd(a, N)
    if k > 1: return k, N//k
    
    # 2. Find period r
    r = find_period_classical(a, N)
    
    if r % 2 != 0: return "Failed (Odd period)"
    
    # 3. Factors are gcd(a^(r/2) ± 1, N)
    val = pow(a, r//2, N)
    p = math.gcd(val - 1, N)
    q = math.gcd(val + 1, N)
    return p, q

print(f"Factors of 15: {shors_factorization_toy(15)}")
