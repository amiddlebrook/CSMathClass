Section 1

Newton's Method

Standard GD uses 1st derivative. Newton uses 2nd derivative (Hessian `H`).

x_{k+1} = x_k - H^{-1} ∇f(x_k)

It converges quadratically (extremely fast) but inverting `H` is expensive ($O(n^3)$).

Section 2

Subgradients

What if `f` is not differentiable (e.g. `|x|`)?

We use a subgradient `g` such that `f(y) ≥ f(x) + g^T (y-x)`.

Subgradient method is slow ($O(1/√k)$) but works for non-smooth functions.
Section 3

Proximal Gradient

Ideally for `f(x) + g(x)` where `f` is smooth and `g` is not (like L1 norm).

x_{k+1} = prox_{αg} (x_k - α ∇f(x_k))

For L1 norm ($g(x) = λ|x|$), the prox operator is Soft Thresholding.

Section 4

Momentum

Nesterov Acceleration adds "momentum" to jump over small bumps and move faster along flat valleys.

Momentum turns $O(1/k)$ convergence into $O(1/k^2)$.
Section 5

Python: Comparisons

Python Newton vs GD 
import numpy as np

# Minimize x^4
def f(x): return x**4
def grad(x): return 4*x**3
def hess(x): return 12*x**2

x = 2.0
# Newton Step
x_newton = x - grad(x)/hess(x)
print(f"After 1 step Newton: {x_newton}")

# GD Step (rate 0.1)
x_gd = x - 0.1 * grad(x)
print(f"After 1 step GD: {x_gd}")
