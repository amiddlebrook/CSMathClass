Section 0

Why Integer Math Matters

Reality: Computers fundamentally operate on integers. Array indices, memory addresses, loop counts—all integers. Understanding integer math prevents subtle bugs.

Where you'll use this

  • Array Indexing: Finding the middle element: mid = (lo + hi) // 2
  • Pagination: Number of pages = ⌈items / page_size⌉
  • Hash Tables: Bucket index = hash(key) mod table_size
  • Cryptography: RSA, Diffie-Hellman use modular arithmetic
  • Graphics: Pixel coordinates, grid snapping
  • Time: hours = minutes // 60, remaining = minutes % 60
Section 1

The Floor Function ⌊x⌋

Definition

⌊x⌋ = the greatest integer less than or equal to x Also called: "floor of x" or "round down" Examples: ⌊3.7⌋ = 3 ⌊5.0⌋ = 5 ⌊-2.3⌋ = -3 (← watch out for negatives!) ⌊π⌋ = 3

Key properties

For any real x and integer n: 1. ⌊x⌋ ≤ x < ⌊x⌋ + 1 2. ⌊x + n⌋=⌊x⌋ + n 3. ⌊⌊x⌋⌋=⌊x⌋ (idempotent) 4. x - 1 < ⌊x⌋ ≤ x
Negative numbers!
⌊-2.3⌋ = -3, NOT -2
Floor goes toward negative infinity, not toward zero.

In code

Python
import math

# Python's math.floor
math.floor(3.7)   # 3
math.floor(-2.3)  # -3

# Integer division with // (floor division)
7 // 2    # 3
-7 // 2   # -4 (floors toward -∞)

# int() truncates toward zero (different!)
int(-2.3)  # -2 (not floor!)
Section 2

The Ceiling Function ⌈x⌉

Definition

⌈x⌉ = the least integer greater than or equal to x Also called: "ceiling of x" or "round up" Examples: ⌈3.2⌉ = 4 ⌈5.0⌉ = 5 ⌈-2.7⌉ = -2 (toward zero for negatives) ⌈π⌉ = 4

Key properties

For any real x and integer n: 1. ⌈x⌉ - 1 < x ≤ ⌈x⌉ 2. ⌈x + n⌉=⌈x⌉ + n 3. ⌈-x⌉=-⌊x⌋ (important identity!) 4. x ≤ ⌈x⌉ < x + 1

Relationship to floor

Key identity:

⌈x⌉ = -⌊-x⌋

For non-integers: ⌈x⌉ = ⌊x⌋ + 1
For integers: ⌈n⌉ = ⌊n⌋ = n

In code

Python
import math

math.ceil(3.2)   # 4
math.ceil(-2.7)  # -2
math.ceil(5.0)   # 5

# Ceiling division (no built-in operator)
def ceildiv(a, b):
    return -(-a // b)  # trick: negate, floor div, negate

ceildiv(7, 2)  # 4
ceildiv(6, 2)  # 3
Section 3

Integer Division

The division algorithm

For integers a and positive b, there exist unique integers q and r: a = b × q + r where 0 ≤ r < b q=quotient=⌊a/b⌋ r=remainder=a mod b Example: 17=5 × 3 + 2 17 ÷ 5: quotient=3, remainder=2

Comparison table

a b a / b (float) a // b (floor) a % b (mod)
17 5 3.4 3 2
-17 5 -3.4 -4 3
17 -5 -3.4 -4 -3
20 5 4.0 4 0
Languages differ on negative division!
Python: floors toward -∞ (mathematically consistent)
C/Java: truncates toward 0 (different for negatives)
Always test with negative numbers in a new language.
Section 4

Modulo Operator

Definition

a mod b = a - b × ⌊a/b⌋ "a mod b" is the remainder when a is divided by b. Examples: 17 mod 5 = 2 (17 = 5×3 + 2) 23 mod 7 = 2 (23 = 7×3 + 2) 100 mod 10 = 0 (100 = 10×10 + 0)

Key properties

(a + b) mod n = ((a mod n) + (b mod n)) mod n (a × b) mod n = ((a mod n) × (b mod n)) mod n (a - b) mod n = ((a mod n) - (b mod n) + n) mod n These let you compute mod of huge numbers!

Common uses

PythonModulo patterns
# Check if even/odd
is_even = n % 2 == 0

# Wrap around (circular array)
next_index = (i + 1) % len(arr)

# Extract last digit
last_digit = n % 10

# Check divisibility
divisible_by_3 = n % 3 == 0

# Hash table index
bucket = hash(key) % num_buckets

# Convert to hours/minutes
hours = total_minutes // 60
minutes = total_minutes % 60
Section 5

CS Applications

Binary search midpoint

Python
# Classic (can overflow in some languages)
mid = (lo + hi) // 2

# Overflow-safe version
mid = lo + (hi - lo) // 2

Pagination

Python
items = 47
page_size = 10

# How many pages needed? Use ceiling!
num_pages = -(-items // page_size)  # = ⌈47/10⌉ = 5

# Or equivalently:
num_pages = (items + page_size - 1) // page_size

Circular buffers

Python
class CircularBuffer:
    def __init__(self, size):
        self.buf = [None] * size
        self.head = 0
        self.size = size
    
    def push(self, item):
        self.buf[self.head] = item
        self.head = (self.head + 1) % self.size  # wrap around!

Digits of a number

Python
def get_digits(n):
    digits = []
    while n > 0:
        digits.append(n % 10)  # last digit
        n = n // 10           # remove last digit
    return digits[::-1]

get_digits(12345)  # [1, 2, 3, 4, 5]
