Truth Tables

Truth tables are the brute-force method of logic. They list every possible input combination. They are identical to checking every case in a switch statement.

The Implication Table (p ⇒ q)

Crucial: "If p then q" is only False when p is True and q is False. If p is False, the statement is vacuously True.
p | q | p ⇒ q --+---+------- T | T | T T | F | F <-- The only false case F | T | T F | F | T
Section 3

Predicates & Quantifiers

Propositional logic is limited. "x > 3" is not a proposition until we know x. A predicate P(x) is a function that returns a proposition.

Negating Quantifiers (De Morgan's Laws for Logic)

¬(∀x, P(x)) ≡ ∃x, ¬P(x)
¬(∃x, P(x)) ≡ ∀x, ¬P(x)

To disprove "All integers are odd", you only need to show "There exists one integer that is even".

Section 4

Logical Equivalence

Variations of p ⇒ q

Original p ⇒ q "If it rains, ground is wet"
Inverse ¬p ⇒ ¬q "If it doesn't rain, ground isn't wet" (Not equivalent!)
Converse q ⇒ p "If ground is wet, it rained" (Not equivalent!)
Contrapositive ¬q ⇒ ¬p "If ground isn't wet, it didn't rain" (Equivalent)
Common Error: Assuming the converse is true. Only the contrapositive is logically equivalent to the original statement.
Section 5

Proof Techniques

1. Direct Proof

Assume p is true, use algebra/logic to show q is true.

Theorem: If n is odd, n² is odd. Proof: Let n be odd. Then n = 2k + 1 for some k ∈ ℤ. n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1 This is of the form 2m + 1, so n² is odd. □

2. Proof by Contrapositive

Instead of p ⇒ q, prove ¬q ⇒ ¬p.

Theorem: If n² is linear, then n is linear (Wait, bad example). Theorem: If n² is even, then n is even. Proof (Contrapositive): Assuming ¬q: Let n be odd. We showed above that if n is odd, n² is odd. Thus ¬q ⇒ ¬p. Therefore, p ⇒ q. □

3. Proof by Contradiction

Assume p is true AND q is False. Show this leads to nonsense (e.g., 0 = 1).

Theorem: √2 is irrational. Proof: Assume √2 is rational. Then √2 = a/b (simplified fraction). 2 = a²/b² ⇒ a² = 2b². So a² is even ⇒ a is even (a = 2k). (2k)² = 2b² ⇒ 4k² = 2b² ⇒ 2k² = b². So b² is even ⇒ b is even. If both a and b are even, the fraction wasn't simplified. Contradiction. Therefore, √2 is irrational. □
