Section 1

Divisibility & Primes

We say $a$ divides $b$ (written $a | b$) if there exists an integer $k$ such that $b = ak$.

Properties

  • If $a | b$ and $b | c$, then $a | c$ (Transitivity).
  • If $a | b$ and $a | c$, then $a | (mb + nc)$ for any integers $m, n$ (Linearity).

Primes

A number $p > 1$ is prime if its only positive divisors are 1 and $p$.

Fundamental Theorem of Arithmetic:
Every integer $n > 1$ can be written as a product of primes in exactly one way (ignoring order).
Section 2

GCD & The Euclidean Algorithm

The Greatest Common Divisor (GCD) of $a$ and $b$ is the largest integer that divides both.

lcm(a,b) = \frac{|a \cdot b|}{gcd(a,b)}

The First Algorithm

Euclid ($~300$ BC) proved that $gcd(a, b) = gcd(b, a \text{ mod } b)$. This lets us efficiently compute GCD.

PythonEuclid's Algorithm (Recursive)
def gcd(a, b):
    # Base case: if remainder is 0, we found it
    if b == 0:
        return a
    # Recursive step: reduce the problem
    return gcd(b, a % b)
Time Complexity: $O(\log(\min(a, b)))$. It is extremely fast, roughly related to the number of digits.
Section 3

Modular Arithmetic

We say $a \equiv b \pmod n$ if $n | (a - b)$. Conceptually, $a$ and $b$ have the "same remainder" when divided by $n$.

Congruence Properties

  • Addition: If $a \equiv b$, then $a+c \equiv b+c$.
  • Multiplication: If $a \equiv b$, then $ac \equiv bc$.
  • Exponentiation: If $a \equiv b$, then $a^k \equiv b^k$.
Division Warning: You cannot simply divide both sides! $2 \cdot 3 \equiv 2 \cdot 0 \pmod 6$, but $3 \not\equiv 0$. You can only divide by $k$ if $gcd(k, n) = 1$.
