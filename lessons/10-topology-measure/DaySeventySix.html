Section 1

Syntax

Everything is a function.

  • Variable: `x`
  • Abstraction: `\lambda x. M` (Function taking x, returning M)
  • Application: `(M N)` (Invoke M with N)

That's it. No numbers, no booleans, no loops. We build them all.

Section 2

Beta Reduction

The "Rule of Computation". Only one thing happens:

(\lambda x. M) N \Rightarrow M[x := N]

Replace all occurrences of `x` in `M` with `N`.

Example: `(\lambda x. x) y \Rightarrow y` (Identity function).

Section 3

Church Numerals

How to represent numbers with functions? By the concept of "doing something n times".

  • `0 = \lambda f. \lambda x. x`
  • `1 = \lambda f. \lambda x. f x`
  • `2 = \lambda f. \lambda x. f (f x)`

Addition `PLUS` is a function that composes two numbers.

Section 4

Common Combinators

  • `I = \lambda x. x` (Identity)
  • `K = \lambda x. \lambda y. x` (True / First)
  • `S = \lambda x. \lambda y. \lambda z. (x z) (y z)` (Substitution)

SKI Calculus is a simplified version using only S, K, and I.

Section 5

Recursion (Y Combinator)

In lambda calculus, functions don't have names, so they can't call themselves. How to recurse?

Y = \lambda f. (\lambda x. f (x x)) (\lambda x. f (x x))

Property: `Y g = g (Y g)`. It finds the Fixed Point of `g`.

This allows us to implement Factorial and infinite loops.

Section 6

Python: Functional Encoding

Python Church Encoding 
# Defining Numbers purely with Functions
# ZERO takes func f and arg x, returns x
ZERO = lambda f: lambda x: x
ONE  = lambda f: lambda x: f(x)
TWO  = lambda f: lambda x: f(f(x))

# SUCC takes number n, returns n+1
SUCC = lambda n: lambda f: lambda x: f(n(f)(x))

# Helper to convert back to int for printing
def to_int(church_num):
    return church_num(lambda n: n + 1)(0)

print(f"Zero: {to_int(ZERO)}")
print(f"One: {to_int(ONE)}")
print(f"Two: {to_int(TWO)}")
print(f"Succ(Two): {to_int(SUCC(TWO))}")

# Y Combinator in Python (using Z for eager evaluation)
Z = lambda f: (lambda x: f(lambda v: x(x)(v))) (lambda x: f(lambda v: x(x)(v)))

FACT = Z(lambda f: lambda n: 1 if n == 0 else n * f(n-1))
print(f"5! using Z: {FACT(5)}")
