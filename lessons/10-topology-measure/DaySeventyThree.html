Section 1

Monotone Convergence Theorem (MCT)

The Theorem

If 0 ≤ f₁ ≤ f₂ ≤ ... is a sequence of measurable functions, then: lim ∫ fₙ = ∫ (lim fₙ)
Intuition: If the functions only go UP, you can't have weird cancellations or mass disappearing at infinity.
Section 2

Fatou's Lemma

The Inequality

For any sequence of non-negative measurable functions fₙ: ∫ (liminf fₙ) ≤ liminf (∫ fₙ)
Strict Inequality Possible: Example: Moving bump fₙ = indicator on [n, n+1]. Limit is 0 everywhere (∫ 0 = 0). But ∫ fₙ = 1 for all n. 0 ≤ 1.
Section 3

Dominated Convergence Theorem (DCT)

The Gold Standard

If fₙ → f pointwise (almost everywhere), AND |fₙ| ≤ g for all n, where g is integrable (∫ |g| < ∞), THEN: lim ∫ fₙ=∫ f
The "Dominating" function g acts as a "lid" that prevents mass from escaping to infinity. This theorem justifies differentiating under the integral sign!

Visualization: Why we need Domination

Example of "Escaping Mass".

Python The "Moving Bump" 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 10, 500)

# Function f_n(x) = 1 if x in [n, n+1], else 0
def bump(x, n):
    return 1.0 * ((x >= n) & (x < n+1))

# Plot n=1, 2, 3
plt.plot(x, bump(x, 1), label="n=1")
plt.plot(x, bump(x, 2), label="n=2")
plt.plot(x, bump(x, 3), label="n=3")

# Pointwise Limit:
# At any fixed x, bump(x, n) eventually becomes 0 as n gets large.
# f(x) = 0. Integral = 0.

# Sequence Integrals:
# Integral of each bump is 1. Limit = 1.

# 0 != 1. DCT fails because there is no integrable 'g' that covers all bumps forever.
