Section 1

First-Order Logic (FOL)

Propositional Logic: `P \implies Q`. (Boolean).

First-Order Logic: `\forall x \exists y, P(x, y)`. Adds predicates, quantifiers, and objects.

It is expressive enough to formalize most of mathematics (ZFC Set Theory).

Section 2

Models, Soundness, Validity

  • Syntax: The symbols written down. `P \vdash \phi` (Provable).
  • Semantics: The meaning/interpretation. `P \vDash \phi` (True in all models).
  • Soundness: If `\vdash \phi` then `\vDash \phi`. (Proofs don't lie).
Section 3

Gödel's Completeness Theorem

Theorem: In First-Order Logic, if `\vDash \phi`, then `\vdash \phi`.

Meaning: If something is true in every possible mathematical universe, there exists a proof for it.

Section 4

Gödel's Incompleteness Theorem

Different from Completeness! This deals with Systems capable of Arithmetic (like PA).

Theorem 1: Any consistent formal system `F` sufficient for arithmetic is incomplete. There exist statements `G` such that `F \not\vdash G` and `F \not\vdash \neg G`.

`G` essentially says "I am not provable in F". If computable, it's true but unprovable.

Theorem 2: `F` cannot prove its own consistency. `F \not\vdash Cons(F)`.
Section 5

Peano Arithmetic

The standard axioms for natural numbers (Successor, Induction). It is incomplete.

Truth in Math > Provability in Math.

Section 6

Python: Quine (Self-Reference)

Python Quine 
# A Quine is a program that prints its own source code.
# This demonstrates the mechanism behind Gödel's proof:
# "Diagonalization Lemma" allows a formula to refer to itself.

s = 's = {0}{1}{0}{2}print(s.format(chr(39), s, chr(10)))'
print(s.format(chr(39), s, chr(10)))

# In Logic:
# Let P(x) be the predicate "x is the Gödel number of a provable formula".
# We construct a sentence G <-> ~P(G).
# G says "I am not provable".
