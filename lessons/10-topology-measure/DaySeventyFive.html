Section 1

The Definition

E[X]

Expectation: E[X] = ∫_Ω X(ω) dP(ω) (Lebesgue integral of X w.r.t. measure P).
Linearity: E[aX + bY] = aE[X] + bE[Y] comes directly from linearity of Integration. Monotonicity: X ≤ Y a.s. ⟹ E[X] ≤ E[Y].
Section 2

Law of the Unconscious Statistician

We usually integrate over ℝ (outcomes), not Ω (abstract space). Change of Variables: ∫_Ω X(ω) dP(ω) = ∫_ℝ x dF_X(x) where F_X is probability measure on ℝ induced by X. If F_X has a density f(x), this becomes: ∫ x f(x) dx (Riemann style).
Section 3

When do densities exist?

Radon-Nikodym Theorem

If a measure ν is "absolutely continuous" w.r.t. μ (meaning μ(A)=0 ⟹ ν(A)=0), then there exists a function f (the density) such that: ν(A) = ∫_A f dμ. In Probability: If distribution P(X ∈ A) assigns 0 prob to sets of 0 Lebesgue measure, then a PDF f(x) exists!
Section 4

Python: Infinite Expectation

Some integrals diverge. The Cauchy distribution has No Expectation.

Python Cauchy Mean Convergence? 
import numpy as np
import matplotlib.pyplot as plt

# Generate samples from Cauchy Distribution
# PDF = 1 / (pi * (1 + x^2)) -> Fat tails!
samples = np.random.standard_cauchy(size=10000)

# Calculate running mean
running_mean = np.cumsum(samples) / np.arange(1, 10001)

plt.plot(running_mean)
plt.title("Cauchy Running Mean (Does not converge!)")
plt.ylabel("Sample Mean")
plt.xlabel("n")
# plt.show()

# Why? Integral of x / (1+x^2) diverges. E[X] is undefined.
# LLN does NOT apply.
