Section 1

The Probability Triplet

In undergraduate stats, we wave our hands about "chance". In rigorous math, Probability is just Measure Theory on a space with total mass 1.

Definition: A Probability Space is a triple `(\Omega, \mathcal{F}, P)` where:
  1. `\Omega`: Sample Space (Set of all outcomes).
  2. `\mathcal{F}`: Event Space (`\sigma`-algebra of `\Omega`).
  3. `P`: Probability Measure (`P(\Omega) = 1`, `P(\emptyset) = 0`, countably additive).

An "Event" is just a measurable set `E \in \mathcal{F}`.

Section 2

Random Variables

A Random Variable is neither random nor a variable. It is a function.

X: \Omega \to \mathbb{R}

However, it cannot be just any function. It must be Measurable.

For any Borel set `B \subseteq \mathbb{R}`, the set `\{\omega \in \Omega \mid X(\omega) \in B\}` must be an event in `\mathcal{F}`.

This ensures we can answer: "What is the probability that `X` is in `B`?"

P(X \in B) = P(X^{-1}(B)) = P(\{\omega \mid X(\omega) \in B\})
Section 3

Distributions (Push-forward Measure)

The random variable `X` "pushes" the measure `P` from `\Omega` onto `\mathbb{R}`.

We define the Distribution of X, denoted `\mu_X`, as:

\mu_X(B) = P(X^{-1}(B))

This `\mu_X` is a measure on `\mathbb{R}`. It is what we usually plot (PDF/CDF). The original `\Omega` (coin flips, particle states) often fades into the background, but it is always there.

Section 4

Independence

In classical probability: `P(A \cap B) = P(A)P(B)`.

In Measure Theory, `X` and `Y` are independent if the `\sigma`-algebras generated by them are independent.

\sigma(X) = \{X^{-1}(B) \mid B \in \mathcal{B}\}

This definition works for continuous, discrete, or mixed variables without needing separate definitions.

Section 5

Python: The Measurable Map

Python Mapping Outcomes to Values 
import numpy as np

# Let Omega be the unit interval [0, 1] with Lebesgue Measure.
# This single space can model ANY Random Variable.

def coin_flip_rv(omega):
    """Maps [0, 1] -> {Heads, Tails}"""
    # 0 <= omega < 0.5 -> Heads (0)
    # 0.5 <= omega <= 1 -> Tails (1)
    return 0 if omega < 0.5 else 1

def uniform_rv(omega):
    """Maps [0, 1] -> [a, b]"""
    return omega # Identity map is Uniform(0, 1)

def exponential_rv(omega, lambd=1.0):
    """Inverse Transform Sampling"""
    # X = -ln(1-U) / lambda
    # This transforms the Uniform measure on Omega to Exponential on R
    return -np.log(1 - omega) / lambd

# Simulate
n_samples = 10000
omegas = np.random.uniform(0, 1, n_samples) # Sampling from Omega

# Transform
X_exp = exponential_rv(omegas, lambd=1.0)

print(f"Mean of Exp: {np.mean(X_exp):.4f} (Expected 1.0)")
print(f"P(X > 1): {np.mean(X_exp > 1):.4f} (Expected 1/e ~ 0.3679)")
