Section 1

The Kolmogorov Triple

Formalizing Luck

Probability Space (Ω, F, P): 1. Ω (Sample Space): The set of all possible outcomes. (Any set X). 2. F (Sigma Algebra): The set of "Events". (Not every subset is an event!). 3. P (Probability Measure): Measure P: F → [0, 1] with P(Ω) = 1.
Why F? If outcome x is in event A, we know A happened. If we roll a die, "Even number" is an Event {2, 4, 6}. We assign probabilities to Events, not just outcomes.
Section 2

Random Variables

Just Measurable Functions

Random Variable X: A measurable function X: Ω → ℝ. (That's it!)
Inverse Image Logic: We want to ask "What is the probability X is between a and b?" P(a ≤ X ≤ b) = P( {ω ∈ Ω | a ≤ X(ω) ≤ b} ) This set must be in F for P to measure it! Hence, X must be measurable.
Section 3

"Almost Surely"

Almost Everywhere (a.e.)

In Measure Theory: Property holds "Almost Everywhere" if it holds on X \ N where μ(N) = 0. In Probability: Property holds "Almost Surely" (a.s.) (Probability 1).
Coin Toss Sequence: P(Heads forever) = 0. Does that mean "Heads forever" is impossible? No, that specific outcome exists in Ω. But it has measure 0.

Python: Discrete vs Measure

Simulating how P is distributed differently.

Python Sampling Distributions 
import numpy as np

# Discrete Measure (Dirac Masses)
# P({1})=0.5, P({-1})=0.5
discrete_samples = np.random.choice([1, -1], size=10)
print(f"Discrete: {discrete_samples}")
# Measures: Sum of point masses.

# Continuous Measure (Lebesgue Density)
# P({x}) = 0 for any specific x.
continuous_samples = np.random.uniform(0, 1, size=5)
print(f"Continuous: {continuous_samples}")
# Measure is integral of density f(x)=1.

# Mixed Measure?
# X = 0 with prob 0.5, X ~ U[0,1] with prob 0.5
# Valid measure! (Decomposition theorem)
