Section 1

Transport Equation

The simplest PDE via conservation of mass.

u_t + c u_x = 0

Solution: `u(x, t) = f(x - ct)`. It just travels with speed `c`.

Section 2

Heat Equation (Diffusion)

Diffusion of energy/particles.

u_t = \alpha \Delta u

Properties:

  • Smoothing: Infinite speed of propagation. Rough data instantly becomes smooth ($C^\infty$).
  • Maximum Principle: Heat flows from hot to cold; max/min values occur at boundaries or initial time.
  • Irreversible: You cannot solve it backwards in time (entropy increases).
Section 3

Wave Equation

Strings, sound, light.

u_{tt} = c^2 \Delta u

Properties:

  • Conservation of Energy: Energy is neither created nor destroyed (unlike Heat, which dissipates).
  • Finite Speed: Information propagates at speed `c`.
  • Singularities Preserved: A kick remains a kick. It does not smooth out.
Section 4

Laplace Equation

Steady state of Heat/Wave.

\Delta u = 0

Solutions are called Harmonic Functions. Mean Value Property: The value at center is average of neighbors.

Section 5

Green's Functions

Technique to solve Inhomogeneous Linear ODEs/PDEs: `Lu = f`.

The Green's Function `G(x, y)` is the response at `x` due to a point source (impulse) at `y`.

L G(x, y) = \delta(x - y)
Section 6

Linear Algebra Analogy

Matrix Equation: `Ax = b`. Solution: `x = A^{-1}b`.

Operator Equation: `Lu = f`. Solution:

u(x) = \int G(x,y)f(y) dy

The Green's Function IS the inverse matrix (with continuous indices).

`\int G(x,y) f(y) dy` corresponds to `\sum_j (A^{-1})_{ij} b_j`.

Section 7

Fundamental Solution

Often `G(x,y)` depends only on distance `|x-y|` for free space problems.

For Laplace equation `-\Delta u = \delta` in 3D:

G(x) = \frac{1}{4\pi |x|}

This is the familiar electric potential `V = k q / r`.

Section 8

Method of Images

If we have boundaries (like `u=0` at `x=0`), we add "fake" sources to cancel out the field at the boundary.

To enforce `G(0, y) = 0`, we place a negative charge at `-y`. `G_{new} = G_{free}(x-y) - G_{free}(x+y)`.

Section 9

Python: Heat & Green's

Python Heat Sim & Inverse Matrix 
import numpy as np
import matplotlib.pyplot as plt

# 1. Solving u_t = u_xx (Heat Eq)
nx = 100
dx = 1.0 / (nx - 1)
x = np.linspace(0, 1, nx)
nt = 1000
dt = 0.00005 
alpha = 1.0
u = np.zeros(nx)
u[nx//2] = 1.0 # Sharp spike initial condition

for n in range(nt):
    un = u.copy()
    for i in range(1, nx-1):
        u[i] = un[i] + alpha * dt / dx**2 * (un[i+1] - 2*un[i] + un[i-1])

plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1)
plt.plot(x, u)
plt.title(f"Heat Diffusion after {nt} steps")

# 2. Green's Matrix (Inverse of Laplacian)
# 1D Poisson Equation: -u'' = f
N = 50
h = 1.0/N
D2 = np.zeros((N, N))
for i in range(N):
    D2[i, i] = 2
    if i > 0: D2[i, i-1] = -1
    if i < N-1: D2[i, i+1] = -1
D2 = D2 / h**2

# Green's Matrix is Inverse of D2
G = np.linalg.inv(D2)
f_source = np.zeros(N)
f_source[N//2] = 1/h # Delta source

u_static = G @ f_source # Steady state response

plt.subplot(1, 2, 2)
plt.plot(u_static)
plt.title("Green's Function (Static Response)")
plt.tight_layout()
plt.show()
