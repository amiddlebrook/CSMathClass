Section 1

Measurable Functions

Definition

f: X → ℝ is Measurable if: For every open set U in ℝ, the preimage f⁻¹(U) is in Σ (the measurable sets of X).
This looks just like continuity! Continuous maps pull open sets to open sets. Measurable maps pull open sets to measurable sets. (Almost all reasonable functions are measurable).
Section 2

Constructing the Integral

Simple Functions

Step 1: Simple Functions A simple function s(x) takes only finitely many values {a₁, ..., aₙ}. ∫ s dμ = ∑ aᵢ · μ(f⁻¹({aᵢ})) (Sum of height × measure of the base). Step 2: Non-negative Functions ∫ f dμ = sup { ∫ s dμ | 0 ≤ s ≤ f, s is simple } (Approximate generally from below).
Section 3

Lebesgue vs Riemann

The Difference

Riemann: Partition the Domain (x-axis). Lebesgue: Partition the Range (y-axis).

Dirichlet Function

f(x) = 1 if x ∈ ℚ, 0 if x ∉ ℚ • Riemann Integral: Undefined (oscillates everywhere). • Lebesgue Integral: 0. (Because μ(ℚ) = 0, so f=0 almost everywhere).
Section 4

Visualization

Visualizing "Horizontal Slicing" (Lebesgue).

Python Horizontal Partitioning 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 10, 1000)
y = np.sin(x) + 2

# Determine sets where y is in range [val, val + delta]
# This corresponds to f^-1([a, b])
val = 2.5
delta = 0.1
mask = (y >= val) & (y < val + delta)

# The "measure" of the base is the length of x where mask is true
measure = np.sum(np.diff(x)[mask[:-1]])

print(f"Measure of set {f: f(x) in [{val}, {val+delta}]} : {measure:.4f}")
# Lebesgue sums height * measure
