Section 1

Bits & Surprise

What is information? It is the resolution of uncertainty.

If I tell you "The sun rose today", learning that is low information (high probability).

If I tell you "I won the lottery", learning that is high information (low probability).

I(x) = -\log_2 P(x)
Section 2

Shannon Entropy

Entropy `H(X)` is the average information content of a random variable.

H(X) = - \sum P(x) \log_2 P(x)

It represents the minimum average number of bits needed to encode the variable.

Section 3

Source Coding Theorem

Theorem: You cannot compress data below its entropy without losing information.

Lossless compression (ZIP, PNG) relies on patterns to approach entropy.

Lossy compression (JPEG, MP3) throws away "unimportant" bits to go below entropy.

Section 4

Huffman Coding

An algorithm to construct optimal prefix codes.

High frequency symbols get short codes (e.g. 'e' = 0). Low frequency symbols get long codes (e.g. 'z' = 1101).

This is a Greedy Algorithm that builds a binary tree from the bottom up.

Section 5

Channel Capacity

Noisy Channel Coding Theorem: For any channel with capacity `C`, data can be transmitted reliably at rate `R < C` using error-correcting codes.

This birthed the digital age. Errors can be detected and corrected, making digital storage perfect.

Section 6

Python: Entropy Calc

Python Entropy 
import math
from collections import Counter

def calculate_entropy(data):
    if not data:
        return 0
    
    counts = Counter(data)
    total = len(data)
    entropy = 0
    
    for count in counts.values():
        p = count / total
        entropy -= p * math.log2(p)
        
    return entropy

s1 = "AAAAABBBBB" # 50/50 split
s2 = "AAAAAAAAAB" # Highly predictable
s3 = "ABCDEFGHIJ" # Maximum surprise

print(f"Entropy s1: {calculate_entropy(s1):.2f} bits (Max 1.0)")
print(f"Entropy s2: {calculate_entropy(s2):.2f} bits (Low)")
print(f"Entropy s3: {calculate_entropy(s3):.2f} bits (High)")

# Huffman Logic (Conceptual)
# s2 could be encoded: A=0, B=1. 
# Code: 0000000001 (10 bits).
# Avg bits per symbol = 1.0. 
# But Entropy is 0.47! 
# Optimal block coding would do better than 1 bit per symbol.
