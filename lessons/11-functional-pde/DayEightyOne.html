Section 1

Convex Sets

Definition

A set `C \subseteq \mathbb{R}^n` is convex if the line segment between any two points in `C` lies lies entirely in `C`.

\forall x, y \in C, \theta \in [0, 1] \implies \theta x + (1-\theta)y \in C
Section 1.1

Operations Preserving Convexity

How do we construct complex convex sets from simple ones?

  • Intersection: The intersection of ANY number (even infinite) of convex sets is convex.
  • Affine Map: `f(x) = Ax + b` maps convex sets to convex sets. `f^{-1}(C)` is also convex.
  • Perspective Function: `P(x, t) = x/t` (for `t > 0`) preserves convexity.
Section 2

Convex Functions

Definition

A function `f: \mathbb{R}^n \to \mathbb{R}` is convex if its domain is a convex set and for all `x, y` in the domain and `\theta \in [0, 1]`:

f(\theta x + (1-\theta)y) \le \theta f(x) + (1-\theta)f(y)

Geometrically, the chord connecting `(x, f(x))` and `(y, f(y))` lies above the graph of `f`.

First-Order Condition

If `f` is differentiable, it is convex if and only if its graph lies above its tangent hyperplanes:

f(y) \ge f(x) + \nabla f(x)^T (y-x)

Second-Order Condition

If `f` is twice differentiable, it is convex if and only if its Hessian is Positive Semidefinite (PSD):

\nabla^2 f(x) \succeq 0 \quad (\forall z, z^T \nabla^2 f(x) z \ge 0)
Section 3

Jensen's Inequality

This is arguably the most important inequality in analysis and probability.

Theorem: If `f` is convex, then for any random variable `X`:
f(\mathbb{E}[X]) \le \mathbb{E}[f(X)]

Example: `f(x) = x^2` is convex. Thus `(\mathbb{E}[X])^2 \le \mathbb{E}[X^2]`. This implies `\text{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 \ge 0`, which makes sense!

Section 4

Separating Hyperplane Theorem

Theorem Statement

If `C` and `D` are disjoint nonempty convex sets, there exists a nonzero vector `a` and a scalar `b` such that:

a^T x \le b \quad \forall x \in C \\ a^T x \ge b \quad \forall x \in D

This defines a hyperplane `H = \{x | a^T x = b\}` that separates the sets.

This is the geometric engine behind duality and Support Vector Machines (SVM). It guarantees that if a point `y` is not in a convex set `C`, we can separate it from `C`.
Section 5

Fenchel Conjugate

The conjugate `f^*(y)` of a function `f(x)` is defined as:

f^*(y) = \sup_{x \in \text{dom } f} (y^T x - f(x))

Visually, `f^*(y)` is the maximum gap between the linear function `y^T x` and `f(x)`. It encodes the optimal intercept of a supporting hyperplane with slope `y`.

Crucial Property: `f^*` is always convex, even if `f` is not! This is because it is the pointwise supremum of a family of affine functions (of `y`).

Also, if `f` is convex and closed, then `f^{**} = f` (Fenchel-Moreau Theorem).

Section 6

Deriving the Dual (LP Example)

Let's derive the dual of a Linear Program (LP) from first principles (Lagrangian).

\text{Primal: } \min c^T x \quad \text{s.t.} \quad Ax = b, x \ge 0

Step 1: The Lagrangian. Introduce dual variables `\nu` (for `Ax=b`) and `\lambda \ge 0` (for `-x \le 0`).

L(x, \lambda, \nu) = c^T x + \nu^T (Ax - b) - \lambda^T x \\ = (c + A^T \nu - \lambda)^T x - b^T \nu

Step 2: The Dual Function. `g(\lambda, \nu) = \inf_x L(x, \lambda, \nu)`.

The term is linear in `x`. If the coefficient `(c + A^T \nu - \lambda)` is nonzero, the infimum is `-\infty`. If it is zero, the infimum is `-b^T \nu`.

Step 3: The Dual Problem. Maximize `g(\lambda, \nu)` subject to `\lambda \ge 0`.

\text{Dual: } \max -b^T \nu \quad \text{s.t.} \quad A^T \nu + c - \lambda = 0, \lambda \ge 0

Eliminating `\lambda` (which just means `A^T \nu + c \ge 0`):

\max -b^T \nu \quad \text{s.t.} \quad A^T \nu + c \ge 0

Weak Duality: `d^* \le p^*`. The dual always provides a lower bound.

Section 7

Standard Form Problem

We consider the standard optimization problem:

\begin{aligned} \min_{x} \quad & f_0(x) \\ \text{subject to} \quad & f_i(x) \le 0, \quad i = 1, \dots, m \\ & h_i(x) = 0, \quad i = 1, \dots, p \end{aligned}

The domain of the problem is `D = \bigcap \text{dom } f_i \cap \bigcap \text{dom } h_i`.

Section 8

The Lagrangian

The Lagrangian function `L: \mathbb{R}^n \times \mathbb{R}^m \times \mathbb{R}^p \to \mathbb{R}` augments the objective with weighted constraint violations.

L(x, \lambda, \nu) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{i=1}^p \nu_i h_i(x)

Here `\lambda` and `\nu` are called dual variables or Lagrange multipliers. Crucially: `\lambda_i` must be non-negative (`\lambda_i \ge 0`).

Section 9

The KKT Conditions

If Strong Duality holds (see below), any pair of primal and dual optimal points `(x^*, (\lambda^*, \nu^*))` must satisfy the KKT conditions.

1. Stationarity

The gradient of the Lagrangian with respect to `x` vanishes:

\nabla f_0(x^*) + \sum \lambda_i^* \nabla f_i(x^*) + \sum \nu_i^* \nabla h_i(x^*) = 0

2. Primal Feasibility

The point `x^*` must satisfied the constraints:

f_i(x^*) \le 0, \quad h_i(x^*) = 0

3. Dual Feasibility

The inequality multipliers must be non-negative:

\lambda_i^* \ge 0

4. Complementary Slackness

Either the constraint is active (`f_i(x) = 0`) or the multiplier is zero (`\lambda_i = 0`).

\lambda_i^* f_i(x^*) = 0 \quad \forall i

For convex problems, the KKT conditions are also sufficient for optimality!

Section 10

Strong Duality

Let `p^*` be the primal optimal value and `d^*` be the dual optimal value.

  • Weak Duality: `d^* \le p^*` (Always true).
  • Strong Duality: `d^* = p^*` (The interesting case).

Strong duality means we can solve the (often easier) dual problem to find the optimal value of the primal.

Section 11

Slater's Condition

When does Strong Duality hold? For convex problems, a sufficient condition is Slater's Condition.

Strict Feasibility: There exists a point `x \in \text{relint}(D)` such that:
f_i(x) < 0 \quad \forall i=1, \dots, m \quad \text{and} \quad h_i(x)=0

Basically, if there is a point strictly inside the feasible set, strong duality holds.

Section 12

Python: Convex & SLSQP

The Convex Hull is the smallest convex set containing a given set of points. We can compute it efficiently using `scipy.spatial`.

Python Convex Hull (SciPy) 
import numpy as np
import matplotlib.pyplot as plt
from scipy.spatial import ConvexHull

# 1. Generate random points in 2D
points = np.random.rand(30, 2)

# 2. Compute Convex Hull
hull = ConvexHull(points)

# 3. Plotting
plt.figure(figsize=(8, 6))

# Plot all points
plt.plot(points[:,0], points[:,1], 'o', color='blue')

# Plot hull edges
for simplex in hull.simplices:
    # simplex is an array of indices forming an edge
    plt.plot(points[simplex, 0], points[simplex, 1], 'r-', lw=2)

# Highlight hull vertices
plt.plot(points[hull.vertices,0], points[hull.vertices,1], 'ro')

plt.title('Convex Hull of 30 Random Points')
plt.grid(True)
plt.show()

# Verification of Convexity Definition:
# Pick any two points inside the hull (e.g., averages of vertices)
p_avg = np.mean(points[hull.vertices], axis=0) # Centroid is definitely inside
print(f"Centroid: {p_avg}") # This point is a convex combination!
Python Minimize with Constraints (SLSQP) 
import numpy as np
from scipy.optimize import minimize

# 1. Objective Function: x^2 + y^2
def objective(x):
    return x[0]**2 + x[1]**2

# 2. Constraints
# Constraint: x + y - 1 = 0
cons = ({'type': 'eq', 'fun': lambda x: x[0] + x[1] - 1})

# 3. Solve
x0 = [2.0, 2.0]
res = minimize(objective, x0, method='SLSQP', constraints=cons)

print("Status:", res.message)
print("Optimal Solution x*:", res.x)
print("Optimal Value p*:", res.fun)
print("Expected x*:", [0.5, 0.5])
