Section 1

Normed Linear Spaces

A Normed Space is a vector space `V` with a function `||·||: V → [0, ∞)` satisfying:

1. ||x|| ≥ 0, and ||x||=0 iff x=0 (Positivity) 2. ||cx|| = |c| ||x|| (Homogeneity) 3. ||x + y|| ≤ ||x|| + ||y|| (Triangle Inequality)

It generalizes the notion of "length" to general spaces.

Section 2

Banach Spaces

A Banach Space is a normed space that is complete.
"Complete" means every Cauchy sequence converges to a limit inside the space.

Example: `ℝⁿ` is Banach. `C[0,1]` (continuous functions) with the sup-norm `||f|| = max|f(x)|` is Banach.

Non-Example: `C[0,1]` with the L¹ norm `\int |f|` is NOT Banach (limiting sequence can be discontinuous).

Section 3

Hilbert Spaces

A Hilbert Space has even more structure: an inner product `⟨x, y⟩`. The norm is defined by `||x|| = \sqrt{⟨x, x⟩}`.

• Complete inner product space. • Allows geometry: angles, orthogonality, projections. • Example: L² (square integrable functions). • "The most minimal infinite dimensional structure."
Pythagorean Theorem: If `x ⊥ y` (i.e., `⟨x, y⟩ = 0`), then `||x+y||² = ||x||² + ||y||²`. True in Hilbert spaces (like Euclidean space), but rarely in general Banach spaces.
Section 4

Standard Examples: Lᵖ and ℓᵖ

ℓᵖ (Little L-p)

Sequences `x = (x_1, x_2, ...)` such that `\sum |x_i|^p < \infty`.

Norm: ||x||_p = (\sum |x_i|^p)^{1/p}

For p=2, it's a Hilbert space `ℓ²`.

Lᵖ (Big L-p)

Functions `f` on measure space `\Omega` such that `\int |f|^p < \infty`.

Norm: ||f||_p = (\int |f|^p)^{1/p}

L² is the only Lᵖ space that is Hilbert.

Section 5

Python: p-Norms Visualization

Visualizing the unit balls (set of x where ||x|| ≤ 1) for different p values.

Python Unit Balls for p=1, 2, ∞ 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-1.5, 1.5, 400)
X, Y = np.meshgrid(x, x)

# Define norms
L1 = np.abs(X) + np.abs(Y)
L2 = np.sqrt(X**2 + Y**2)
Linf = np.maximum(np.abs(X), np.abs(Y))

plt.contour(X, Y, L1, levels=[1], colors='r') # Diamond
plt.contour(X, Y, L2, levels=[1], colors='g') # Circle
plt.contour(X, Y, Linf, levels=[1], colors='b') # Square
plt.title("Unit Balls: p=1 (Red), p=2 (Green), p=∞ (Blue)")
plt.axis('equal')
