Section 1

Concept

Technique to solve Inhomogeneous Linear ODEs/PDEs: `Lu = f`.

The Green's Function `G(x, y)` is the response at `x` due to a point source (impulse) at `y`.

L G(x, y) = \delta(x - y)
Section 2

Linear Algebra Analogy

Matrix Equation: `Ax = b`. Solution: `x = A^{-1}b`.

Operator Equation: `Lu = f`. Solution:

u(x) = \int G(x,y)f(y) dy

The Green's Function IS the inverse matrix (with continuous indices).

`\int G(x,y) f(y) dy` corresponds to `\sum_j (A^{-1})_{ij} b_j`.

Section 3

Fundamental Solution

Often `G(x,y)` depends only on distance `|x-y|` for free space problems.

For Laplace equation `-\Delta u = \delta` in 3D:

G(x) = \frac{1}{4\pi |x|}

This is the familiar electric potential `V = k q / r`.

Section 4

Poisson Equation

`-\Delta u = f`.

Solution via Convolution:

u = f * G

We just sum up the potentials from all the little bits of charge density `f`.

Section 5

Method of Images

If we have boundaries (like `u=0` at `x=0`), we add "fake" sources to cancel out the field at the boundary.

To enforce `G(0, y) = 0`, we place a negative charge at `-y`. `G_{new} = G_{free}(x-y) - G_{free}(x+y)`.

Section 6

Python: Discrete Green's

Python Inverting Laplacian 
import numpy as np
import matplotlib.pyplot as plt

# 1D Poisson Equation: -u'' = f
# Discrete Laplacian Matrix D2
N = 50
h = 1.0/N
# Construct D2 matrix (-1, 2, -1) on diag
D2 = np.zeros((N, N))
for i in range(N):
    D2[i, i] = 2
    if i > 0: D2[i, i-1] = -1
    if i < N-1: D2[i, i+1] = -1
D2 = D2 / h**2

# Green's Matrix is Inverse of D2
G = np.linalg.inv(D2)

# Verify: Solve -u'' = delta (point source at center)
f = np.zeros(N)
f[N//2] = 1/h # Delta approx

# u = G * f
u = G @ f

plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1)
plt.imshow(G)
plt.title("Green's Matrix (Discrete Inverse)")
plt.colorbar()

plt.subplot(1, 2, 2)
plt.plot(u)
plt.title("Response to Point Source (Green's Function)")
# Should look like a triangle (solution to -u'' = delta is u = |x|/2 approx)
plt.show()
