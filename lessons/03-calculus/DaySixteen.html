Section 0

Why Multiple Integrals Matter

Real-world data is multidimensional. Computing areas, volumes, mass, probability over regions requires multiple integration.

Applications

  • Probability: P(X,Y ∈ Region) = ∫∫ f(x,y) dA
  • Physics: Mass = ∫∫∫ ρ(x,y,z) dV
  • Graphics: Rendering, volume calculations
  • Machine Learning: Marginal distributions, normalizing constants
Section 1

Double Integrals

Definition

Double Integral over Region R
∫∫_R f(x,y) dA = lim Σ f(xᵢ,yⱼ) ΔA

Iterated integrals

For a rectangular region [a,b] × [c,d]: ∫∫_R f(x,y) dA = ∫ₐᵇ ∫_c^d f(x,y) dy dx = ∫_c^d ∫ₐᵇ f(x,y) dx dy Fubini's Theorem: Order can be switched if f is continuous.

Non-rectangular regions

Type I: a ≤ x ≤ b, g₁(x) ≤ y ≤ g₂(x) ∫∫_R f(x,y) dA = ∫ₐᵇ ∫_{g₁(x)}^{g₂(x)} f(x,y) dy dx Type II: c ≤ y ≤ d, h₁(y) ≤ x ≤ h₂(y) ∫∫_R f(x,y) dA = ∫_c^d ∫_{h₁(y)}^{h₂(y)} f(x,y) dx dy

Example

∫∫_R xy dA where R is 0 ≤ x ≤ 1, 0 ≤ y ≤ x = ∫₀¹ ∫₀ˣ xy dy dx = ∫₀¹ x[y²/2]₀ˣ dx = ∫₀¹ x·(x²/2) dx = ∫₀¹ x³/2 dx = [x⁴/8]₀¹ = 1/8
Section 2

Triple Integrals

Setup

∫∫∫_E f(x,y,z) dV For a box [a,b] × [c,d] × [p,q]: = ∫ₐᵇ ∫_c^d ∫_p^q f(x,y,z) dz dy dx Volume: ∫∫∫_E 1 dV = Volume(E)

Applications

Mass: M = ∫∫∫_E ρ(x,y,z) dV Center of mass: x̄ = (1/M) ∫∫∫ x·ρ dV ȳ = (1/M) ∫∫∫ y·ρ dV z̄ = (1/M) ∫∫∫ z·ρ dV
Section 3

Change of Variables

The formula

Change of Variables
∫∫_R f(x,y) dxdy = ∫∫_S f(x(u,v), y(u,v)) |J| dudv

where J is the Jacobian:

J = ∂(x,y)/∂(u,v) = |∂x/∂u ∂x/∂v| |∂y/∂u ∂y/∂v|

Why the Jacobian?

Key insight: |det(J)| measures how much area changes under the transformation. A small rectangle duv in uv-space becomes a parallelogram with area |J| dudv in xy-space.
Section 4

Polar & Spherical Coordinates

Polar coordinates (2D)

x = r cos(θ), y = r sin(θ) Jacobian: |J| = r ∫∫_R f(x,y) dxdy = ∫∫_S f(r,θ) r dr dθ Use when: circular symmetry, x² + y² appears

Example: Area of circle

Area = ∫∫_{x²+y²≤R²} 1 dxdy In polar: 0 ≤ r ≤ R, 0 ≤ θ ≤ 2π = ∫₀^{2π} ∫₀^R r dr dθ = ∫₀^{2π} [r²/2]₀^R dθ = ∫₀^{2π} R²/2 dθ = πR²

Spherical coordinates (3D)

x = ρ sin(φ) cos(θ) y = ρ sin(φ) sin(θ) z = ρ cos(φ) ρ = radius, φ = angle from z-axis, θ = angle in xy-plane |J| = ρ² sin(φ) dV = ρ² sin(φ) dρ dφ dθ

Cylindrical coordinates

x = r cos(θ) y = r sin(θ) z = z |J| = r dV = r dr dθ dz Use when: cylindrical symmetry
Section 5

Line Integrals

Scalar line integral

Line Integral of Scalar Function
∫_C f ds = ∫ₐᵇ f(r(t)) |r'(t)| dt

Integrates f along the curve C. Think: "weighted arc length."

If f = 1: ∫_C 1 ds = arc length of C Example: C is unit circle, f = 1 Arc length = ∫₀^{2π} 1 · 1 dt = 2π

Vector line integral (work)

Line Integral of Vector Field
∫_C F · dr = ∫ₐᵇ F(r(t)) · r'(t) dt

Computes work done by force F along path C.

Physical meaning: • F · dr = component of force along path × distance • Sum up these tiny work contributions If F ⊥ path everywhere: work = 0 If F ∥ path everywhere: work = |F| × length
Section 6

Conservative Vector Fields

Definition

F is conservative if F = ∇φ for some potential function φ. Then: ∫_C F · dr = φ(end) - φ(start) Path independent! Only endpoints matter.

Test for conservative

In 2D: F = (P, Q) is conservative if ∂P/∂y = ∂Q/∂x In 3D: F is conservative if curl(F) = ∇ × F = 0 Conservative ⟺ curl-free (in simply connected domains)
Key insight: Gravity is conservative. Friction is not. This is why energy is conserved in some systems but not others.
Section 7

Surface Integrals

Scalar surface integral

Surface Integral (Scalar)
∫∫_S f dS = "sum f over surface"

If f = 1: this gives surface area.

Flux integral

Flux Through Surface
∫∫_S F · dS = ∫∫_S F · n̂ dS
Measures how much F "flows through" surface S. n̂ = unit normal to surface F · n̂ = component of F perpendicular to surface Examples: • Water flow through a net • Electric field through a surface (Gauss's law)
Section 8

The Big Theorems (Overview)

Green's Theorem (2D)

∮_C (P dx + Q dy) = ∫∫_D (∂Q/∂x - ∂P/∂y) dA Relates line integral around boundary to double integral over region.

Stokes' Theorem (3D)

∮_C F · dr = ∫∫_S (∇ × F) · dS Relates line integral around boundary to surface integral of curl.

Divergence Theorem

∫∫_S F · dS = ∫∫∫_V (∇ · F) dV Relates flux through closed surface to triple integral of divergence.
Pattern: All three theorems relate a boundary integral to an integral over the interior. They're all versions of the Fundamental Theorem of Calculus in higher dimensions!
