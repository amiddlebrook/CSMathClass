Why Differential Equations?

ODEs describe how things change. Population growth, system dynamics, signal processing—all modeled by differential equations.

CS applications

  • Control systems: PID controllers, robotics
  • Machine learning: Neural ODEs, gradient flow
  • Graphics: Physics simulation, animation
  • Signal processing: Filters, frequency response
  • Networking: Queueing theory, TCP dynamics
Section 1

First-Order ODEs

General form

First-Order ODE
dy/dx = f(x, y)

One derivative, one unknown function y(x). Initial condition: y(x₀) = y₀.

The exponential equation

dy/dx = ky Solution: y = Ce^{kx} With y(0) = y₀: y = y₀ e^{kx} k > 0: exponential growth k < 0: exponential decay
Key insight: The solution to dy/dx = ky is the ONLY function that equals its own derivative (up to scaling). This is why e^x is so special!
Section 2

Separable Equations

Method

If dy/dx = g(x)h(y), then: 1. Separate: dy/h(y) = g(x) dx 2. Integrate both sides: ∫ dy/h(y) = ∫ g(x) dx 3. Solve for y if possible

Example

dy/dx = xy Separate: dy/y = x dx Integrate: ln|y| = x²/2 + C Solve: y = Ae^{x²/2} where A = ±e^C
Section 3

Linear First-Order ODEs

Standard form

Linear First-Order
dy/dx + P(x)y = Q(x)

Integrating factor method

1. Compute μ(x) = e^{∫P(x)dx} 2. Multiply through: μy' + μPy = μQ 3. Recognize: (μy)' = μQ 4. Integrate: μy = ∫μQ dx 5. Solve: y = (1/μ)∫μQ dx

Example

dy/dx + 2y = e^x P(x) = 2, so μ = e^{2x} (e^{2x}y)' = e^{2x}·e^x = e^{3x} e^{2x}y = e^{3x}/3 + C y = e^x/3 + Ce^{-2x}
Section 4

Second-Order Linear ODEs

Homogeneous with constant coefficients

Second-Order Homogeneous
ay'' + by' + cy = 0

Characteristic equation

Try y = e^{rx}. Substituting gives: ar² + br + c = 0 Roots r₁, r₂ determine solution type: Case 1: Two distinct real roots r₁ ≠ r₂ y = C₁e^{r₁x} + C₂e^{r₂x} Case 2: Repeated root r = r₁ = r₂ y = (C₁ + C₂x)e^{rx} Case 3: Complex roots r = α ± βi y = e^{αx}(C₁cos(βx) + C₂sin(βx))

Physical interpretation

Mass-spring-damper: my'' + cy' + ky = 0 • Overdamped (distinct real): slow return, no oscillation • Critically damped (repeated): fastest return, no oscillation • Underdamped (complex): oscillates while decaying
Section 5

Numerical Methods

Euler's method

Given dy/dx = f(x,y), y(x₀) = y₀ yₙ₊₁ = yₙ + h·f(xₙ, yₙ) Step forward by h, using slope at current point.
PythonEuler's Method
def euler(f, x0, y0, h, n_steps):
    """Solve dy/dx = f(x,y) from x0 with y(x0)=y0"""
    x, y = x0, y0
    points = [(x, y)]
    for _ in range(n_steps):
        y = y + h * f(x, y)
        x = x + h
        points.append((x, y))
    return points

# Example: dy/dx = y, y(0) = 1 → y = e^x
f = lambda x, y: y
euler(f, 0, 1, 0.1, 10)  # Approximates e^1

Better methods

• Runge-Kutta 4 (RK4): Most common, very accurate • Adams-Bashforth: Multi-step method • scipy.integrate.odeint: Use this in practice!
