Why Systems of ODEs?

Real systems have multiple interacting parts. Population dynamics, coupled oscillators, neural networks—all require systems of equations.

Applications

  • Control systems: State-space models
  • Machine learning: Recurrent dynamics
  • Epidemiology: SIR models
  • Economics: Dynamic equilibrium
Section 1

Systems of ODEs

Example: Coupled equations

dx/dt = ax + by dy/dt = cx + dy Two unknowns x(t), y(t), each affects the other.

Higher-order → system

Any higher-order ODE can be rewritten as a system: y'' + 3y' + 2y = 0 Let x₁ = y, x₂ = y'. Then: x₁' = x₂ x₂' = y'' = -3y' - 2y = -3x₂ - 2x₁
Section 2

Matrix Form

Vector notation

System in Matrix Form
x' = Ax where x = [x₁, x₂, ...]ᵀ
dx/dt = ax + by ┌x'┐ ┌a b┐ ┌x┐ dy/dt = cx + dy → └y'┘ = └c d┘ └y┘ Or: x' = Ax where A = [a b; c d]

Solution form

Analogous to scalar case (y' = ky → y = Ce^{kt}): x' = Ax → x(t) = e^{At} x₀ Matrix exponential! But computing e^{At} requires diagonalization.
Section 3

Eigenvalue Method

Key insight

Try x = ve^{λt} where v is a constant vector. Then:
x' = λve^{λt} = Ax = Ave^{λt}
So: Av = λv — eigenvalue equation!

Algorithm

1. Find eigenvalues: det(A - λI) = 0 2. Find eigenvectors: (A - λI)v = 0 3. General solution: x(t) = c₁v₁e^{λ₁t} + c₂v₂e^{λ₂t} + ...

Example

A = [3 1] [0 2] det(A - λI) = (3-λ)(2-λ) = 0 → λ₁ = 3, λ₂ = 2 For λ₁ = 3: v₁ = [1, 0]ᵀ For λ₂ = 2: v₂ = [1, -1]ᵀ Solution: x(t) = c₁[1,0]ᵀe^{3t} + c₂[1,-1]ᵀe^{2t}

Complex eigenvalues

If λ = α ± βi: Solution involves e^{αt}cos(βt) and e^{αt}sin(βt) Spiraling behavior: • α < 0: spiral inward (stable) • α> 0: spiral outward (unstable) • α = 0: pure oscillation
Section 4

Phase Portraits

Classification by eigenvalues

Real eigenvalues: • Both negative: stable node (all solutions → origin) • Both positive: unstable node (all solutions → ∞) • Opposite signs: saddle point (unstable) Complex eigenvalues (α ± βi): • α < 0: stable spiral • α> 0: unstable spiral • α = 0: center (periodic orbits)
PythonPlot phase portrait
import numpy as np
import matplotlib.pyplot as plt

A = np.array([[-1, 1], [-1, -1]])  # stable spiral

# Create flow field
x = np.linspace(-2, 2, 20)
X, Y = np.meshgrid(x, x)
U = A[0,0]*X + A[0,1]*Y
V = A[1,0]*X + A[1,1]*Y

plt.streamplot(X, Y, U, V)
plt.show()
