Section 1

Riemann Sums (The Foundations)

Before Newton discovered the fancy rules, he just added up rectangles. That's all an integral is.

$$ \int_{a}^{b} f(x) dx \approx \sum_{i=0}^{N} f(x_i) \cdot \Delta x $$ ‚Ä¢ $f(x_i)$: Height of the rectangle. ‚Ä¢ $\Delta x$: Width of the rectangle. ‚Ä¢ $\sum$: The For Loop.
ü•™ Analogy: The Infinite Sandwich
Imagine a loaf of bread. To find its volume, you can:
1. Slice it into a few thick slices (Bad approximation).
2. Slice it into 1000 thin wafers (Good approximation).
3. Slice it into infinitely many infinitely thin slices (The Exact Integral).
Each slice is $f(x) \cdot dx$.
‚¨ÖÔ∏è Left Riemann Sum
Use left endpoint: $f(x_i)$
Underestimates increasing functions.
‚û°Ô∏è Right Riemann Sum
Use right endpoint: $f(x_{i+1})$
Overestimates increasing functions.

Mean Value Theorem for Integrals

There exists a point $c$ in $[a,b]$ such that:

$f(c) = \frac{1}{b-a} \int_a^b f(x)dx$

Intuition: $f(c)$ is the Average Value of the function. If you flattened all the hills and filled all the valleys, the height of the resulting rectangle would be $f(c)$.

Python Riemann Sum as a For Loop 
def riemann_sum(f, a, b, n=1000):
    """The integral IS the for loop."""
    dx = (b - a) / n
    total = 0
    for i in range(n):
        x = a + i * dx
        total += f(x) * dx  # Rectangle area
    return total

# Integral of x^2 from 0 to 3
result = riemann_sum(lambda x: x**2, 0, 3)
print(f"Approx: {result:.4f}")  # ~9.0
Section 2

Fundamental Theorem of Calculus

The bridge between Slope (Derivative) and Area (Integral).

Part 1 (Differentiation of Integrals):
$\frac{d}{dx}\int_a^x f(t)dt = f(x)$
Taking the derivative of an integral gives back the original function.
Part 2 (Evaluation):
$\int_{a}^{b} f(x) dx = F(b) - F(a)$
Where $F'(x) = f(x)$ (antiderivative).
üöó Example: Distance Traveled
Car velocity: $v(t) = 2t$ m/s. How far in 5 seconds?
1. Antiderivative of $2t$ is $t^2$.
2. $F(5) - F(0) = 25 - 0 = 25$ meters.
Section 3

Integration Rules & Techniques

Basic Rules

Power Rule:
$\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (n ‚â† -1)
$\int x^{-1} dx = \ln|x| + C$
Exponential & Trig:
$\int e^x dx = e^x + C$
$\int \sin(x) dx = -\cos(x) + C$
$\int \cos(x) dx = \sin(x) + C$

Advanced Techniques

U-Substitution

Reverse of chain rule.

$\int f(g(x))g'(x)dx = \int f(u)du$

Integration by Parts

Reverse of product rule.

$\int u \cdot dv = uv - \int v \cdot du$

Section 5

Numerical Integration Methods

When you can't find an antiderivative analytically (most real functions!), use numerical methods.

Trapezoid Rule

Use trapezoids instead of rectangles.

$\int_a^b f(x)dx \approx \sum \frac{f(x_i) + f(x_{i+1})}{2} \cdot \Delta x$

Error: O(Œîx¬≤)

Simpson's Rule

Fit parabolas to triplets of points.

Much more accurate for smooth functions.

Error: O(Œîx‚Å¥)

Python scipy.integrate 
from scipy import integrate

def f(x):
    return x**2

# Exact: integral of x^2 from 0 to 3 = 9.0
result, error = integrate.quad(f, 0, 3)
print(f"Result: {result:.6f}, Error: {error:.2e}")

# Output: Result: 9.000000, Error: 9.99e-14
Section 6

Monte Carlo Integration (The Probabilistic Approach)

What if the function is in 100 dimensions? Rectangles and trapezoids fail (The Curse of Dimensionality). In computer science, we use Randomness.

üéØ Analogy: Darts on a Board
If you want to find the area of an irregular shape on a wall:
1. Draw a known rectangle around it.
2. Throw 10,000 darts randomly at the wall.
3. Count how many landed inside the shape.
4. Area ‚âà (Darts Inside / Total Darts) * Area of Rectangle.
Python Monte Carlo from Scratch 
import random

def monte_carlo_integral(f, a, b, height, n=100000):
    """Estimate area by throwing 'darts'."""
    inside = 0
    for _ in range(n):
        x = random.uniform(a, b)
        y = random.uniform(0, height)
        if y <= f(x):
            inside += 1
            
    # fraction * bounding_box_area
    return (inside / n) * (b - a) * height

# Estimate area of x^2 from 0 to 3 (Max height is 9)
area = monte_carlo_integral(lambda x: x**2, 0, 3, 9)
print(f"Monte Carlo Est: {area:.4f}") # ~9.0
Section 7

Improper Integrals (To Infinity & Beyond)

What happens if the area goes on forever? Does it explode, or converge?

$\int_1^\infty \frac{1}{x^p} dx$
‚Ä¢ If $p > 1$: Converges (Finite Area).
‚Ä¢ If $p \leq 1$: Diverges (Infinite Area).
ü™ú Analogy: The Infinite Ladder
Imagine a ladder where each rung is half the height of the previous one.
Even if you have infinitely many rungs, you will never reach a height of 2 meters.
This is how an infinite sum (or integral) can be finite.
The Gabriel's Horn Paradox:
The surface $y = 1/x$ rotated around the x-axis from $1$ to $\infty$.
It has infinite surface area but finite volume ($\pi$).
You could fill it with paint but never paint the surface!
Section 8

Line Integrals: Paths in Space

In CS and Robotics, we don't just integrate over a line segment. We integrate over a path.

Work along a Path

$\int_C \mathbf{F} \cdot d\mathbf{r}$

If you move a robot through a wind field $\mathbf{F}$ along path $C$, the line integral calculates the total work done against the wind.

üó∫Ô∏è Analogy: The Hiking Trail
Walking from A to B on a map is 5 miles.
But if the trail winds up and down hills (a path in 3D), the effort you expend is the integral of the slope along that path.
Section 9

Integration vs. Automatic Differentiation

In Day 13, we learned that differentiation is "solved" in CS via Autodiff. Is integration also solved?

Differentiation (Easy):
Any function you can write as code has a derivative you can calculate via the chain rule.
Integration (Hard):
Many simple functions (like $e^{-x^2}$) have no "elementary" integral. We must use numerical approximations.
ü§ñ Why this matters for AI:
We can differentiate a loss function with a billion parameters instantly.
But integrating a probability distribution in a billion dimensions is basically impossible.
This is why Bayesian Deep Learning is so much harder than standard Deep Learning!
Section 10

Case Study: How GPS uses Integration

Your phone knows where you are because it integrates signals from satellites.

1. Dead Reckoning

When you go into a tunnel and lose GPS, your phone uses its Accelerometer. It integrates Acceleration ($a$) once to get Velocity ($v$), and integrates Velocity to get Position ($s$).

$s(t) = s_0 + \int_0^t (v_0 + \int_0^\tau a(\lambda) d\lambda) d\tau$

2. Drifting Errors

Integration accumulates error. If your accelerometer is off by just 0.01 m/s¬≤, after 1 minute of integration, your estimated position will be off by 18 meters. This is why "Sensory Fusion" (Kalman Filters) is needed to correct the integral.

üõ∞Ô∏è Relativity:
GPS satellites also have to integrate "Time Dilation" effects. Without integrating Einstein's equations, GPS would be off by 10km a day!
Python Trapezoid Rule 
import numpy as np

def trapezoid(f, a, b, n=1000):
    x = np.linspace(a, b, n+1)
    y = f(x)
    dx = (b - a) / n
    return np.trapz(y, x)

result = trapezoid(lambda x: x**2, 0, 3)
print(f"Trapezoid: {result:.6f}")

2D Integration: Calculating Volume

What if the ground isn't a line, but a floor? We sum up Columns instead of Rectangles.

$V = \iint_R f(x, y) dA$

Double Riemann Sums

Instead of dividing a line into segments $\Delta x$, we divide a plane into a grid of squares $\Delta x \cdot \Delta y$.

Iterated Integrals

$\int_{y_1}^{y_2} [ \int_{x_1}^{x_2} f(x, y) dx ] dy$

Integrate inside-out, treating one variable as a constant at a time.

üß± Analogy: Minecraft
Finding the volume of a Minecraft mountain is a Double Integral.
Each block is a column with area 1. You just sum up the heights of every block in the grid.

Glossary

Core Concepts:
‚Ä¢ Integral: Accumulated sum, area under curve
‚Ä¢ Antiderivative: F'(x) = f(x)
‚Ä¢ FTC: $\int_a^b f(x)dx = F(b) - F(a)$
‚Ä¢ Averaging: $\frac{1}{b-a} \int_a^b f(x)dx$
Numerical Methods:
‚Ä¢ Riemann: Sum of rectangles
‚Ä¢ Trapezoid: Sum of trapezoids
‚Ä¢ Simpson: Parabolic approximation
‚Ä¢ Monte Carlo: Random dart throwing
Common Gotchas:
‚Ä¢ Don't forget +C: Indefinite integrals have a constant.
‚Ä¢ Numerical errors: More slices = more accuracy, but diminishing returns.
‚Ä¢ Some integrals have no closed form: e.g., $\int e^{-x^2}dx$

"Integration is just a for loop with infinitely many infinitely small terms."
