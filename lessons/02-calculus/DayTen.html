Section 1

Critical Points

To find the maximum or minimum of a smooth function, we look for places where the slope is zero. Imagine standing on top of a hill; the ground is flat.

Fermat's Theorem: If $f$ has a local extremum at $c$, and $f'(c)$ exists, then $f'(c) = 0$.

Points where $f'(c) = 0$ (or undefined) are called Critical Points.

Section 2

Second Derivative Test

Once we find a point where $f'(x) = 0$, is it a min or a max? We check the curvature (concavity).

  • Concave Up ($f''(x) > 0$): Like a cup $\cup$. Point is a Minimum.
  • Concave Down ($f''(x) < 0$): Like a frown $\cap$. Point is a Maximum.
f(x) = x^2 \implies f'(x) = 2x. Critical point at x=0. f''(x) = 2 > 0 \implies Minimum.
Section 3

Gradient Descent

For complex functions (like Neural Networks with millions of variables), we can't solve $f'(x)=0$ algebraically. Instead, we "roll down the hill".

The Algorithm: $$ x_{new} = x_{old} - \alpha \cdot f'(x_{old}) $$
$\alpha$ (Alpha) is the Learning Rate (step size). We subtract the derivative because we want to move against the slope to go down.
Section 4

Implementation

Let's implement Gradient Descent to find the minimum of $f(x) = (x-3)^4$ starting from $x=0$.

PythonGradient Descent
def gradient_descent(start_x, learning_rate, epochs):
    x = start_x
    for i in range(epochs):
        # Derivative of (x-3)^4 is 4(x-3)^3
        grad = 4 * (x - 3)**3
        
        # Update Step
        x = x - learning_rate * grad
        
        if i % 10 == 0:
            print(f"Step {i}: x = {x:.4f}, f(x) = {(x-3)**4:.4f}")
            
    return x

min_x = gradient_descent(0, 0.001, 100)
print("Minimum found at:", min_x) # Should approach 3
