Section 1

The Need for Limits

Consider the function $f(x) = \frac{x^2 - 1}{x - 1}$. If you plug in $x = 1$, you get $\frac{0}{0}$, which is undefined. However, if you plug in numbers close to 1:

PythonNumerical Approximation
def f(x):
    return (x**2 - 1) / (x - 1)

for x in [0.9, 0.99, 0.999, 1.001, 1.01]:
    print(f(f"f({x}) = {f(x)}")

The values approach 2. We say the limit as $x$ approaches 1 is 2.

Section 2

Precise Definition (Epsilon-Delta)

We write $\lim_{x \to a} f(x) = L$ if we can keep $f(x)$ arbitrarily close to $L$ by keeping $x$ sufficiently close to $a$.

The Definition:
For every $\epsilon > 0$ (tolerance), there exists a $\delta > 0$ (precision) such that:
If $0 < |x - a| < \delta$, then $|f(x) - L| < \epsilon$.
This is arguably the most important definition in Analysis. It converts "approaching" into static algebra.
Section 3

Limit Laws

Limits behave nicely with arithmetic.

  • Sum: $\lim (f + g) = \lim f + \lim g$
  • Product: $\lim (f \cdot g) = \lim f \cdot \lim g$
  • Quotient: $\lim (f / g) = \lim f / \lim g$ (provided $\lim g \neq 0$)
Example: \lim_{x \to 2} (x^2 + 5x) = \lim_{x \to 2} x^2 + \lim_{x \to 2} 5x = 2^2 + 5(2) = 14
Section 4

Continuity

A function is continuous at $a$ if:

\lim_{x \to a} f(x) = f(a)

This implies three things:

  1. $f(a)$ is defined.
  2. The limit exists.
  3. They are equal.
Intermediate Value Theorem: If $f$ is continuous on $[a, b]$ and $k$ is between $f(a)$ and $f(b)$, there is some $c \in [a, b]$ such that $f(c) = k$. This is used to prove roots exist (like determining if a polynomial crosses zero).
