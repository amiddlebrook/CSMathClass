Section 1

Multivariable Limits

For $f(x, y)$, a limit exists at $(a, b)$ only if every path to $(a, b)$ gives the same value.

$$ \lim_{(x,y) \to (a,b)} f(x, y) = L $$ means approaching $(a,b)$ from any direction yields $L$.
Path dependence: To show a limit DNE, find two paths that give different values.
Example: Does lim(x,y)→(0,0) xy/(x² + y²) exist? Path y = 0: lim = 0·0/(x² + 0) = 0 Path y = x: lim = x·x/(x² + x²) = x²/2x² = 1/2 Different values → Limit DNE ✗
Section 2

Partial Derivatives

Differentiate with respect to one variable, treating all others as constants.

Notation: $$ \frac{\partial f}{\partial x} = f_x = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h} $$
Example: f(x, y) = x²y + 3xy² ∂f/∂x = 2xy + 3y² (treat y as constant) ∂f/∂y = x² + 6xy (treat x as constant)
Higher partials: $f_{xy} = \frac{\partial^2 f}{\partial y \partial x}$ (differentiate w.r.t. x first, then y). Clairaut's Theorem: $f_{xy} = f_{yx}$ if both are continuous.
Section 3

The Gradient Vector

Collect all partial derivatives into a vector:

Gradient: $$ \nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right) $$

Key properties:

  • $\nabla f$ points in the direction of steepest ascent
  • $-\nabla f$ points in the direction of steepest descent (used in gradient descent!)
  • $|\nabla f|$ gives the rate of change in that direction

Computing Gradients in Python

Both symbolic (SymPy) and numerical approaches:

PythonGradients
from sympy import symbols, diff, Matrix

# Symbolic gradient
x, y = symbols('x y')
f = x**2 * y + 3*x*y**2

grad_f = Matrix([diff(f, x), diff(f, y)])
print("∇f =", grad_f)  # [2*x*y + 3*y**2, x**2 + 6*x*y]

# Numerical gradient (finite differences)
import numpy as np

def numerical_gradient(f, point, h=1e-5):
    grad = []
    for i in range(len(point)):
        p_plus = point.copy()
        p_plus[i] += h
        grad.append((f(p_plus) - f(point)) / h)
    return np.array(grad)

# Test on f(x,y) = x² + y²
f_num = lambda p: p[0]**2 + p[1]**2
print(numerical_gradient(f_num, np.array([1.0, 2.0])))  # [2.0, 4.0]
