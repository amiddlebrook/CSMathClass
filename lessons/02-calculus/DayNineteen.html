Section 1

Gradient (∇f)

The gradient of a scalar field $f(x, y, z)$ is a vector pointing in the direction of steepest increase:

$$ \nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right) $$

Input: Scalar field → Output: Vector field

ML Connection: Gradient descent uses $-\nabla f$ to find minima. The gradient tells us which way is "uphill" on the loss surface.
Section 2

Divergence (∇·F)

The divergence of a vector field $\mathbf{F} = (F_x, F_y, F_z)$ measures how much it "spreads out" from a point:

$$ \nabla \cdot \mathbf{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z} $$

Input: Vector field → Output: Scalar field

• div > 0: Source (field flows outward) • div < 0: Sink (field flows inward) • div=0: Incompressible flow
Section 3

Curl (∇×F)

The curl measures the rotation or "swirl" of a vector field:

$$ \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \partial_x & \partial_y & \partial_z \\ F_x & F_y & F_z \end{vmatrix} $$

Input: Vector field → Output: Vector field

Physics: Maxwell's equations use curl to describe how electric fields create magnetic fields and vice versa.
Section 4

Computing in Python

Use SymPy for symbolic vector calculus:

PythonVector Calculus
from sympy import symbols, diff
from sympy.vector import CoordSys3D, divergence, curl, gradient

# Set up 3D coordinate system
C = CoordSys3D('C')
x, y, z = C.x, C.y, C.z

# Scalar field
f = x**2 + y**2 + z**2
print("∇f =", gradient(f))  # 2x i + 2y j + 2z k

# Vector field F = (xy, yz, xz)
F = x*y*C.i + y*z*C.j + x*z*C.k

print("∇·F =", divergence(F))  # y + z + x
print("∇×F =", curl(F))  # (-y)i + (-z)j + (-x)k
