Section 1

Sequences

A sequence is an ordered list of numbers: $a_1, a_2, a_3, \ldots$

Convergence: A sequence $\{a_n\}$ converges to $L$ if: $$ \lim_{n \to \infty} a_n = L $$ Otherwise, it diverges.
Examples: • aₙ = 1/n → 0 (converges) • aₙ = n → ∞ (diverges) • aₙ = (-1)ⁿ → oscillates (diverges)
Section 2

Series

A series is the sum of a sequence's terms:

$$ \sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \cdots $$

Geometric Series

The most important series: $\sum_{n=0}^{\infty} r^n$

Converges to $\frac{1}{1-r}$ if $|r| < 1$. Diverges otherwise.

Harmonic Series

$\sum_{n=1}^{\infty} \frac{1}{n} = 1 + \frac{1}{2} + \frac{1}{3} + \cdots$

Diverges! Even though $\frac{1}{n} \to 0$, the sum grows unboundedly (slowly).
Section 3

Convergence Tests

Divergence Test

If $\lim_{n \to \infty} a_n \neq 0$, then $\sum a_n$ diverges.

The converse is FALSE! ($a_n \to 0$ does NOT guarantee convergence.)

Ratio Test

Let $L = \lim_{n \to \infty} \left| \frac{a_{n+1}}{a_n} \right|$
  • If $L < 1$: Converges (absolutely)
  • If $L > 1$: Diverges
  • If $L = 1$: Inconclusive

p-Series Test

$\sum \frac{1}{n^p}$ converges if $p > 1$, diverges if $p \leq 1$.

Section 4

Partial Sums in Python

Visualize convergence by computing partial sums:

PythonSeries Exploration
import numpy as np

def partial_sum(terms, n):
    """Compute sum of first n terms."""
    return sum(terms(k) for k in range(1, n+1))

# Geometric series: 1/2 + 1/4 + 1/8 + ...
geo = lambda n: (1/2)**n
print(f"Geometric S₁₀ = {partial_sum(geo, 10):.6f}")  # → 0.999...

# Harmonic series: 1 + 1/2 + 1/3 + ... (diverges!)
harm = lambda n: 1/n
print(f"Harmonic S₁₀₀ = {partial_sum(harm, 100):.2f}")  # → ~5.19
print(f"Harmonic S₁₀₀₀ = {partial_sum(harm, 1000):.2f}")  # → ~7.49

# p-series with p=2 (converges to π²/6)
p2 = lambda n: 1/n**2
print(f"p=2 series S₁₀₀₀ = {partial_sum(p2, 1000):.6f}")  # → 1.6439...
print(f"π²/6 = {np.pi**2/6:.6f}")  # → 1.6449...
