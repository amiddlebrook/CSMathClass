Section 1

Double Integrals

A double integral sums $f(x, y)$ over a 2D region $R$:

$$ \iint_R f(x, y) \, dA $$

If $f(x, y) = 1$, this gives the area of $R$. If $f$ is height, it gives volume.

Interpretation: Just like 1D integration sums infinitely thin rectangles, 2D integration sums infinitely small boxes of height $f(x,y)$ and base $dA = dx \, dy$.
Section 2

Iterated Integrals

We evaluate double integrals by integrating one variable at a time:

Fubini's Theorem: $$ \iint_R f(x,y) \, dA = \int_a^b \int_{c(x)}^{d(x)} f(x, y) \, dy \, dx $$
Example: ∬ xy dA over R = [0,1] × [0,2] = ∫₀¹ ∫₀² xy dy dx = ∫₀¹ x[y²/2]₀² dx = ∫₀¹ x(2) dx = ∫₀¹ 2x dx = [x²]₀¹ = 1
Section 3

Polar Coordinates

For circular regions, convert to polar: $x = r\cos\theta$, $y = r\sin\theta$.

Jacobian: $dA = dx \, dy = r \, dr \, d\theta$

The extra factor of $r$ accounts for the stretching of coordinates.

Example: Area of circle radius R ∬ 1 dA = ∫₀²π ∫₀ᴿ r dr dθ = ∫₀²π [r²/2]₀ᴿ dθ = ∫₀²π R²/2 dθ = πR² ✓
Section 4

Numerical Double Integration

Use SciPy's dblquad for numerical evaluation:

PythonDouble Integrals
from scipy.integrate import dblquad
import numpy as np

# ∬ xy dA over [0,1] × [0,2]
result, error = dblquad(
    lambda y, x: x * y,  # inner (y) first!
    0, 1,                 # x bounds
    lambda x: 0,          # y lower
    lambda x: 2           # y upper
)
print(f"Result: {result}")  # 1.0

# Area of unit circle
area, _ = dblquad(
    lambda y, x: 1,
    -1, 1,
    lambda x: -np.sqrt(1 - x**2),
    lambda x: np.sqrt(1 - x**2)
)
print(f"Circle area: {area:.4f}")  # 3.1416 ≈ π
Section 5

Gradient (∇f)

The gradient of a scalar field $f(x, y, z)$ is a vector pointing in the direction of steepest increase:

$$ \nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right) $$

Input: Scalar field → Output: Vector field

ML Connection: Gradient descent uses $-\nabla f$ to find minima.
Section 6

Divergence (∇·F)

The divergence of a vector field $\mathbf{F} = (F_x, F_y, F_z)$ measures how much it "spreads out" from a point:

$$ \nabla \cdot \mathbf{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z} $$

Input: Vector field → Output: Scalar field

• div > 0: Source (field flows outward) • div < 0: Sink (field flows inward) • div=0: Incompressible flow
Section 7

Curl (∇×F)

The curl measures the rotation or "swirl" of a vector field:

$$ \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \partial_x & \partial_y & \partial_z \\ F_x & F_y & F_z \end{vmatrix} $$

Input: Vector field → Output: Vector field

Physics: Maxwell's equations use curl to describe how electric fields create magnetic fields and vice versa.
Section 8

Vector Calculus in Python

Use SymPy for symbolic vector calculus:

PythonVector Calculus
from sympy import symbols
from sympy.vector import CoordSys3D, divergence, curl, gradient

# Set up 3D coordinate system
C = CoordSys3D('C')
x, y, z = C.x, C.y, C.z

# Scalar field
f = x**2 + y**2 + z**2
print("∇f =", gradient(f))  # 2x i + 2y j + 2z k

# Vector field F = (xy, yz, xz)
F = x*y*C.i + y*z*C.j + x*z*C.k
print("∇·F =", divergence(F))  # y + z + x
print("∇×F =", curl(F))
