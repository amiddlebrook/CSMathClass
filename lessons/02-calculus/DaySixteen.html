Section 1

Taylor Series

A Taylor Series represents a function $f(x)$ as an infinite sum of polynomial terms, centered at a point $a$:

Taylor Series: $$ f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x - a)^n $$

Each term uses the $n$-th derivative at the center point $a$. The factorial $n!$ controls the rate of decay.

Why it works: We're "matching" all the derivatives of $f$ at $a$ with a polynomial that has the same derivatives at $a$.

Maclaurin Series (a = 0)

When centered at $a = 0$, we call it a Maclaurin Series. Here are the essential ones:

eˣ = 1 + x + x²/2! + x³/3! + ... sin(x) = x - x³/3! + x⁵/5! - ... cos(x) = 1 - x²/2! + x⁴/4! - ... 1/(1-x) = 1 + x + x² + x³ + ... (|x| < 1) ln(1+x)=x - x²/2 + x³/3 - ... (|x| ≤ 1)
Memorize these! They appear constantly in physics, engineering, and CS.
Section 3

Error & Convergence

When we truncate after $n$ terms, we get the Taylor Polynomial $T_n(x)$. The error is:

Lagrange Error Bound: $$ |R_n(x)| \leq \frac{M}{(n+1)!} |x - a|^{n+1} $$ where $M = \max|f^{(n+1)}(c)|$ for $c$ between $a$ and $x$.
CS Connection: Libraries compute $e^x$, $\sin(x)$, etc. using truncated Taylor polynomials with error bounds to guarantee precision.
Section 4

Taylor Approximation in Python

Let's implement Taylor approximations for $e^x$ and $\sin(x)$:

PythonTaylor Series
import math

def taylor_exp(x, n_terms=10):
    """Approximate e^x using Taylor series."""
    return sum(x**n / math.factorial(n) for n in range(n_terms))

def taylor_sin(x, n_terms=10):
    """Approximate sin(x) using Taylor series."""
    return sum(
        ((-1)**n * x**(2*n+1)) / math.factorial(2*n+1)
        for n in range(n_terms)
    )

# Test
x = 1.0
print(f"Taylor e^1 = {taylor_exp(x):.10f}")
print(f"Actual e^1 = {math.exp(x):.10f}")

x = math.pi / 4
print(f"Taylor sin(π/4) = {taylor_sin(x):.10f}")
print(f"Actual sin(π/4) = {math.sin(x):.10f}")
