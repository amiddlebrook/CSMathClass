Section 1

FTC Part 1: Differentiation Undoes Integration

Define an "accumulator function" $F(x) = \int_a^x f(t) \, dt$. This function measures the area under $f$ from a fixed point $a$ up to a variable point $x$.

FTC Part 1: If $f$ is continuous on $[a, b]$, then: $$ \frac{d}{dx} \left[ \int_a^x f(t) \, dt \right] = f(x) $$
Intuition: The rate of change of the accumulated area at $x$ is exactly the height of the curve $f(x)$ at that point. If the curve is taller, area accumulates faster!
Section 2

FTC Part 2: The Evaluation Theorem

If we can find an antiderivative $F(x)$ such that $F'(x) = f(x)$, then we can compute definite integrals without limits or sums!

FTC Part 2: If $F'(x) = f(x)$, then: $$ \int_a^b f(x) \, dx = F(b) - F(a) $$
Notation: We write $F(x) \Big|_a^b$ or $[F(x)]_a^b$ to mean $F(b) - F(a)$.
Example: ∫₀¹ x² dx Antiderivative of x² is F(x) = x³/3 Answer: F(1) - F(0) = 1/3 - 0 = 1/3 ✓
Section 3

Common Antiderivatives

Finding antiderivatives is the reverse of differentiation. Here are the essential ones:

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (n ≠ -1) ∫ 1/x dx = ln|x| + C ∫ eˣ dx = eˣ + C ∫ sin(x) dx = -cos(x) + C ∫ cos(x) dx = sin(x) + C
The +C: For indefinite integrals (no bounds), we add a constant $C$ because the derivative of a constant is zero. $F(x) + C$ is a family of antiderivatives.
Section 4

Symbolic vs Numerical Integration

FTC gives us an exact answer using symbolic antiderivatives. Here's how to use Python's SymPy for symbolic integration:

PythonSymbolic Integration
from sympy import symbols, integrate, sin, cos, exp

x = symbols('x')

# Indefinite integral (returns antiderivative)
print(integrate(x**2, x))       # x**3/3

# Definite integral (returns number)
print(integrate(x**2, (x, 0, 1)))  # 1/3

# Compare symbolic vs numerical
from scipy.integrate import quad
result, _ = quad(lambda x: x**2, 0, 1)
print(f"Numerical: {result}")    # 0.33333...
CS Connection: Computer Algebra Systems (CAS) like SymPy, Mathematica, and WolframAlpha use pattern matching and heuristics to find antiderivatives symbolically. When no closed form exists, they fall back to numerical methods.
