Section 1

BFS & DFS

BFS (Breadth-First)

• Uses Queue • Level by level • Finds shortest path (unweighted) • O(V + E)

DFS (Depth-First)

• Uses Stack/Recursion • Go deep before wide • Cycle detection, topological sort • O(V + E)
Python BFS and DFS 
from collections import deque

def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    while queue:
        node = queue.popleft()
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

def dfs(graph, node, visited=None):
    if visited is None: visited = set()
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

Shortest Paths

ALGORITHM | TIME | WEIGHTS | NEGATIVE -----------------|--------------|------------|---------- BFS | O(V + E) | Unweighted | N/A Dijkstra | O((V+E)logV) | Non-neg | No Bellman-Ford | O(VE) | Any | Yes Floyd-Warshall | O(V³) | Any | Yes (all-pairs)
Dijkstra fails with negative weights! Use Bellman-Ford instead.
Section 3

Minimum Spanning Tree

MST: Connect all vertices with minimum total edge weight.
Prim's: Grow tree one vertex at a time (greedy) O((V+E) log V) with binary heap Kruskal's: Sort edges, add if no cycle (greedy) O(E log E) with Union-Find
Section 4

Loop Invariants

Loop Invariant: Property that's true before/after each iteration.
PROVING CORRECTNESS: 1. Initialization: Invariant true before first iteration 2. Maintenance: If true before iteration, true after 3. Termination: When loop ends, invariant implies correctness
Section 5

Mathematical Induction

INDUCTION STEPS: 1. Base case: Prove P(1) or P(0) 2. Inductive step: Assume P(k), prove P(k+1) 3. Conclusion: P(n) holds for all n ≥ base STRONG INDUCTION: Assume P(1), P(2), ..., P(k) to prove P(k+1)

Big-O Analysis

NOTATION | MEANING ------------|------------------ O(f) | Upper bound (at most) Ω(f) | Lower bound (at least) Θ(f) | Tight bound (exactly) o(f) | Strictly less than ω(f) | Strictly greater than COMMON COMPLEXITIES: O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ) < O(n!)
Drop constants and lower terms: 3n² + 2n + 5 = O(n²)
Section 7

Solving Recurrences

MASTER THEOREM for T(n) = aT(n/b) + f(n): Case 1: f(n) = O(n^(log_b(a) - ε)) → T(n) = Θ(n^log_b(a)) Case 2: f(n) = Θ(n^log_b(a)) → T(n) = Θ(n^log_b(a) · log n) Case 3: f(n) = Ω(n^(log_b(a) + ε)) → T(n) = Θ(f(n)) Examples: • Merge sort: T(n) = 2T(n/2) + n → Θ(n log n) • Binary search: T(n) = T(n/2) + 1 → Θ(log n)
