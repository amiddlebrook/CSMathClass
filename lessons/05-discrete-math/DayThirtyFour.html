Python Quick Sort 
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    mid = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + mid + quicksort(right)

Searching

Linear Search

O(n) - check every element Works on unsorted data

Binary Search

O(log n) - halve search space Requires sorted data
Python Binary Search 
def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
Section 3

Greedy Algorithms

Greedy: Make locally optimal choice at each step, hoping for global optimum.
Classic Examples: • Activity Selection • Huffman Coding • Coin Change (certain denominations) • Minimum Spanning Tree (Prim, Kruskal) • Dijkstra's Shortest Path
Greedy doesn't always work! Must prove greedy choice property and optimal substructure.
Section 4

Divide & Conquer

D&C: Divide problem into subproblems, solve recursively, combine solutions.
Classic Examples: • Merge Sort: T(n) = 2T(n/2) + O(n) = O(n log n) • Quick Sort: T(n) = 2T(n/2) + O(n) average • Binary Search: T(n) = T(n/2) + O(1) = O(log n) • Strassen's Matrix Mult: O(n^2.807) Master Theorem: T(n) = aT(n/b) + f(n)
