Section 1

Euclidean Algorithm

To find $GCD(a, b)$:
If $b=0$, return $a$.
Else, return $GCD(b, a \pmod b)$.

Bézout's Identity: There exist integers $x, y$ such that $ax + by = GCD(a, b)$. We find them using the Extended Euclidean Algorithm.
Section 2

Modular Inverse

We want to solve $ax \equiv 1 \pmod m$.
The solution exists if and only if $GCD(a, m) = 1$.
The inverse $x$ is the coefficient found by Extended Euclid.

Example: Find inverse of 3 mod 7. 3 * 5 = 15 = 2 * 7 + 1. So 3 * 5 ≡ 1 mod 7. Inverse is 5.
Section 3

RSA Encryption

Key Generation

  1. Pick two large primes $p, q$.
  2. Compute $n = pq$ and $\phi(n) = (p-1)(q-1)$.
  3. Choose public exponent $e$ such that $GCD(e, \phi(n)) = 1$ (usually 65537).
  4. Compute private key $d$ such that $ed \equiv 1 \pmod{\phi(n)}$.

Encryption / Decryption

  • Encrypt: $C = M^e \pmod n$
  • Decrypt: $M = C^d \pmod n$

Security relies on the difficulty of factoring $n$ back into $p$ and $q$.

Section 4

Python Implementation

Python Mini RSA 
def extended_gcd(a, b):
    if a == 0:
        return b, 0, 1
    gcd, x1, y1 = extended_gcd(b % a, a)
    x = y1 - (b // a) * x1
    y = x1
    return gcd, x, y

def mod_inverse(a, m):
    gcd, x, y = extended_gcd(a, m)
    if gcd != 1:
        raise Exception("Inverse doesn't exist")
    return (x % m + m) % m

# RSA Setup
p = 61
q = 53
n = p * q # 3233
phi = (p-1) * (q-1) # 3120

e = 17 # Public exponent
d = mod_inverse(e, phi) # Private key

print(f"Public Key: (e={e}, n={n})")
print(f"Private Key: (d={d}, n={n})")

# Message
message = 123
ciphertext = pow(message, e, n)
decrypted = pow(ciphertext, d, n)

print(f"Original: {message}")
print(f"Encrypted: {ciphertext}")
print(f"Decrypted: {decrypted}")
