Section 1

Mean Value Theorem

Rolle's Theorem

If f is: • Continuous on [a, b] • Differentiable on (a, b) • f(a) = f(b) Then ∃c ∈ (a, b) such that f'(c) = 0 Intuition: If you start and end at the same height, you must have a horizontal tangent somewhere.

Mean Value Theorem

MVT: Under the same conditions (continuous on [a,b], differentiable on (a,b)), there exists c ∈ (a,b) such that:
f'(c) = [f(b) - f(a)] / (b - a) Intuition: At some point, the instantaneous rate of change equals the average rate of change. APPLICATIONS: • Speed: At some point, you traveled at exactly your average speed • Proving inequalities (e.g., sin(x) < x for x> 0) • Error bounds in numerical analysis
Python Finding MVT point numerically 
import numpy as np
from scipy.optimize import brentq

def find_mvt_point(f, f_prime, a, b):
    """Find c where f'(c) = [f(b)-f(a)]/(b-a)"""
    avg_slope = (f(b) - f(a)) / (b - a)
    
    # Find c where f'(c) = avg_slope
    g = lambda x: f_prime(x) - avg_slope
    c = brentq(g, a, b)
    return c, avg_slope

# Example: f(x) = x² on [0, 2]
f = lambda x: x**2
f_prime = lambda x: 2*x

c, slope = find_mvt_point(f, f_prime, 0, 2)
print(f"MVT point: c = {c}")  # c = 1
print(f"f'(c) = avg slope = {slope}")  # f'(1) = 2 = (4-0)/(2-0)

Fundamental Theorem of Calculus

Part 1: Differentiation of integrals

FTC Part 1: If f is continuous on [a, b] and F(x) = ∫ₐˣ f(t) dt, then F'(x) = f(x).
d/dx ∫ₐˣ f(t) dt = f(x) Meaning: The derivative of an integral (with variable upper limit) gives back the original function. CHAIN RULE VERSION: d/dx ∫ₐ^g(x) f(t) dt = f(g(x)) · g'(x)

Part 2: Evaluation of integrals

FTC Part 2: If F is any antiderivative of f (i.e., F' = f), then ∫ₐᵇ f(x) dx = F(b) - F(a).
∫ₐᵇ f(x) dx = F(b) - F(a) where F' = f Meaning: To compute a definite integral, find any antiderivative and evaluate at the endpoints. NOTATION: F(x)|ₐᵇ = F(b) - F(a) EXAMPLE: ∫₀¹ x² dx = [x³/3]₀¹ = 1/3 - 0 = 1/3
The Deep Connection: FTC says differentiation and integration are inverse operations. This is why we can compute areas using antiderivatives!
Section 3

Applications

L'Hôpital's Rule

If lim f(x)/g(x) gives 0/0 or ∞/∞: lim f(x)/g(x) = lim f'(x)/g'(x) (if the right limit exists) EXAMPLES: lim(x→0) sin(x)/x = lim cos(x)/1 = 1 lim(x→∞) ln(x)/x = lim (1/x)/1 = 0
Python Numerical integration (FTC in practice) 
from scipy import integrate
import numpy as np

# Numerical integration with scipy
f = lambda x: x**2
result, error = integrate.quad(f, 0, 1)
print(f"∫₀¹ x² dx = {result:.10f}")  # 0.3333333333

# Compare with exact: 1/3
print(f"Exact: {1/3:.10f}")

# Gaussian integral (no elementary antiderivative!)
result, _ = integrate.quad(lambda x: np.exp(-x**2), -np.inf, np.inf)
print(f"∫ e^(-x²) dx = {result:.6f}")  # √π ≈ 1.772...
print(f"√π = {np.sqrt(np.pi):.6f}")
