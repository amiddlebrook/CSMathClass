Section 1

The Swap Problem

Suppose you have a sequence of functions `f_n(x)` converging to `f(x)`. Is it true that:

\lim_{n \to \infty} \int f_n \, d\mu = \int \lim_{n \to \infty} f_n \, d\mu
ðŸ§© Simple: Mathematical concepts are like building blocks. Each new idea builds on previous ones, so mastering the fundamentals makes everything else easier to understand.

Not always! Example: "Moving Bump". Let `f_n` be a bump of height 1 and width 1 shifting to infinity. `\lim f_n(x) = 0` pointwise. But `\int f_n = 1` for all `n`. So `\lim \int = 1 \ne 0`.

Section 2

Monotone Convergence Theorem (MCT)

The first big gun.

Theorem: If `0 \le f_1 \le f_2 \le \dots` is an increasing sequence of measurable functions, then:
`\lim_{n \to \infty} \int f_n = \int \lim_{n \to \infty} f_n`

Crucial: Functions must be non-negative and increasing.

Section 3

Fatou's Lemma

An inequality that always holds for non-negative functions.

\int \liminf_{n \to \infty} f_n \le \liminf_{n \to \infty} \int f_n

Mnemonic: "Integration makes things bigger/stable". Mixing things up (limit) might lose mass (like the escaping bump), so the integral of the limit is smaller.

Section 4

Dominated Convergence Theorem (DCT)

The most useful one.

Theorem: If `f_n \to f` pointwise, and there exists a "dominating" function `g` such that `|f_n(x)| \le g(x)` for all `n, x` and `\int g < \infty`, then:
`\lim \int f_n = \int f`.

The function `g` acts as a "ceiling" that prevents mass from escaping to infinity or exploding.

â˜” Simple: The Umbrella Analogy.
Imagine a crowd of people (the sequence `f_n`) running around in the rain.
If everyone stays under a giant tent (the dominating function `g` with finite area), no one can run "too far" or "too high" to escape.
Therefore, the crowd's final formation `f` has the same total area as the limit of their individual areas.
Section 5

Summary of Conditions

Theorem Requires Result
MCT `0 \le f_1 \le f_2 \dots` Equality
Fatou `f_n \ge 0` Inequality (`\le`)
DCT `|f_n| \le g` (integrable) Equality
Section 6

Python: Converging Bumps

Python Sequence Limit Check 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 10, 1000)

# Example 1: Typewriter / Traveling Bump
# f_n(x) = 1 if x in [n, n+1], else 0
# Pointwise limit: 0 (for any fixed x, bump eventually passes)
# Integral limit: 1 (mass is preserved)
# Violation: No dominating function g(x) with finite integral covers all bumps up to infinity.

def bump(n, x):
    return np.where((x >= n) & (x < n+1), 1, 0)

# Example 2: Squashing Bump
# f_n(x) = n * 1_[0, 1/n]
# Pointwise limit: 0 (every x > 0 eventually outside 1/n)
# Integral limit: 1
# Violation: Bound grows to infinity at 0.

n_values = [1, 2, 3, 5]
plt.figure(figsize=(10, 4))
for n in n_values:
    plt.plot(x, bump(n, x), label=f'n={n}')

plt.title("Traveling Bumps (Limit of Integral != Integral of Limit)")
plt.legend()
plt.show()

# DCT Check
# If we had f_n(x) = exp(-n * x), dominated by exp(-x) (integrable)
# Then limits would match.
