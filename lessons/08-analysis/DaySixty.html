Section 1

The Probability Triplet

In undergraduate stats, we wave our hands about "chance". In rigorous math, Probability is just Measure Theory on a space with total mass 1.

Definition: A Probability Space is a triple `(\Omega, \mathcal{F}, P)` where:
  1. `\Omega`: Sample Space (Set of all outcomes).
  2. `\mathcal{F}`: Event Space (`\sigma`-algebra of `\Omega`).
  3. `P`: Probability Measure (`P(\Omega) = 1`, `P(\emptyset) = 0`, countably additive).

An "Event" is just a measurable set `E \in \mathcal{F}`.

Section 2

Random Variables

A Random Variable is neither random nor a variable. It is a function.

X: \Omega \to \mathbb{R}

However, it cannot be just any function. It must be Measurable.

For any Borel set `B \subseteq \mathbb{R}`, the set `\{\omega \in \Omega \mid X(\omega) \in B\}` must be an event in `\mathcal{F}`.

This ensures we can answer: "What is the probability that `X` is in `B`?"

P(X \in B) = P(X^{-1}(B)) = P(\{\omega \mid X(\omega) \in B\})
Section 3

Distributions (Push-forward Measure)

The random variable `X` "pushes" the measure `P` from `\Omega` onto `\mathbb{R}`.

We define the Distribution of X, denoted `\mu_X`, as:

\mu_X(B) = P(X^{-1}(B))

This `\mu_X` is a measure on `\mathbb{R}`. It is what we usually plot (PDF/CDF). The original `\Omega` (coin flips, particle states) often fades into the background, but it is always there.

Section 4

Independence

In classical probability: `P(A \cap B) = P(A)P(B)`.

In Measure Theory, `X` and `Y` are independent if the `\sigma`-algebras generated by them are independent.

\sigma(X) = \{X^{-1}(B) \mid B \in \mathcal{B}\}

This definition works for continuous, discrete, or mixed variables without needing separate definitions.

Section 5

Expectation as Integral

Definition: `E[X] = \int_{\Omega} X(\omega) \, dP(\omega)`

This is the Lebesgue integral of the function `X` with respect to the measure `P`. It unifies the definitions:

  • Discrete: `\sum x p(x)` (Integral with respect to counting measure)
  • Continuous: `\int x f(x) dx` (Integral with respect to Lebesgue measure)
Section 6

LOTUS

Law of the Unconscious Statistician: To find `E[g(X)]`, you don't need the distribution of `Y = g(X)`. You can use the distribution of `X`.

E[g(X)] = \int_{\Omega} g(X(\omega)) \, dP(\omega) = \int_{\mathbb{R}} g(x) \, d\mu_X(x)

This effectively says expectation is linear.

Section 7

Conditional Expectation

`Z = E[X|Y]` is not a number, but a Random Variable. It is `\sigma(Y)`-measurable and satisfies:

\int_A Z \, dP = \int_A X \, dP \quad \forall A \in \sigma(Y)

Intuitively, it is the best guess of `X` given the information in `Y` (orthogonal projection in `L^2` space).

Section 8

Python: Measurable Mapping & LOTUS

Python RV Mapping & LOTUS Check 
import numpy as np

# Omega method vs PDF method

# 1. Omega Method (Monte Carlo on Sample Space)
# Omega is [0, 1]. X(omega) = -ln(1-omega) (Exponential RV)
n_samples = 100000
omegas = np.random.uniform(0, 1, n_samples)
X_samples = -np.log(1 - omegas)

# Check Mean (E[X])
print(f"E[X] (Mean): {np.mean(X_samples):.4f} (Expected 1.0)")

# Check LOTUS: E[X^2]
# g(x) = x^2
g_samples = X_samples**2
E_g_X = np.mean(g_samples)

print(f"E[X^2] via Monte Carlo: {E_g_X:.4f}")
# Theoretical Variance of Exp(1) is 1. Mean is 1. E[X^2] = Var + Mean^2 = 2.
print(f"Theoretical E[X^2]: 2.0000")
