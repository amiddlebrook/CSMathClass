Section 1

Normed Vector Spaces

Functional Analysis is the study of vector spaces equipped with a structure (norm, inner product) that allows us to do analysis (limits).

Definition: A Norm `\| \cdot \|` satisfies:
  1. Positivity: `\|x\| \ge 0`, `\|x\|=0 \iff x=0`.
  2. Homogeneity: `\|\alpha x\| = |\alpha| \|x\|`.
  3. Triangle Inequality: `\|x+y\| \le \|x\| + \|y\|`.
üß© Simple: Mathematical concepts are like building blocks. Each new idea builds on previous ones, so mastering the fundamentals makes everything else easier to understand.
Section 2

Banach Spaces

In finite dimensions (`\mathbb{R}^n`), every normed space is complete (Cauchy sequences converge). In infinite dimensions, this is NOT true (e.g., polynomials with sup norm often converge to non-polynomial continuous functions).

Definition: A Banach Space is a normed vector space that is Complete.

Why care? Calculus requires limits. If your space has "holes", you can't solve differential equations (the solution might fall into a hole). Banach spaces are "safe" spaces.

Section 3

Hilbert Spaces

A Hilbert Space is a Banach space where the norm comes from an Inner Product `\langle x, y \rangle`.

\|x\| = \sqrt{\langle x, x \rangle}

Hilbert Spaces are the nicest possible infinite-dimensional spaces. They have:

  • Angles and Orthogonality.
  • pythagorean Theorem.
  • Projections (closest point intuition).
üõ∞Ô∏è Simple: The "GPS" Analogy.
‚Ä¢ In 3D space (`\mathbb{R}^3`), we find our position using 3 coordinates (Projections onto x, y, z axes).
‚Ä¢ A Hilbert Space is just a space where we can do this with infinite coordinates.
‚Ä¢ We can measure angles, find distances, and decompose a signal into "infinite axes" (like Fourier Series uses sine waves as axes). It's the only infinite-dimensional space where "Geometry" works like High School Geometry.
Section 4

L^p Spaces

The playground of modern analysis.

L^p(\Omega) = \left\{ f : \int_\Omega |f(x)|^p \, d\mu < \infty \right\}

Norm: `\|f\|_p = (\int |f|^p)^{1/p}`.

  • `L^1`: Absolutely integrable.
  • `L^2`: Square integrable (Energy). The only Hilbert space in the family.
  • `L^\infty`: Essentially bounded.

"Completeness of `L^p`" is the Fisher-Riesz Theorem (requires Lebesgue integral!).

Section 5

Schauder Basis

In Linear Algebra, a Hamel Basis means every vector is a finite sum. In Banach spaces, we allow infinite sums.

x = \sum_{n=1}^\infty a_n e_n

Fourier Series work because the trig functions form a basis for `L^2[-\pi, \pi]`.

Section 6

Bounded Linear Operators

A linear map `T: X \to Y` is "nice" if it doesn't blow up vectors too much.

Theorem: For linear maps, Continuous `\iff` Bounded.

Bounded means `\|Tx\| \le C \|x\|` for all `x`.

Differentiation `D = d/dx` is Unbounded! `D(\sin(nx)) = n \cos(nx)`. Limit blows up. This is why PDE theory is hard.
Section 7

Operator Norm

The space of operators `\mathcal{L}(X, Y)` is itself a normed space.

\|T\|_{op} = \sup_{\|x\|=1} \|Tx\|

It's the "maximum stretch factor" of the matrix/operator.

Section 8

Dual Spaces

The Dual Space `X^*` is the space of all continuous linear functionals `f: X \to \mathbb{R}`.

Examples:

  • If `X = \mathbb{R}^n`, `X^* \cong \mathbb{R}^n`.
  • If `X = L^p` ($1 < p < \infty$), `X^* \cong L^q` where `1/p + 1/q=1`.
  • If `X = L^1`, `X^* = L^\infty`.
Section 9

Riesz Representation Theorem

Theorem: Every continuous linear functional `f` on a Hilbert Space `H` looks like an inner product. `\exists ! y \in H` such that `f(x) = \langle x, y \rangle`.
Section 10

Hahn-Banach Theorem

Theorem: If you have a bounded linear functional on a subspace, you can extend it to the whole space without increasing its norm.

Python: L2 & Operators

Python L2 Projection & Operator Norm 
import numpy as np
import matplotlib.pyplot as plt

# 1. Fourier Approximation (Hilbert Space Projection)
# Approximating a Square Wave in L2
x = np.linspace(-np.pi, np.pi, 500)
f = np.where(x > 0, 1, -1)

def partial_sum(N, x):
    s = np.zeros_like(x)
    for n in range(1, N+1, 2):
        bn = 4 / (n * np.pi)
        s += bn * np.sin(n * x)
    return s

plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1)
plt.plot(x, f, 'k', label='Target')
plt.plot(x, partial_sum(3, x), label='N=3')
plt.plot(x, partial_sum(10, x), label='N=10')
plt.title("Hilbert Space Projection")
plt.legend()

# 2. Operator Norm (Matrix Max Stretch)
plt.subplot(1, 2, 2)
A = np.array([[1, 2], [0, 2]])
thetas = np.linspace(0, 2*np.pi, 100)
vecs = np.array([np.cos(thetas), np.sin(thetas)])
mapped = A @ vecs
# SVD for true norm
U, S, Vt = np.linalg.svd(A)
norm_A = S[0]

plt.plot(vecs[0,:], vecs[1,:], 'k--', label='Unit Circle')
plt.plot(mapped[0,:], mapped[1,:], 'b-', label='Mapped Ellipse')
plt.title(f"Operator Norm = {norm_A:.2f}")
plt.axis('equal')
plt.legend()
plt.tight_layout()
plt.show()
