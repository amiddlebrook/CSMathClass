Section 1

The Problem with Length

We want a function `m(A)` that assigns a "size" to subsets of `\mathbb{R}`. It should satisfy:

  1. `m([a, b]) = b - a` (Matches intuition)
  2. `m(A \cup B) = m(A) + m(B)` if disjoint (Additivity)
  3. Translation Invariance: `m(A + x) = m(A)`

Vitali Set Paradox: Assuming the Axiom of Choice, it is IMPOSSIBLE to define such a function on ALL subsets of `\mathbb{R}`. We must restrict our attention to "Measurable Sets".

Section 2

Sigma-Algebras

A `\sigma`-algebra `\mathcal{F}` is a collection of subsets of `X` that is safe for "countable operations".

1. `\emptyset \in \mathcal{F}`
2. Closed under complement: `A \in \mathcal{F} \implies A^c \in \mathcal{F}`
3. Closed under Countable Unions: `A_i \in \mathcal{F} \implies \bigcup_{i=1}^\infty A_i \in \mathcal{F}`

The pair `(X, \mathcal{F})` is a Measurable Space.

Section 3

Definition of Measure

A Measure is a function `\mu: \mathcal{F} \to [0, \infty]` such that:

  1. `\mu(\emptyset) = 0`
  2. Countable Additivity (`\sigma`-additivity): If `\{A_i\}` are disjoint:
    \mu\left(\bigcup_{i=1}^\infty A_i\right) = \sum_{i=1}^\infty \mu(A_i)

Note: We allow `\infty` as a value (e.g., length of `\mathbb{R}`).

Section 4

Lebesgue Measure

The standard measure on `\mathbb{R}`.

It is constructed using Outer Measure:

m^*(A) = \inf \left\{ \sum \text{len}(I_k) \mid A \subseteq \bigcup I_k, I_k \text{ are open intervals} \right\}

A set `A` is Lebesgue Measurable if it satisfies the Carathéodory condition:

m^*(E) = m^*(E \cap A) + m^*(E \cap A^c) \quad \forall E \subseteq \mathbb{R}

(Intuitively: `A` cuts all other sets cleanly).

Section 5

Borel Sets

The Borel `\sigma`-algebra `\mathcal{B}` is the smallest `\sigma`-algebra containing all open sets.

Any set you can write down (open, closed, unions of open, etc.) is Borel.

Note: There ARE Lebesgue measurable sets that are NOT Borel sets (subset of measure 0).

Section 6

Python: The Vitali Construction

We cannot "compute" non-measurable sets, but we can simulate the equivalence relation that creates them.

Python Rational Equivalence 
import numpy as np

# Equivalence Relation: x ~ y if x - y is Rational.
# Vitali Set extracts ONE element from each equivalence class of [0, 1].

def is_equivalent(x, y, tol=1e-9):
    # Check if x - y is roughly rational (hard for floats!)
    diff = abs(x - y)
    # Check if diff * N is integer for some small N
    for n in range(1, 100):
        if abs(diff * n - round(diff * n)) < tol:
            return True, f"{round(diff*n)}/{n}"
    return False, None

x = np.pi - 3 # 0.1415...
y = np.pi - 2.5 # 0.6415... (diff = 0.5 = 1/2)

print(f"Checking {x:.4f} vs {y:.4f}")
equiv, frac = is_equivalent(x, y)
print(f"Equivalent? {equiv} (Diff is {frac})")

z = np.sqrt(2) / 2
print(f"Checking {x:.4f} vs {z:.4f}")
print(f"Equivalent? {is_equivalent(x, z)[0]}")

# Concept: The interval [0, 1] is the disjoint union of 
# countably many shifted copies of the Vitali Set (by rationals).
# If V had measure m:
# Sum of m = 1 (Length of interval)
# But Sum of constant m is either 0 or Infinity.
# Contradiction.
