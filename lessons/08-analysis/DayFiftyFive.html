Why Rigorous Definitions Matter

Before the 19th century, calculus worked with vague notions like "infinitely small" quantities. The ε-δ definition, developed by Cauchy and Weierstrass, replaced intuition with precise logical statements that can be verified.

Key Insight: Every calculus theorem (derivatives, integrals, series) ultimately rests on the ε-δ definition of limits. Understanding it deeply unlocks the entire subject.

CS applications

  • Numerical analysis: Convergence of algorithms
  • Computer graphics: Continuity of curves and surfaces
  • Machine learning: Gradient descent convergence
  • Formal verification: Proving program correctness
Section 1

The Epsilon-Delta Definition of Limits

Formal statement

Definition: lim(x→a) f(x) = L means:
∀ε > 0, ∃δ > 0 such that: 0 < |x - a| < δ ⟹ |f(x) - L| < ε In words: "For every epsilon (tolerance around L), there exists a delta (tolerance around a), such that if x is within delta of a (but not equal to a), then f(x) is within epsilon of L."

Breaking it down

THE CHALLENGE: ε is given (any positive number) Your job: find a δ that works |x - a| < δ means: x is within δ of a a - δ < x < a + δ
THE GOAL: |f(x) - L| < ε means: f(x) is within ε of L L - ε < f(x) < L + ε Shrink ε → must shrink δ
Intuition: We can make f(x) arbitrarily close to L by taking x sufficiently close to a. The ε-δ language makes "arbitrarily" and "sufficiently" precise.
Section 2

Continuity

Definition

f is continuous at a if and only if: ∀ε > 0, ∃δ > 0 such that: |x - a| < δ ⟹ |f(x) - f(a)| < ε Note: No "0 <" on the left side (x can equal a) Equivalently: lim(x→a) f(x)=f(a) THREE CONDITIONS FOR CONTINUITY AT a: 1. f(a) is defined 2. lim(x→a) f(x) exists 3. lim(x→a) f(x)=f(a)

Types of discontinuity

REMOVABLE: Limit exists but ≠ f(a) Example: f(x) = (x²-1)/(x-1) at x = 1 JUMP: Left and right limits differ Example: f(x) = sign(x) at x = 0 ESSENTIAL: Limit doesn't exist (oscillates or unbounded) Example: f(x) = sin(1/x) at x = 0
Python Visualizing ε-δ 
import numpy as np
import matplotlib.pyplot as plt

def epsilon_delta_demo(f, a, L, epsilon):
    """Find delta for given epsilon (numerically)"""
    x = np.linspace(a - 1, a + 1, 1000)
    x = x[x != a]  # Remove x = a
    
    # Find points where |f(x) - L| < epsilon
    valid = np.abs(f(x) - L) < epsilon
    
    # Find largest delta where all points in (a-δ, a+δ) work
    for delta in np.linspace(0.01, 1, 100):
        in_delta = np.abs(x - a) < delta
        if np.all(valid[in_delta]):
            return delta
    return None

# Example: f(x) = 2x + 1, limit at x=2 is 5
f = lambda x: 2*x + 1
delta = epsilon_delta_demo(f, a=2, L=5, epsilon=0.1)
print(f"For ε = 0.1, δ = {delta}")  # δ = 0.05 works!
Section 3

Epsilon-Delta Proof Technique

The template

TO PROVE: lim(x→a) f(x) = L STEP 1: Let ε > 0 be given (arbitrary) STEP 2: SCRATCH WORK (not in final proof) Start with |f(x) - L| < ε Manipulate to find relationship with |x - a| Solve for δ in terms of ε STEP 3: Choose δ=... (from scratch work) STEP 4: Show that if 0 < |x - a| < δ, then |f(x) - L| < ε STEP 5: Conclude: Therefore lim(x→a) f(x)=L □

Example proof

PROVE: lim(x→3) (2x + 1) = 7 Let ε > 0 be given. SCRATCH: |(2x + 1) - 7| < ε |2x - 6| < ε 2|x - 3| < ε |x - 3| < ε/2 Choose δ=ε/2. Suppose 0 < |x - 3| < δ=ε/2. Then |2x + 1 - 7|=|2x - 6|=2|x - 3| < 2(ε/2)=ε ✓ Therefore lim(x→3) (2x + 1)=7. □
Common mistake: The scratch work finds what δ should be, but the proof must go in the other direction: assume |x - a| < δ and show |f(x) - L| < ε.
