Section 1

Completeness of ℝ

What makes ℝ special

Completeness: Every Cauchy sequence in ℝ converges to a limit in ℝ. There are no "holes" in the real line.
COMPLETENESS AXIOM (equivalent formulations): 1. Every Cauchy sequence converges 2. Every nonempty bounded set has a supremum 3. Every bounded monotone sequence converges WHY ℚ IS NOT COMPLETE: Consider √2 ≈ 1.414... Sequence: 1, 1.4, 1.41, 1.414, ... This is Cauchy in ℚ but doesn't converge in ℚ (√2 ∉ ℚ)

Supremum and infimum

SUPREMUM (sup): Least upper bound sup S = smallest number b such that x ≤ b for all x ∈ S INFIMUM (inf): Greatest lower bound inf S = largest number a such that a ≤ x for all x ∈ S EXAMPLE: S = (0, 1) sup S = 1 (not in S!) inf S = 0 (not in S!) max S doesn't exist (no largest element)
CS Application: Completeness is crucial for numerical analysis. It guarantees that iterative algorithms (Newton's method, gradient descent) that produce Cauchy sequences will converge to a real answer.
Section 2

Compactness

Definitions (equivalent in ℝⁿ)

A set K ⊆ ℝⁿ is COMPACT if any of these hold: 1. COVER DEF: Every open cover has a finite subcover {Uα} covers K ⟹ finitely many Uα₁,...,Uαₙ cover K 2. SEQUENTIAL: Every sequence in K has a convergent subsequence with limit in K 3. HEINE-BOREL: K is closed and bounded All three definitions are equivalent in ℝⁿ!

Key properties

COMPACT SETS: • [a, b] ✓ • Finite sets ✓ • {1/n : n ∈ ℕ} ∪ {0} ✓ NOT COMPACT: • (a, b) ✗ (not closed) • [0, ∞) ✗ (not bounded) • ℤ ✗ (not bounded)
KEY THEOREMS: • Continuous image of compact is compact • Compact ⊂ closed • Compact ⊂ bounded • Closed subset of compact is compact
Compactness matters for optimization: The Extreme Value Theorem (continuous functions attain max/min) requires a compact domain. On non-compact domains, max/min may not exist!
Section 3

Uniform Continuity

Definition and comparison

POINTWISE CONTINUITY (at each point a): ∀ε > 0, ∃δ(a,ε) > 0: |x-a| < δ ⟹ |f(x)-f(a)| < ε Note: δ can depend on BOTH ε and the point a UNIFORM CONTINUITY (on a set): ∀ε> 0, ∃δ(ε) > 0: |x-y| < δ ⟹ |f(x)-f(y)| < ε Note: δ depends ONLY on ε (works for all points) UNIFORM ⟹ POINTWISE (but not vice versa)

Key theorem

Heine-Cantor Theorem: Continuous on compact ⟹ uniformly continuous.
EXAMPLE: f(x) = x² on [0, 1] f is continuous on compact [0, 1] Therefore f is uniformly continuous on [0, 1] COUNTEREXAMPLE: f(x) = 1/x on (0, 1) f is continuous (pointwise) on (0, 1) But NOT uniformly continuous! (near 0, need smaller and smaller δ)
Python Testing uniform continuity 
import numpy as np

def check_uniform_continuity(f, domain, epsilon):
    """Find if single δ works for all points (approximately)"""
    x = np.linspace(domain[0] + 0.01, domain[1] - 0.01, 100)
    
    # For each point, find minimum δ needed
    min_deltas = []
    for a in x:
        # Binary search for delta
        for delta in np.logspace(-6, 0, 100):
            nearby = x[(np.abs(x - a) < delta) & (x != a)]
            if len(nearby) > 0 and np.all(np.abs(f(nearby) - f(a)) < epsilon):
                min_deltas.append(delta)
                break
    
    return np.min(min_deltas), np.max(min_deltas)

# x² on [0,1]: uniform (bounded delta range)
d_min, d_max = check_uniform_continuity(lambda x: x**2, [0, 1], 0.1)
print(f"x² on [0,1]: δ range [{d_min:.4f}, {d_max:.4f}]")

# 1/x on (0,1): NOT uniform (δ→0 near x=0)
