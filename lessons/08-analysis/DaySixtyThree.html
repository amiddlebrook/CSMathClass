Section 1

Strong Convergence

Standard convergence in specific norm `\| \cdot \|` is called Strong Convergence.

x_n \to x \iff \|x_n - x\| \to 0
üß© Simple: Mathematical concepts are like building blocks. Each new idea builds on previous ones, so mastering the fundamentals makes everything else easier to understand.
Section 2

Weak Convergence

In infinite dimensions, strong convergence is hard to get. The unit ball isn't compact! Sequences can run away to "higher frequencies".

Definition: `x_n \rightharpoonup x` (converges weakly) if for every functional `f \in X^*`:
`\lim_{n \to \infty} f(x_n) = f(x)`

Intuition: `x_n` looks like `x` if you view it through any linear "lens" `f`.

üîç Simple: The "Hearing vs Seeing" Analogy.
‚Ä¢ Strong: Two orchestra performances sound identical note-for-note.
‚Ä¢ Weak: Two performances sound "similar" to every listener polling questions about the music (was it loud? was it fast?) even if individual notes differ.
Section 3

Example: Oscillation

Consider `u_n(x) = \sin(nx)` in `L^2[0, 2\pi]`.

  • Strongly: `\|u_n - 0\|_2 = \sqrt{\pi} \ne 0`. It does NOT converge to 0 strongly.
  • Weakly: For any `g \in L^2`, `\langle u_n, g \rangle = \int \sin(nx) g(x) dx`. By Riemann-Lebesgue Lemma, this integral goes to 0 as `n \to \infty`.
üñºÔ∏è Simple: The "Pixelated Image" Analogy.
‚Ä¢ Strong Convergence: A high-res photo gradually morphs exactly into the target image. Every pixel matches perfectly in the limit.
‚Ä¢ Weak Convergence: The image might look like static noise (high frequency oscillations), but if you squint (average over regions), it assumes the correct shape.
Essentially, the "average" behavior is correct, even if the fine details are going crazy.
Section 4

Banach-Alaoglu Theorem

This is the Heine-Borel of infinite dimensions.

Theorem: The closed unit ball in `X^*` is Weak-* Compact.

Translation: Any bounded sequence of functionals has a weakly convergent subsequence. This is essential for finding solutions to PDEs (Direct Method in Calculus of Variations).

Section 5

Reflexive Spaces

If `(X^*)^* = X`, the space is Reflexive.

In Reflexive spaces (like `L^p`, `1 < p < \infty` and Hilbert spaces), the unit ball is Weakly Compact.

This means if you have a bounded energy sequence `\|u_n\| \le C`, you can always extract a weakly convergent subsequence `u_{n_k} \rightharpoonup u`.

Python: Oscillations

Python Riemann-Lebesgue Check 
import numpy as np
import matplotlib.pyplot as plt

# Demonstrate that sin(nx) "looks like" 0 when integrated against a test function

x = np.linspace(0, 2*np.pi, 2000)
# Test function g(x) = x
g = x
# Sequence u_n(x) = sin(nx)
ns = [1, 5, 20, 100]

integrals = []

for n in ns:
    u = np.sin(n * x)
    # Inner product  in L2 ~ Integral u(x)g(x) dx
    inner_prod = np.trapz(u * g, x)
    integrals.append(inner_prod)
    
print("Inner products :")
for n, val in zip(ns, integrals):
    print(f"n={n:3d}: {val:.5f}")

# Plot
plt.figure(figsize=(10, 3))
plt.plot(x, np.sin(20*x), label='sin(20x)')
plt.plot(x, g, label='g(x)=x', linewidth=2)
plt.title("High Frequency Oscillation against Test Function")
plt.legend()
plt.show()

# Note: The area of (sin(nx) * x) cancels out positive and negative regions equally
# as frequency increases.
