Section 1

Limit Laws

Algebraic limit laws

If lim(x→a) f(x) = L and lim(x→a) g(x) = M exist, then:
SUM: lim[f(x) + g(x)] = L + M DIFFERENCE: lim[f(x) - g(x)] = L - M PRODUCT: lim[f(x) · g(x)] = L · M QUOTIENT: lim[f(x)/g(x)] = L/M (if M ≠ 0) SCALAR: lim[c·f(x)] = c·L POWER: lim[f(x)ⁿ] = Lⁿ ROOT: lim[ⁿ√f(x)] = ⁿ√L (if L > 0 or n odd)

One-sided limits

LEFT LIMIT: lim(x→a⁻) f(x) = L x approaches a from below RIGHT LIMIT: lim(x→a⁺) f(x) = L x approaches a from above lim(x→a) f(x) exists ⟺ both one-sided limits exist and are equal
Limit laws require limits to exist! You cannot apply these laws to expressions like ∞ - ∞ or 0/0. These are indeterminate forms that require special techniques (L'Hôpital's rule, algebraic manipulation).
Section 2

Squeeze Theorem

Squeeze Theorem: If g(x) ≤ f(x) ≤ h(x) near a (except possibly at a), and lim g(x) = lim h(x) = L, then lim f(x) = L.
Intuition: f is "squeezed" between g and h, and both squeeze toward L. CLASSIC EXAMPLE: lim(x→0) x·sin(1/x) = 0 Proof: -1 ≤ sin(1/x) ≤ 1 -|x| ≤ x·sin(1/x) ≤ |x| lim(-|x|) = lim(|x|) = 0 Therefore lim x·sin(1/x) = 0 ✓
Python Visualizing the squeeze theorem 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-1, 1, 1000)
x = x[x != 0]  # Avoid division by zero

f = x * np.sin(1/x)
g = -np.abs(x)  # Lower bound
h = np.abs(x)   # Upper bound

plt.figure(figsize=(10, 6))
plt.plot(x, f, 'b-', label='x·sin(1/x)', alpha=0.7)
plt.plot(x, g, 'r--', label='-|x|')
plt.plot(x, h, 'g--', label='|x|')
plt.axhline(0, color='k', linewidth=0.5)
plt.legend()
plt.title('Squeeze Theorem: x·sin(1/x) squeezed to 0')
plt.show()

Intermediate Value Theorem

IVT: If f is continuous on [a, b] and k is between f(a) and f(b), then there exists c ∈ (a, b) such that f(c) = k.
REQUIREMENTS: • f continuous on closed interval [a, b] • k strictly between f(a) and f(b) CONCLUSION: • ∃c in open interval (a, b) with f(c) = k APPLICATIONS: • Root finding: f(a) < 0 < f(b) ⟹ ∃c with f(c)=0 • Bisection method: guaranteed to find root
Python Bisection method (based on IVT) 
def bisection(f, a, b, tol=1e-10):
    """Find root of f in [a,b] using IVT"""
    assert f(a) * f(b) < 0, "Need sign change"
    
    while b - a > tol:
        c = (a + b) / 2
        if f(c) == 0:
            return c
        elif f(a) * f(c) < 0:
            b = c  # Root in [a, c]
        else:
            a = c  # Root in [c, b]
    return (a + b) / 2

# Find √2 (root of x² - 2)
f = lambda x: x**2 - 2
root = bisection(f, 1, 2)
print(f"√2 ≈ {root:.10f}")  # 1.4142135624...
Section 4

Extreme Value Theorem

EVT: If f is continuous on a closed interval [a, b], then f attains both a maximum and minimum value on [a, b].
∃c, d ∈ [a, b] such that: f(c) ≤ f(x) ≤ f(d) for all x ∈ [a, b] f(c) = minimum value f(d) = maximum value REQUIREMENTS: • Continuous on [a, b] • Interval must be CLOSED and BOUNDED COUNTEREXAMPLES (when conditions fail): • f(x) = 1/x on (0, 1): no max (goes to ∞) • f(x) = x on (0, 1): no max or min (never attained)
CS Application: The EVT guarantees that optimization problems with continuous objectives over compact (closed, bounded) domains have solutions. This is fundamental to machine learning and operations research.
