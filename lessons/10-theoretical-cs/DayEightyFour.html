Section 1

Beyond the Bit

A classical bit is a light switch: it is either **On (1)** or **Off (0)**. A quantum bit, or **Qubit**, is more like a dimmer switch that can be anywhere in between—until you look at it, at which point it "snaps" to 0 or 1.

This is not just "uncertainty." This is **Superposition**, where the particle exists in a linear combination of states.

Section 2

The State Vector

Recall our work on **Hilbert Spaces** (Day 75). A Qubit is represented as a unit vector in a 2D complex vector space. We use **Dirac Notation** (Bra-Ket):

$$ |\psi\rangle = \alpha|0\rangle + \beta|1\rangle $$
  • $|0\rangle$: The base state "Zero" $\begin{bmatrix} 1 \\ 0 \end{bmatrix}$.
  • $|1\rangle$: The base state "One" $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$.
  • $\alpha, \beta$: **Probability Amplitudes** (Complex numbers).

The core rule: the probabilities must sum to 1. Since probabilities are the square of the amplitude:

$$ |\alpha|^2 + |\beta|^2 = 1 $$
Section 3

The Power of "Both"

If you have 1 bit, you can store 1 value. If you have 10 bits, you can store 1 value (out of 1024). But if you have 10 **qubits** in superposition, you can store a state that is a combination of **all 1024 values at once**.

This is why quantum computers can potentially solve problems that would take a classical computer the age of the universe.

Section 4

Visualizing with the Bloch Sphere

Since a Qubit is a normalized vector, we can represent it as a point on a sphere. The North pole is $|0\rangle$, the South pole is $|1\rangle$, and any point on the surface is a valid superposition state.

Degrees of Freedom

Even though $\alpha$ and $\beta$ are complex (4 real numbers), the normalization and the "global phase" reduce the freedom to just two real angles: $\theta$ and $\phi$.

Section 5

The Collapse

When we measure a qubit $|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$, it instantly collapses into either $|0\rangle$ or $|1\rangle$.

  • The probability of getting $|0\rangle$ is $|\alpha|^2$.
  • The probability of getting $|1\rangle$ is $|\beta|^2$.

Quantum computing is the art of manipulating these amplitudes so that the "correct" answer has the highest probability of being measured.

Section 6

Python: Simulation of a Qubit

Let's represent a Qubit and simulate the measurement process using NumPy.

Python Qubit Simulator 
import numpy as np

class Qubit:
    def __init__(self, alpha, beta):
        # Normalize just in case
        norm = np.sqrt(abs(alpha)**2 + abs(beta)**2)
        self.state = np.array([alpha/norm, beta/norm], dtype=complex)

    def measure(self):
        # Probabilities are amplitudes squared
        p0 = abs(self.state[0])**2
        p1 = abs(self.state[1])**2
        
        # Weighted random choice
        result = np.random.choice([0, 1], p=[p0, p1])
        return result

# Create a qubit in equal superposition (Hadarmard state)
# |psi> = (1/sqrt(2))|0> + (1/sqrt(2))|1>
psi = Qubit(1, 1)

# Measure it 1000 times to see the distribution
results = [psi.measure() for _ in range(1000)]
print(f"Zeros: {results.count(0)}")
print(f"Ones:  {results.count(1)}")

# Create a qubit biased toward |1>
biased = Qubit(1, 4) # 1:16 probability ratio
results = [biased.measure() for _ in range(1000)]
print(f"Biased Zeros: {results.count(0)}")
print(f"Biased Ones:  {results.count(1)}")
