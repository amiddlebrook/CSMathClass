Section 1

The Sumset A+A

Given a set of integers `A`, the sumset is `A+A = {a+b : a ∈ A, b ∈ A}`.

If `A` is an arithmetic progression (like `1, 2, 3...N`), then `|A+A| ≈ 2|A|`.
If `A` is random (like powers of 2), `|A+A| ≈ |A|^2/2`.

Inverse Problem: If `|A+A|` is small, `A` must look like an arithmetic progression.
Section 2

Roth's Theorem

Roth proved that any subset of integers with positive density contains a 3-term arithmetic progression.

x, x+d, x+2d

This was generalized by Szemerédi to k-term progressions, a towering achievement.

Section 3

Freiman's Theorem

Specifically characterizes sets with small sumsets.

If `|A+A| < K|A|`, then `A` is contained in a Generalized Arithmetic Progression (GAP) of small dimension.

Section 4

Sum-Product Phenomenon

Erdős-Szemerédi Conjecture: A set cannot have both a small sumset AND a small product set.

\max(|A+A|, |A \cdot A|) \ge c|A|^{1+\epsilon}
Section 5

Python: Sumset Size

Python Calculate |A+A| and |A*A| 
def calc_sets(A):
    sumset = {a+b for a in A for b in A}
    prodset = {a*b for a in A for b in A}
    return len(sumset), len(prodset)

# Arithmetic Progression
AP = list(range(1, 11))
# Geometric Progression
GP = [2**i for i in range(10)]

print("AP Sum vs Prod:", calc_sets(AP)) # Sum small, Prod large
print("GP Sum vs Prod:", calc_sets(GP)) # Sum large, Prod small
