Section 1

Wigner Matrices

A random symmetric matrix where entries are i.i.d. random variables (usually Gaussian).

Eugene Wigner used them to model Hamiltonian operators of heavy atomic nuclei.

Section 2

Semicircle Law

As N → ∞, the distribution of eigenvalues follows a semicircle shape.

\rho(x) = \frac{1}{2\pi} \sqrt{4 - x^2}

This is the "Central Limit Theorem" for matrices.

Section 3

Circular Law

For non-symmetric random matrices, eigenvalues are distributed uniformly in the unit disk in the complex plane.

Section 4

Universality

The local spacing between eigenvalues depends only on symmetry (Real symmetric vs Complex Hermitian) and NOT the specific distribution of entries (Gaussian, Bernoulli, etc).
Section 5

Python: Eigenvalue Hist

Python Semicircle Check 
import numpy as np
import matplotlib.pyplot as plt

N = 1000
# Create random symmetric matrix
A = np.random.randn(N, N)
S = (A + A.T) / np.sqrt(2*N)

eigvals = np.linalg.eigvalsh(S)
