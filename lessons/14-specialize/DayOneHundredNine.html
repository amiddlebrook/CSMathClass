Section 1

Domains

Harmonic Analysis generalizes "frequency".

  • Periodic on Circle (T): Fourier Series (Discrete frequencies Z).
  • Line (R): Fourier Transform (Continuous frequencies R).
  • Discrete (Z): DTFT (Continuous periodic frequencies on Circle).
  • Finite (Z_N): DFT (Discrete frequencies Z_N).
Section 2

Pontryagin Duality (LCA Groups)

For any Locally Compact Abelian (LCA) group `G`, the Dual Group `\hat{G}` consists of all continuous characters (homomorphisms to the circle group).

\hat{\mathbb{R}} \cong \mathbb{R} \\ \hat{\mathbb{Z}} \cong \mathbb{T} \text{ (Circle)} \\ \hat{\mathbb{T}} \cong \mathbb{Z} \\ \hat{Z_N} \cong Z_N
Section 3

The Uncertainty Principle

A function cannot be localized in both time and frequency.

\sigma_t \cdot \sigma_f \ge \frac{1}{4\pi}

If you squeeze a signal in time (short pulse), its spectrum spreads out (broadband).

Section 4

Wavelets

Fourier uses sines (infinite duration). Wavelets use localized packets.

Multiresolution analysis decomposes a signal into detail at different scales. Key for compression (JPEG 2000).

Section 5

Python: Time-Freq Uncertainty

Python Spectrogram 
import numpy as np
import matplotlib.pyplot as plt
import scipy.signal

# Generate a "Chirp" signal (freq increases with time)
fs = 1000
t = np.linspace(0, 2, 2*fs)
freq = np.linspace(20, 200, len(t))
signal = np.sin(2 * np.pi * freq * t)

# Compute Spectrogram (Short-Time Fourier Transform)
f, t_spec, Sxx = scipy.signal.spectrogram(signal, fs)

plt.figure(figsize=(10, 5))
plt.pcolormesh(t_spec, f, Sxx, shading='gouraud')
plt.ylabel('Frequency [Hz]')
plt.xlabel('Time [sec]')
plt.title('Spectrogram: Seeing Time and Frequency')
plt.colorbar(label='Intensity')
plt.ylim(0, 250)
plt.show()
