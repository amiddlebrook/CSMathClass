Section 1

The Derivative

The derivative is the limit of the difference quotient:

$f'(c) = \lim_{h \to 0} \frac{f(c+h) - f(c)}{h}$

If this limit exists, $f$ is differentiable at $c$.
Differentiability $\implies$ Continuity (but not the other way around, e.g., $|x|$ at 0).

Section 2

Properties

  • Linearity: $(af + bg)' = af' + bg'$
  • Product Rule: $(fg)' = f'g + fg'$
  • Chain Rule: $(f(g(x)))' = f'(g(x))g'(x)$
Section 3

Mean Value Theorem (MVT)

If $f$ is continuous on $[a, b]$ and differentiable on $(a, b)$, then there exists a $c \in (a, b)$ such that: $$ f'(c) = \frac{f(b) - f(a)}{b - a} $$

"At some point, the instantaneous speed MUST equal the average speed."

Corollary: If $f'(x) = 0$ for all $x$, then $f(x)$ is constant. This is crucial for proving uniqueness of solutions.
Section 4

Python Differentiation

We can compute exact derivatives with SymPy or approximate them with Finite Differences.

Python Symbolic & Numeric 
import numpy as np
from sympy import symbols, diff, sin, lambdify

# --- Symbolic ---
x = symbols('x')
expr = x**3 - 2*x

deriv = diff(expr, x)
print(f"d/dx ({expr}) = {deriv}") # 3x^2 - 2

# --- Numeric (Finite Difference) ---
f = lambda v: v**3 - 2*v
c = 2.0
h = 1e-5

numerical_deriv = (f(c + h) - f(c)) / h
exact = 3*(c**2) - 2

print(f"\nAt x={c}:")
print(f"Numerical (h={h}): {numerical_deriv:.6f}")
print(f"Exact: {exact}")
