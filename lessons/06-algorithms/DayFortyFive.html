Section 1

Loop Invariants

Loop Invariant: A property that holds before and after every loop iteration.
PROVING WITH LOOP INVARIANTS: 1. Initialization: True before first iteration 2. Maintenance: If true before iteration, true after 3. Termination: When loop ends, invariant + exit condition implies correctness

Example: Insertion sort

Invariant: At start of each iteration, arr[0..i-1] is sorted and contains original elements.

  • Init: i=1, arr[0..0] is trivially sorted
  • Maint: Insert arr[i] in correct position ‚Üí arr[0..i] sorted
  • Term: i=n, so arr[0..n-1] = entire array is sorted ‚úì

Induction Proofs

MATHEMATICAL INDUCTION: 1. Base Case: Prove P(0) or P(1) is true 2. Inductive Hypothesis: Assume P(k) is true 3. Inductive Step: Prove P(k) ‚Üí P(k+1) 4. Conclusion: P(n) true for all n ‚â• base

Strong induction

Assume P(1), P(2), ..., P(k) all true, prove P(k+1). Useful for divide-and-conquer proofs.

Common use: Proving recursive algorithms correct. The inductive structure mirrors the recursive structure.
