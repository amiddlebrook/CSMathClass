Section 1

What is Dynamic Programming?

DP = Recursion + Memoization. Solve each subproblem once, store the result, reuse it.

When to use DP

  • Optimal substructure: Optimal solution built from optimal subsolutions
  • Overlapping subproblems: Same subproblems solved repeatedly

The Fibonacci example

With memoization: Each fib(k) computed once ‚Üí O(n) time, O(n) space.
Section 2

Top-Down vs Bottom-Up

Top-down (memoization)

Write recursive solution, add cache to avoid recomputation.

PythonFibonacci with memoization
def fib(n, memo={}):
    if n <= 1: return n
    if n not in memo:
        memo[n] = fib(n-1) + fib(n-2)
    return memo[n]

Bottom-up (tabulation)

Build solution iteratively from smallest subproblems.

PythonFibonacci with tabulation
def fib(n):
    if n <= 1: return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
Space optimization: For Fibonacci, only need last 2 values ‚Üí O(1) space possible!
Section 3

Common DP Patterns

DP PATTERNS: 1. Linear DP: dp[i] = f(dp[i-1], dp[i-2], ...) 2. 2D DP: dp[i][j] = f(dp[i-1][j], dp[i][j-1], ...) 3. Interval DP: dp[i][j] for subarray [i..j] 4. Knapsack: dp[i][w] = best using items 1..i with capacity w 5. Subsequence: Compare all (i,j) pairs of two sequences

State definition

The key insight in DP is defining what dp[i] represents:

  • Longest Increasing Subsequence: dp[i] = length of LIS ending at index i
  • Coin Change: dp[a] = min coins needed to make amount a
  • Edit Distance: dp[i][j] = edits to convert s1[0..i] to s2[0..j]

Classic DP Problems

CLASSIC DP PROBLEMS: ‚Ä¢ Fibonacci/Climbing Stairs ‚Äî dp[i] = dp[i-1] + dp[i-2] ‚Ä¢ Coin Change ‚Äî dp[a] = 1 + min(dp[a - coin]) ‚Ä¢ 0/1 Knapsack ‚Äî take or skip each item ‚Ä¢ Longest Common Subsequence ‚Äî dp[i][j] = max match ‚Ä¢ Edit Distance ‚Äî insert/delete/replace ‚Ä¢ Longest Increasing Subseq ‚Äî dp[i] = 1 + max(dp[j]) if a[j] < a[i] ‚Ä¢ House Robber ‚Äî rob or skip each house
PythonCoin Change
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for a in range(1, amount + 1):
        for coin in coins:
            if coin <= a:
                dp[a] = min(dp[a], 1 + dp[a - coin])
    return dp[amount] if dp[amount] != float('inf') else -1
