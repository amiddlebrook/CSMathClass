Section 1

How Memory Works

Think of your computer's RAM as a massive grid of mailboxes, each with a unique ID (address).

Contiguous (Array)

You ask for 10 adjacent boxes. Fast because you know exactly where box #7 is.

Scattered (Predicted)

You ask for one box, and inside it is a note pointing to the next box (pointer).

Section 2

Static & Dynamic Arrays

Static Arrays: Fixed size (e.g. C++ `int arr[10]`). Cannot grow.

Dynamic Arrays: (e.g. Python `list`, Java `ArrayList`). They automatically resize.

Dynamic Resizing (Amortized O(1))

When a dynamic array is full, it creates a new array double the size, copies everyone over, and deletes the old one. This "hiccup" is rare, so the average append time is still $O(1)$.

Section 3

Linked Lists

A chain of nodes. Each node contains:

  1. Data: The value.
  2. Next: A pointer (reference) to the next node.

Doubly Linked List: Has `next` and `prev` pointers.

Section 4

Big O Showdown

Operation Array (Dynamic) Linked List
Access (Index `i`) $O(1)$ (Math calc) $O(n)$ (Must walk)
Append (End) $O(1)^*$ (Amortized) $O(1)$ (If tail pointer)
Prepend (Start) $O(n)$ (Shift all) $O(1)$ (Just update head)
Insert (Middle) $O(n)$ (Shift) $O(1)$ (At pointer)
Section 5

Python Implementation

Implementing a simple Node and Linked List class.

Python Linked List 
class Node:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class LinkedList:
    def __init__(self):
        self.head = None
    
    def prepend(self, val):
        # O(1) - Fast!
        new_node = Node(val, self.head)
        self.head = new_node

    def append(self, val):
        # O(n) - Slow without a tail pointer!
        if not self.head:
            self.head = Node(val)
            return
        
        current = self.head
        while current.next:
            current = current.next
        current.next = Node(val)

    def print_list(self):
        current = self.head
        while current:
            print(current.val, end=" -> ")
            current = current.next
        print("None")

ll = LinkedList()
ll.prepend(1)
ll.prepend(2)
ll.append(3)
ll.print_list()
# Output: 2 -> 1 -> 3 -> None
