Searching Algorithms

Linear search

Check each element sequentially. Works on unsorted data.

LINEAR SEARCH: O(n) Best: O(1) ‚Äî first element Worst: O(n) ‚Äî last or not found

Binary search

Requires: Sorted array. Halves search space each step.
BINARY SEARCH: O(log n) Best: O(1) ‚Äî middle element Worst: O(log n) ‚Äî any other
PythonBinary Search
def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1

Greedy Algorithms

Greedy choice: Make the locally optimal choice at each step, hoping for globally optimal solution.

When greedy works

  • Optimal substructure: Optimal solution contains optimal subsolutions
  • Greedy choice property: Local optimal leads to global optimal

Classic examples

GREEDY ALGORITHMS: ‚Ä¢ Activity Selection ‚Äî pick earliest finish time ‚Ä¢ Huffman Coding ‚Äî combine smallest frequencies ‚Ä¢ Dijkstra's Algorithm ‚Äî pick nearest unvisited ‚Ä¢ Kruskal's/Prim's MST ‚Äî pick smallest edge ‚Ä¢ Coin Change (some cases) ‚Äî pick largest coin
Greedy fails: Coin change with arbitrary denominations (e.g., {1, 3, 4} to make 6: greedy gives 4+1+1=3 coins, but 3+3=2 coins is optimal). Use DP when greedy fails.

Divide & Conquer

D&C Strategy: Divide problem into subproblems, solve recursively, combine solutions.

The pattern

DIVIDE & CONQUER: 1. DIVIDE: Break into smaller subproblems 2. CONQUER: Solve subproblems recursively 3. COMBINE: Merge solutions into final answer

Classic examples

  • Merge Sort: Split, sort halves, merge
  • Quick Sort: Partition around pivot, sort halves
  • Binary Search: Check middle, search one half
  • Strassen's Matrix Mult: O(n^2.81) vs O(n¬≥)
  • Karatsuba Multiplication: O(n^1.58) vs O(n¬≤)

Master theorem

MASTER THEOREM for T(n) = aT(n/b) + O(n^d): ‚Ä¢ If d < log_b(a): T(n)=O(n^(log_b a)) ‚Ä¢ If d=log_b(a): T(n)=O(n^d log n) ‚Ä¢ If d> log_b(a): T(n) = O(n^d) Example: Merge sort T(n) = 2T(n/2) + O(n) a=2, b=2, d=1 ‚Üí log_2(2)=1=d ‚Üí O(n log n)
