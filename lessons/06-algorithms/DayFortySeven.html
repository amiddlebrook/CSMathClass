Section 1

Sequences

A sequence $(a_n)$ converges to $L$ if for every $\epsilon > 0$, terms eventually stay within $L \pm \epsilon$.

Monotone Convergence Theorem: Every bounded monotone sequence converges.
Cauchy Sequence: Terms get closer to each other. In $\mathbb{R}$, Cauchy $\iff$ Convergent (Completeness).
Section 2

Infinite Series

Sum of a sequence: $S = \sum_{n=1}^\infty a_n$. We define this as the limit of partial sums $S_N = \sum_{n=1}^N a_n$.

Geometric Series: $\sum ar^n$ converges if $|r| < 1$. Sum=$a/(1-r)$.
Harmonic Series: $\sum 1/n$ DIVERGES (even though terms go to 0).
Section 3

Convergence Tests

  • Divergence Test: If $\lim a_n \neq 0$, sequences diverges.
  • Ratio Test: If $\lim |a_{n+1}/a_n| = L < 1$, converges absolutely. Excellent for factorials/exponentials.
  • Integral Test: Converges if $\int_1^\infty f(x) dx$ converges.
  • Alternating Series Test: $\sum (-1)^n a_n$ converges if $a_n$ decreases to 0.
Section 4

Visualizing Convergence

Comparing Harmonic vs Geometric series partial sums.

Python Partial Sums 
import numpy as np
import matplotlib.pyplot as plt 
# Note: In browser we just print values

N = 20
n = np.arange(1, N+1)

# 1. Harmonic: 1/n
harmonic = 1/n
harm_sum = np.cumsum(harmonic)

# 2. Geometric: (1/2)^n
geom = (0.5)**n
geom_sum = np.cumsum(geom)

print("n \t Harmonic \t Geometric")
print("-"*30)
for i in range(N):
    print(f"{i+1} \t {harm_sum[i]:.4f} \t {geom_sum[i]:.4f}")

# Output shows Harmonic grows slowly (diverges)
# Geometric approaches 1.0 (converges)
