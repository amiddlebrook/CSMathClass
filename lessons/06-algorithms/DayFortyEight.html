Section 1

The Master Formula

If $f$ is infinitely differentiable at $a$, its Taylor Series is:

$f(x) = \sum_{n=0}^\infty \frac{f^{(n)}(a)}{n!} (x-a)^n$

It matches the value, slope, concavity, jerk, etc., of the function at a single point $a$. When $a=0$, it is called a Maclaurin Series.

Section 2

Common Expansions

$e^x = 1 + x + \frac{x^2}{2!} + \dots$ (Converges everywhere)
$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots$
$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots$
$\frac{1}{1-x} = 1 + x + x^2 + \dots$ (Converges for $|x|<1$)
Euler's Identity: $e^{ix} = \cos x + i \sin x$ becomes obvious when looking at the series expansions.
Section 3

Taylor's Theorem (Error)

The error $R_n(x)$ in using the $n$-th degree polynomial is: $$ R_n(x) = \frac{f^{(n+1)}(c)}{(n+1)!} (x-a)^{n+1} $$ for some $c$ between $a$ and $x$. This allows us to bound the error and prove convergence.

Section 4

Pointwise Convergence

A sequence of functions $f_n(x)$ converges pointwise to $f(x)$ if for every fixed $x$, the sequence of numbers $(f_n(x))$ converges to $f(x)$.
$\lim_{n \to \infty} f_n(x) = f(x)$ for each $x$.

Pointwise convergence does NOT preserve continuity.
Example: $f_n(x) = x^n$ on $[0,1]$.
At $x=1$, converges to 1. At $x<1$, converges to 0. Limit is discontinuous!
Section 5

Uniform Convergence

$f_n \to f$ uniformly if the "error band" shrinks to 0 for ALL $x$ simultaneously.

$\forall \epsilon > 0, \exists N, \forall n \ge N, \forall x \in D: |f_n(x) - f(x)| < \epsilon$

"The choice of $N$ depends only on $\epsilon$, not on $x$."

Section 6

Why we care

Uniform convergence preserves nice properties:

  • Continuity: If continuous $f_n \to f$ uniformly, $f$ is continuous.
  • Integration: $\lim \int f_n = \int \lim f_n$ (You can swap limit and integral).
  • Differentiation: Harder. Requires $f_n'$ to converge uniformly.
Weierstrass M-Test: If $|f_n(x)| \le M_n$ and $\sum M_n$ converges, then $\sum f_n(x)$ converges uniformly.
Section 7

SymPy Expansions

Computing series coefficients automatically.

Python Taylor Series 
from sympy import symbols, series, sin, exp, cos

x = symbols('x')

# Maclaurin series of sin(x) up to x^9
TS_sin = series(sin(x), x, 0, 10)
print(f"Sin(x) ~ {TS_sin}")

# Taylor Series of e^x at x=1
TS_exp = series(exp(x), x, 1, 4)
print(f"e^x at x=1 ~ {TS_exp}")

# Using series to take limits
# Limit of (sin x - x)/x^3 as x->0
# sin x - x ~ -x^3/6. So ratio is -1/6.
expr = (sin(x) - x) / x**3
lim = expr.limit(x, 0)
print(f"Limit: {lim}") # -1/6
Section 8

Failure of Pointwise

Visualizing $x^n$ on $[0,1]$.

Python x^n Convergence 
import numpy as np

# Define domain
x = np.linspace(0, 1, 11)

print("x values:", np.round(x, 2))
print("-"*40)

# Check f_n(x) for increasing n
for n in [1, 5, 20, 100]:
    y = x**n
    print(f"n={n}: {np.round(y, 3)}")

# Notice x=0.99 eventually goes to 0
# But x=1.00 stays at 1
# The convergence gets SLOWER as x gets closer to 1 (from left).
# This non-uniformity breaks the continuity of the limit.
