Section 2

Foundations: Space & Rank

1. Independence

Vectors are independent if they point in "new" directions.

Test: Determinant $\neq 0$ (Volume exists).

2. Rank

The number of true dimensions in your data.

If Rank < Rows, you have redundancy (Dependencies).

The Fundamental Theorem (Part 1):
Rank(Column Space) = Rank(Row Space).
Dimensions of Inputs = Dimensions of Outputs.

Section 4

Capstone: Eigenfaces (PCA)

Let's put it all together. How does Facebook recognize your face? It uses a technique called Principal Component Analysis (PCA), which is just SVD in disguise.

The Algorithm:
  1. flatten each image (100x100 pixels) into a huge vector ($10,000 \times 1$).
  2. Stack all faces into a matrix $X$.
  3. Subtract the "Average Face" (Mean centering).
  4. Compute the Covariance Matrix: $C = X X^T$.
  5. Find the Eigenvectors of $C$.

These Eigenvectors are called "Eigenfaces". They are the "Ghost Faces" that make up all human faces. Any face is just a weighted sum of these ghosts.
