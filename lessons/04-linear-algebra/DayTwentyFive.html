Why Abstract Linear Algebra?

Abstract = powerful. Vector spaces unify polynomials, functions, sequences—not just Rⁿ.

Applications

  • Functional analysis: Infinite-dimensional spaces
  • Quantum mechanics: Hilbert spaces
  • ML: Feature spaces, kernel methods
  • Fourier analysis: Functions as vectors
Section 1

Vector Spaces

Definition

A vector space V over field F is a set with: • Addition: v + w ∈ V • Scalar multiplication: cv ∈ V for c ∈ F Satisfying 8 axioms (commutativity, associativity, identity, inverses, distributivity, etc.)

Examples beyond Rⁿ

• Polynomials of degree ≤ n: Pₙ • Continuous functions on [a,b]: C[a,b] • Sequences: ℓ∞, ℓ² • Matrices: Mₘₓₙ

Subspaces, basis, dimension

Subspace: Subset closed under + and scalar ×. Span: All linear combinations of a set. Basis: Linearly independent spanning set. Dimension: Size of any basis. dim(P₃) = 4 (basis: 1, x, x², x³)
Section 2

Linear Maps

Definition

Linear Map
T: V → W is linear if T(u+v) = Tu + Tv and T(cv) = cTv

Key spaces

Kernel: ker(T) = {v : Tv = 0} Image: im(T) = {Tv : v ∈ V} Rank-Nullity Theorem: dim(ker T) + dim(im T) = dim(V)

Matrix representation

Choose bases B for V, C for W. Then T has a matrix [T]_B^C. Different bases → different matrices (similar) Same linear map!
Key insight: Matrices are just representations of linear maps. The map is the fundamental object.
Section 3

Dual Spaces

Definition

The dual space V* = L(V, F) = all linear functionals on V. A linear functional is φ: V → F (maps to scalars). V* is itself a vector space! dim(V*) = dim(V) (for finite-dim V)

Dual basis

If {e₁,...,eₙ} is basis for V, then {e¹,...,eⁿ} is dual basis for V* where eⁱ(eⱼ) = δᵢⱼ (1 if i=j, 0 otherwise)

Why it matters

• Transpose is dual map: (Aᵀ)* = A* • Row vectors live in dual space • Differential forms, covectors in physics • Gradients are dual to directions
