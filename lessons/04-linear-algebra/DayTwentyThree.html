Section 1

What is a Vector Space?

A Vector Space is a set $V$ of objects (vectors) equipped with two operations: Addition and Scalar Multiplication, satisfying 10 axioms (closure, commutativity, associativity, zero vector, etc.).

Common Spaces:
$\mathbb{R}^n$: Column vectors of size $n$.
$\mathbb{P}_n$: Polynomials of degree at most $n$.
$M_{m \times n}$: Matrices of size $m \times n$.
Section 2

Subspaces

A subset $W \subseteq V$ is a subspace if:
  1. It contains the zero vector $\vec{0}$.
  2. It is closed under addition ($\vec{u}, \vec{v} \in W \implies \vec{u}+\vec{v} \in W$).
  3. It is closed under scalar multiplication ($c \in \mathbb{R}, \vec{u} \in W \implies c\vec{u} \in W$).

Example: A line through the origin in $\mathbb{R}^2$ is a subspace. A line NOT through the origin is NOT.

Section 3

Linear Independence

A set of vectors $\{v_1, \dots, v_k\}$ is Linearly Independent if the only solution to $$ c_1 v_1 + \dots + c_k v_k = \vec{0} $$ is the trivial solution ($c_1 = \dots = c_k = 0$).

If there is a non-trivial solution, they are Dependent (one vector is a combination of others).

Section 4

Basis & Dimension

Span: The set of all linear combinations of a set of vectors.
Basis: A set of vectors that is Linearly Independent AND Spans the space. It is a "minimal spanning set".
Dimension: The number of vectors in a basis. It is unique for a given space.
Dim($\mathbb{R}^3$) = 3. Dim($\mathbb{P}_2$) = 3 ($1, t, t^2$).
Section 5

Python Check

We can check for independence by forming a matrix and calculating the Rank. If Rank = number of vectors, they are independent.

Python Independence Check 
import numpy as np

# Vectors to check: (1,0,0), (0,1,0), (1,1,0)
# We put them as ROWS or COLUMNS. Let's stack them as rows.
vectors = np.array([
    [1, 0, 0],
    [0, 1, 0],
    [1, 1, 0]
])

# Calculate rank
rank = np.linalg.matrix_rank(vectors)
print(f"Number of vectors: {len(vectors)}")
print(f"Rank: {rank}") 
# Rank is 2. But we have 3 vectors. 
# Thus, they are DEPENDENT. (v3 = v1 + v2)

if rank < len(vectors):
    print("Linearly Dependent")
else:
    print("Linearly Independent")
