Linear Systems

A linear system $Ax = b$ represents a set of linear equations.

2x + 3y = 5 4x - y = 3 Matrix Form: [ 2 3 ] [ x ] [ 5 ] [ 4 -1 ] [ y ] = [ 3 ]
Section 2

Operations

Addition: Element-wise. Requires same dimensions.
Multiplication (Dot Product): Row times Column.
$(AB)_{ij} = \sum_k A_{ik} B_{kj}$
Requires inner dimensions to match ($M \times N$ times $N \times K$).
Transpose ($A^T$): Flip rows and columns.
Inverse ($A^{-1}$): $A A^{-1} = I$. Only for square, non-singular matrices.
Section 3

Gaussian Elimination

The systematic algorithm to solve $Ax=b$ by converting $A$ to Row Echelon Form (REF).

  1. Subtract multiples of the pivot row from rows below to produce zeros.
  2. Move to next pivot.
  3. Back Substitution: Solve for variables from bottom up.
LU Decomposition: Gaussian elimination factors $A$ into lower ($L$) and upper ($U$) triangular matrices. $A = LU$. This allows solving $Ax=b$ quickly for multiple vectors $b$.
Section 4

Rank & Solutions

The Rank of a matrix is the number of linearly independent rows (or pivots).
  • Rank $r = m = n$: Full rank. Unique solution.
  • Rank $r < n$: Infinite solutions (free variables exist).
  • Rank $r < m$: May have no solution (if system inconsistent).
Section 5

NumPy Matrix Power

NumPy is the standard for Linear Algebra in Python.

Python Solving Ax=b 
import numpy as np

# Define Matrix A and Vector b
A = np.array([
    [2, 3],
    [4, -1]
])
b = np.array([5, 3])

# Solve Ax = b
x = np.linalg.solve(A, b)
print(f"Solution x: {x}") # [1. 1.]

# Matrix Multiplication
B = np.array([[1, 0], [0, 1]]) # Identity
C = A @ B # @ operator is matrix mult
print(f"\nA @ I:\n{C}")

# Rank
rank = np.linalg.matrix_rank(A)
print(f"Rank: {rank}")
