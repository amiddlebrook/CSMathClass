Section 1

Discrete Distributions

Bernoulli(p)

Single trial with success probability p X ∈ {0, 1} P(X = 1) = p, P(X = 0) = 1-p E[X] = p Var(X) = p(1-p) Use: Single yes/no experiment

Binomial(n, p)

Number of successes in n independent Bernoulli trials X ∈ {0, 1, ..., n} P(X = k) = C(n,k) · p^k · (1-p)^(n-k) E[X] = np Var(X) = np(1-p) Note: Sum of n independent Bernoulli(p) RVs

Geometric(p)

Number of trials until first success X ∈ {1, 2, 3, ...} P(X = k) = (1-p)^(k-1) · p E[X] = 1/p Var(X) = (1-p)/p² Memoryless: P(X > s+t | X > s) = P(X > t)

Poisson(λ)

Number of events in fixed interval X ∈ {0, 1, 2, ...} P(X = k) = λ^k · e^(-λ) / k! E[X] = λ Var(X) = λ Use: Rare events, arrival processes
Python Discrete distributions in SciPy 
from scipy import stats
import numpy as np

# Binomial: 10 coin flips, P(heads) = 0.5
binom = stats.binom(n=10, p=0.5)
print(f"P(X = 5) = {binom.pmf(5):.4f}")  # 0.2461
print(f"P(X ≤ 5) = {binom.cdf(5):.4f}")  # 0.6230
print(f"E[X] = {binom.mean()}")           # 5.0

# Geometric: Expected flips until heads
geom = stats.geom(p=0.5)
print(f"E[flips] = {geom.mean()}")  # 2.0

# Poisson: Average 3 emails per hour
poisson = stats.poisson(mu=3)
print(f"P(0 emails) = {poisson.pmf(0):.4f}")  # 0.0498
print(f"P(≥5 emails) = {1 - poisson.cdf(4):.4f}")  # 0.1847
Section 2

Continuous Distributions

Uniform(a, b)

Equal probability on interval [a, b] f(x) = 1/(b-a) for x ∈ [a, b] E[X] = (a+b)/2 Var(X) = (b-a)²/12 Use: Random number generation baseline

Exponential(λ)

Time between Poisson events f(x) = λ·e^(-λx) for x ≥ 0 E[X] = 1/λ Var(X) = 1/λ² Memoryless: P(X > s+t | X > s) = P(X > t) Only continuous memoryless distribution!

Normal(μ, σ²)

The Normal (Gaussian) distribution: The most important distribution in statistics.
f(x) = (1/(σ√(2π))) · e^(-(x-μ)²/(2σ²)) E[X] = μ Var(X) = σ² Standard Normal N(0,1): Z = (X - μ)/σ 68-95-99.7 rule: • 68% within 1σ of mean • 95% within 2σ of mean • 99.7% within 3σ of mean
Central Limit Theorem: The sum of many independent RVs (regardless of their distribution) approaches a Normal distribution! This explains why the Normal is so ubiquitous.
Python Continuous distributions in SciPy 
from scipy import stats

# Standard Normal
norm = stats.norm(loc=0, scale=1)
print(f"P(Z < 1.96) = {norm.cdf(1.96):.4f}")  # 0.9750
print(f"P(|Z| < 2) = {norm.cdf(2) - norm.cdf(-2):.4f}")  # 0.9545

# Inverse: what z gives P(Z < z) = 0.95?
print(f"z for 95% = {norm.ppf(0.95):.4f}")  # 1.6449

# Exponential: server response time, avg 100ms
exp = stats.expon(scale=100)  # scale = 1/λ = mean
print(f"P(response < 50ms) = {exp.cdf(50):.4f}")  # 0.3935
print(f"P(response > 200ms) = {1 - exp.cdf(200):.4f}")  # 0.1353

# Central Limit Theorem demo
import numpy as np
samples = [np.mean(np.random.uniform(0, 1, 30)) for _ in range(10000)]
print(f"Mean of sample means: {np.mean(samples):.4f}")  # ~0.5
print(f"Sample means are normally distributed!")
Section 3

Relationships Between Distributions

IMPORTANT CONNECTIONS: Bernoulli(p) → sum of n → Binomial(n, p) Binomial(n, p) as n→∞ → Poisson(np) when p small Geometric(p) → sum of n → NegativeBinomial(n, p) Poisson(λ) ←→ Exponential(λ) (counts vs. waiting times) Sum of many anything → Normal (CLT) Binomial(n, p) → Normal(np, np(1-p)) for large n
When to use which:
  • Binomial: Fixed n trials, count successes
  • Geometric: Trials until first success
  • Poisson: Events in interval, rare events approximation
  • Exponential: Time between events
  • Normal: Sums, averages, measurement errors
