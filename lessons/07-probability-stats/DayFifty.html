Section 1

Random Variables

Definition

Random Variable X: A function X: Ω → ℝ that maps each outcome in the sample space to a real number.

Random variables let us talk about numerical properties of random experiments. Instead of asking "what's the probability of HHT?", we ask "what's the expected number of heads?"

Discrete RV

Takes countable values PMF: P(X = x) = probability mass Σ P(X = x) = 1 Example: number of heads in 3 coin flips X ∈ {0, 1, 2, 3}

Continuous RV

Takes uncountable values PDF: f(x) = probability density P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx ∫ f(x)dx = 1 Example: exact height of a randomly chosen person
Python Random variables in NumPy 
import numpy as np

# Discrete RV: number of heads in 10 flips
samples = np.random.binomial(n=10, p=0.5, size=10000)
print(f"Mean: {samples.mean():.2f}")  # ~5.0
print(f"Var:  {samples.var():.2f}")   # ~2.5

# Continuous RV: normal distribution
samples = np.random.normal(loc=0, scale=1, size=10000)
print(f"Mean: {samples.mean():.2f}")  # ~0.0
print(f"Std:  {samples.std():.2f}")   # ~1.0
Section 2

Expected Value

Definition

DISCRETE: E[X] = Σ x · P(X = x) CONTINUOUS: E[X] = ∫ x · f(x) dx Intuition: Weighted average of all possible values, where weights are probabilities.

Properties of expectation

Linearity of Expectation: The most powerful property in probability!
E[aX + b] = a·E[X] + b (scaling/shifting) E[X + Y] = E[X] + E[Y] (ALWAYS TRUE!) E[X · Y] = E[X] · E[Y] (only if independent) LINEARITY WORKS EVEN FOR DEPENDENT VARIABLES! This is incredibly useful for algorithm analysis.
Python Expected value calculation 
def expected_value(values, probabilities):
    """Compute E[X] for discrete RV"""
    return sum(x * p for x, p in zip(values, probabilities))

# Fair die
values = [1, 2, 3, 4, 5, 6]
probs = [1/6] * 6
E_die = expected_value(values, probs)
print(f"E[die] = {E_die}")  # 3.5

# Linearity example: sum of two dice
# E[X + Y] = E[X] + E[Y] = 3.5 + 3.5 = 7
# No need to enumerate all 36 outcomes!
Algorithm analysis tip: Use linearity to break complex expectations into simple pieces. Example: E[comparisons in QuickSort] = Σ E[comparing elements i,j].
Section 3

Variance & Standard Deviation

Definition

Var(X) = E[(X - μ)²] where μ = E[X] = E[X²] - (E[X])² (computational formula) σ = √Var(X) (standard deviation) Intuition: Average squared distance from the mean. How "spread out" is the distribution?

Properties of variance

Var(aX + b) = a² · Var(X) (scaling squares, shift ignored) Var(X) ≥ 0 (always non-negative) Var(X) = 0 ⟺ X is constant (no randomness) For INDEPENDENT X, Y: Var(X + Y) = Var(X) + Var(Y) WARNING: Variance is NOT linear! Var(X + Y) ≠ Var(X) + Var(Y) in general
Common mistake: Unlike expectation, variance does NOT satisfy Var(X + Y) = Var(X) + Var(Y) for dependent variables. You need the covariance term: Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y).
Python Computing variance 
def variance(values, probabilities):
    """Compute Var(X) = E[X²] - E[X]²"""
    E_X = sum(x * p for x, p in zip(values, probabilities))
    E_X2 = sum(x**2 * p for x, p in zip(values, probabilities))
    return E_X2 - E_X**2

# Fair die variance
values = [1, 2, 3, 4, 5, 6]
probs = [1/6] * 6
var_die = variance(values, probs)
print(f"Var(die) = {var_die:.4f}")  # 2.9167
print(f"σ(die) = {var_die**0.5:.4f}")  # 1.7078
Section 4

Indicator Random Variables

Definition

Indicator variable Iₐ: Iₐ = 1 if event A occurs, 0 otherwise.
E[Iₐ] = 1·P(A) + 0·P(Aᶜ) = P(A) This simple fact + linearity = powerful technique! Example: Count heads in n flips X = I₁ + I₂ + ... + Iₙ where Iᵢ = 1 if flip i is heads E[X] = E[I₁] + ... + E[Iₙ] = n · (1/2) = n/2
Python Indicator variables for algorithm analysis 
# Example: Expected number of fixed points in random permutation
# Fixed point: position i where π(i) = i

import random

def count_fixed_points(n, trials=100000):
    total = 0
    for _ in range(trials):
        perm = list(range(n))
        random.shuffle(perm)
        fixed = sum(1 for i in range(n) if perm[i] == i)
        total += fixed
    return total / trials

# Theory: E[fixed points] = Σ E[Iᵢ] = Σ P(π(i)=i) = n · (1/n) = 1
print(f"n=10: {count_fixed_points(10):.2f}")   # ~1.0
print(f"n=100: {count_fixed_points(100):.2f}")  # ~1.0
# Same expected value regardless of n!
