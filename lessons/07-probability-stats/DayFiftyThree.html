Section 1

Point Estimation

Estimators and estimates

Estimator: A statistic (function of sample data) used to approximate an unknown population parameter.
COMMON ESTIMATORS: Sample mean X̄ = (1/n)Σxᵢ → estimates μ Sample variance s² = Σ(xᵢ-X̄)²/(n-1) → estimates σ² Sample proportion p̂ = k/n → estimates p TERMINOLOGY: θ = true parameter (unknown) θ̂ = estimator (random variable) θ̂ = estimate (specific value)

Desirable properties

UNBIASED: E[θ̂] = θ Estimator is correct on average. CONSISTENT: θ̂ → θ as n → ∞ Converges to truth with more data.
EFFICIENT: Minimum variance among unbiased estimators. Best precision. MSE = Var(θ̂) + Bias² Mean squared error (trade-off)
Python Demonstrating estimator properties 
import numpy as np

# True population parameters
true_mu, true_sigma = 100, 15

# Demonstrate unbiasedness of sample mean
estimates = [np.mean(np.random.normal(true_mu, true_sigma, 30)) 
             for _ in range(10000)]
print(f"E[X̄] ≈ {np.mean(estimates):.2f}")  # ~100 (unbiased)

# Demonstrate consistency: larger n → tighter estimates
for n in [10, 100, 1000]:
    estimates = [np.mean(np.random.normal(true_mu, true_sigma, n)) 
                 for _ in range(1000)]
    print(f"n={n:4d}: std of X̄ = {np.std(estimates):.3f}")
# Output: std decreases as 1/√n
Section 2

Confidence Intervals

Definition

95% Confidence Interval: A random interval [L, U] such that P(L ≤ θ ≤ U) = 0.95. If we repeat the experiment many times, 95% of the intervals will contain the true parameter.

Confidence interval for mean

KNOWN VARIANCE σ²: CI = X̄ ± z_{α/2} · (σ/√n) UNKNOWN VARIANCE (use s): CI = X̄ ± t_{α/2, n-1} · (s/√n) COMMON z VALUES: 90% CI: z = 1.645 95% CI: z = 1.96 99% CI: z = 2.576
Common misconception: A 95% CI does NOT mean "95% probability that the true mean is in this interval." The true mean is fixed! It means 95% of similarly-constructed intervals would contain the true mean.
Python Computing confidence intervals 
from scipy import stats
import numpy as np

def confidence_interval(data, confidence=0.95):
    """Compute CI for mean using t-distribution"""
    n = len(data)
    mean, se = np.mean(data), stats.sem(data)
    t_crit = stats.t.ppf((1 + confidence) / 2, n - 1)
    margin = t_crit * se
    return mean - margin, mean + margin

# Example: sample data
data = np.random.normal(100, 15, 30)
low, high = confidence_interval(data)
print(f"95% CI: [{low:.2f}, {high:.2f}]")

# Verify coverage: repeat many times
true_mu = 100
covered = 0
for _ in range(1000):
    data = np.random.normal(true_mu, 15, 30)
    low, high = confidence_interval(data)
    if low <= true_mu <= high:
        covered += 1
print(f"Coverage: {covered/10:.1f}%")  # ~95%

Hypothesis Testing

The framework

H₀: Null hypothesis (status quo, no effect) H₁: Alternative hypothesis (what we want to show) PROCEDURE: 1. State H₀ and H₁ 2. Choose significance level α (typically 0.05) 3. Compute test statistic from data 4. Find p-value: P(data this extreme | H₀ true) 5. Decision: Reject H₀ if p-value < α

Types of errors

TYPE I ERROR: Reject H₀ when H₀ is true "False positive" P(Type I) = α We control this by choosing α
TYPE II ERROR: Fail to reject H₀ when H₀ is false "False negative" P(Type II) = β Power = 1 - β
Python One-sample t-test 
from scipy import stats
import numpy as np

# Test: is the mean different from 100?
data = np.array([105, 98, 112, 103, 95, 108, 101, 99, 107, 104])
null_mean = 100

# One-sample t-test
t_stat, p_value = stats.ttest_1samp(data, null_mean)
print(f"t-statistic: {t_stat:.3f}")
print(f"p-value: {p_value:.4f}")

alpha = 0.05
if p_value < alpha:
    print("Reject H₀: Mean is significantly different from 100")
else:
    print("Fail to reject H₀: No significant difference")
Common tests: t-test (means), chi-square (proportions/independence), ANOVA (multiple groups), Mann-Whitney (non-parametric).
